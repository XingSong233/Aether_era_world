# 设定发布索引

本文件对应 Matrees“设定”栏目，用于说明本地专题如何映射到网站分类。

## 分类口径

| 网站分类 | 本地内容类型 | 说明 |
| --- | --- | --- |
| 设定集 | 总体制度、社会结构、经济、航行、科技状态、术语总表 | 适合承载跨领域专题和说明性资料。 |
| 能力 | 灵核、灵脉、灵械、能力成长、灵脉谱系、标准化、职业能力模板 | 对应能力系统。 |
| 角色 | 主角、候选角色、人物设定卡、视觉设定卡 | 后续建议一人一页。 |
| 组织 | 文明圈、机构、舰队、商盟组织、议会、军工体系 | 当前文明圈对照表可先归入组织或设定集。 |
| 地理 | 星域、航道、空间锚、星港、暗区、深层以太边界 | 航行专题可先归入设定集，后续拆地点对象。 |
| 物品 | 灵械、舰船、机甲、遗产样本、工具装备 | 当前载人机甲可先归入设定集或物品。 |
| 其他 | 工作台索引、发布说明、非正文辅助资料 | 不应误标为世界内正式设定。 |

详细文件级映射见 `../90_发布记录/Matrees发布映射表.md`。
