# 设定工作台

本目录用于存放辅助世界观扩写、口径冻结、状态追踪和保守推导的工具文件。

这里的文件可以具有权威参考价值，但它们不是世界观正文专题。写作时应先用工作台文件确认科技状态、文明态度、风险边界和待扩写项，再把稳定内容整理进 `../../02_总纲/以太纪元世界观总纲.md` 或 `../../03_专题设定/` 下的对应正文专题。

## 当前文件

- `关键科技与生产关系状态表.md`：确认关键科技是否存在、是否成熟、由谁掌握，以及它对生产关系推导的边界。
- `四文明关键科技态度矩阵.md`：从文明制度出发，保守推导四大文明对关键科技簇的态度、管控方式和冲突点。
- `经济体系设计蓝图.md`：设计星海多层经济、多结算制、基础服务权利、四文明经济制度、主要经济冲突和外部结构参照转译；用于辅助 `../../03_专题设定/经济体系.md` 与 `../../03_专题设定/生产关系与职业结构.md` 扩写，不是世界观正文专题。
- `航行自由跃迁与开拓秩序设计蓝图.md`：设计星际航行、自由跃迁、航权治理、空间锚链、边境救援、开拓许可和殖民地承认秩序；用于辅助 `../../03_专题设定/星际航行和开拓秩序/航行自由跃迁与开拓秩序.md` 扩写，不是世界观正文专题。
- `星海地理与星图节点设计蓝图.md`：设计主星图坐标范围、文明节点数量、悬臂段分布、结构化地图数据字段和后续地图扩展规则；用于辅助 `../../03_专题设定/星际航行和开拓秩序/星海地理与星图节点.md` 与发布层地图扩展，不是世界观正文专题。
- `星海疆域生成器/`：按随机数种子、航路成本、深层以太区、以太洋流、星门、遗迹和四文明扩张参数生成可回放的主星图疆域模拟结果；用于辅助地图布点和疆域形态推演，不是世界观正文专题。
- `信息治理与社会制度设计蓝图.md`：设计共鸣网络、感官记录、预测算法、审计账本、身份凭证、自动执法、AI 信息和深层污染信号的治理边界；用于辅助 `../../03_专题设定/信息治理与社会制度.md` 扩写，不是世界观正文专题。
- `专题写作辅助与检查清单.md`：集中存放从正式专题迁出的写作用途、模板格式、检查清单、避免写法和旧口径修正说明；用于辅助扩写，不是世界观正文专题。
- `Matrees写文本技能目录.md`：保存当前项目使用的 Matrees 写文本与富文本发布技能目录，包含网站正文、世界内文档和叙事文本风格，以及富文本发布检查规则；用于项目内追溯和后续技能维护，不是世界观正文专题。

## 使用原则

- 不把状态表、矩阵或检查清单直接当作世界内叙述。
- 不把“保守推导”写成已经生效的硬制度。
- 正式专题应优先保留世界内设定、制度结构、技术边界和示例对象；写作辅助、检查清单和模板统一放入本工作台。
- 若工作台判断改变核心规则，应同步更新总纲、相关专题文件、版本索引和 CHANGELOG。
