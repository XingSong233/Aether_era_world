# 关键科技与生产关系状态表

状态：采用
基准：`../../02_总纲/以太纪元世界观总纲.md`、`../../03_专题设定/能力系统/能力系统索引.md`、`../../03_专题设定/文明圈对照表.md`、`../../03_专题设定/深层以太设定.md`、`../../03_专题设定/术语表.md`

本文件用于冻结和追踪“以太纪元”中会显著影响生产关系的关键科技状态。它不是技术原理正文，而是判断社会制度、产业组织、职业分工、所有权结构和文明冲突时的索引表。

使用本表时，应优先确认一项科技的状态，再推导它会怎样改变生产资料、劳动组织、阶层结构、监管制度和日常生活。若某项科技改变核心规则，应同步更新总纲、相关专题文件和版本记录。

## 一、状态口径

| 状态 | 含义 |
| --- | --- |
| 采用 | 已纳入当前世界观，可作为后续推导依据。 |
| 采用，待扩写 | 方向已确认，但技术原理、性能边界、分级或制度细节仍需后续专题补充。 |
| 军用管制 | 存在且成熟度较高，但主要由军方、舰队、执法或高危任务机构掌握。 |
| 民用普及 | 普通公民、家庭、学校、企业或城市基础设施可合法接触。 |
| 职业许可 | 需要培训、执照、维护记录或行会/机构担保才能使用。 |
| 高危任务 | 只在深层以太、遗产、污染区、暗区、灾害救援或极端工程中使用。 |
| 遗产隔离 | 来自古文明、深层以太或未知来源，必须经过沙箱、单向转译、污染熔断和人工复核。 |
| 禁限 | 技术存在或理论可行，但因伦理、灾害、污染、人格风险或文明安全被限制。 |
| 黑市存在 | 正规制度禁止或限制，但灰色市场、走私网络、非法工坊或激进派仍会使用。 |
| 待确认 | 尚未决定是否存在、是否成熟或是否纳入世界观。 |

## 四、基于当前状态的生产关系推导

本节只从已确认或已采用但待扩写的科技状态继续推导生产关系，不替代后续经济制度、AI 专题、金融专题或航行专题。已确认但边界未完全展开的科技只能按本表保守口径使用，不得提前写成无限能力或单一制度答案。

### 3. 阶层与冲突来源

| 冲突来源 | 推导口径 | 可用于剧情或专题扩写的表现 |
| --- | --- | --- |
| 权限阶层 | 普遍基本供给存在，基础生存不是主要贫困来源；差异更多来自接口权限、航权、教育机会、医疗许可、居住场域和风险暴露。 | 边境居民有基本生活，却缺少合法接口升级、稳定中继和高质量职业训练；他们不饿死，但很难进入核心职业链。 |
| 维护阶层 | 高端设备和能力模块需要持续维护、校准、复检和责任记录。 | 灵械维护员、标准审查官、事故调查员和保险评估师拥有现实权力；他们不一定拥有生产资料，却能决定生产资料是否继续合法运行。 |
| 平台依附 | 灵械租赁、授权密钥、商盟商品规格和风险定价让个体可以使用昂贵能力模块，但也可能被合约、债务和密钥冻结控制。 | 商盟自由市场表面上降低门槛，实际通过质保、仲裁、密钥续费和事故赔付条款制造长期依附。 |
| 安全阈值冲突 | 环带、联邦、裂谷和商盟对危险技术的容忍度不同。 | 同一深层样本在联邦是受控研究，在环带是封存对象，在裂谷是进化材料，在商盟是可包装的高风险资产。 |
| 身份与人格边界 | AI、数字人格、克隆、人造子宫和跨种族生育改变身份来源，但复活和记忆编辑不存在，死亡与人格连续性仍有重量。 | 法庭不能靠记忆编辑解决证词，继承不能靠复活回滚；数字人格备份、非法克隆体、军用/实验克隆体和生命结构承载 AI 的权利问题仍需制度处理。克隆体没有生物本质劣等性，权利剥夺来自制度选择。 |
| 自动化后的劳动价值 | 通用机器人和分子装配机压缩体力劳动，但不能替代职业经验、事故判断、伦理责任和深层以太接触后的复检。 | “普通劳动消失”不应写成全民无事可做，而应写成低技能岗位减少、认证岗位增多、公共供给兜底与机会竞争并存。 |
| 信息治理边界 | 共鸣网络、预测算法、感官共享和大规模监控存在，但联邦不采用单一社会信用体系，政策上不允许算法直接干预个体。 | 地方机构、保险公司或企业可能灰色越权；冲突来自“原则上不允许”和“执行层很想这么做”之间。 |
| 高危职业外包 | 私人军事公司、商盟安保、合约勘探和遗产开发使风险可被市场分配，但事故后仍会回到文明责任。 | 商盟可以用保险和合同拆分责任，联邦与环带会追问谁授权、谁复检、谁允许样本进入正式系统。 |
| 生育设计边界 | 胚胎设计已成体系但无法定制灵核，非灵核生理设计、跨种族兼容和殖民适应仍会制造公平问题。 | 基础疾病筛除和人造子宫服务可广泛提供，但高端优化、跨种族定制人造子宫和特殊环境适应仍受费用、补贴和权限影响。监管机构必须说明这不是能力天赋定制；裂谷胚胎期灵核干预只能写成争议性、不稳定探索。 |
| 自动合约救济权 | 智能合约可以自动执行冻结、赔付、租赁到期和授权撤回，但必须保留仲裁、司法、伦理审查和文明监管入口。 | 商盟客户被密钥冻结后仍可申请仲裁；联邦和环带会要求高危样本、军用规格和医疗服务不能只按自动条款处置。 |
| 能源额度与航权阶层 | 能源不是免费，能源额度、航权、星门窗口和中继稳定性会成为比货币本身更具体的机会门槛。 | 边境世界有基本供给，却缺少高质量能源额度和航路保险；小企业能买设备，但无法稳定获得生产时段。 |
| AI 公民劳动边界 | 高等级 AI 与数字人格可参与劳动和治理，但不是无成本复制的劳动力。心智核心、心智之种、人工灵核、人格稳定、载体维护和文明权利地位限制其被当作工具。 | 联邦 AI 公民拒绝被平台拆成无限工作副本；环带限制 AI 权利并禁止接入灵核/灵脉；裂谷开放完整公民权但承担实验风险；商盟以合约法人处理 AI 劳动，容易出现债务、载体维护和保险条款压迫。 |
| 巨构维护权争议 | 轨道级和行星级巨构成熟后，真正稀缺的是维护权、居住资格、能源配额、防护优先级和事故解释权。 | 居民不一定拥有居住设施的维护决策权；一次护盾或生态系统检修可能暴露巨构管理者、能源部门和居民自治组织的冲突。 |

### 4. 文明差异推导

## 五、生产关系推导模板

新增或修改关键科技时，建议按以下字段记录：

| 字段 | 用途 |
| --- | --- |
| 技术名称 | 使用世界内术语，避免直接沿用临时外部比喻。 |
| 当前状态 | 采用、待扩写、军用管制、民用普及、禁限、遗产隔离等。 |
| 掌握主体 | 哪个文明、机构、企业、行会、军方、黑市或古文明遗产掌握。 |
| 普及程度 | 普通家庭、职业机构、城市基础设施、舰队、巨头企业或封存机构。 |
| 替代了什么劳动 | 体力、感知、认知、管理、制造、医疗、运输、照护或战斗。 |
| 创造了什么新职业 | 维护、认证、调度、审查、保险、仲裁、训练、封存或黑市岗位。 |
| 形成什么所有权 | 私有、国有、军方、行会、企业平台、文明公共设施、封存库。 |
| 主要限制 | 能源、材料、接口、算力、污染、伦理、法律、权限或维护成本。 |
| 生产关系影响 | 对阶层、产业链、家庭、城市、殖民地、军队、市场和国家能力的影响。 |

## 六、写作检查清单

- 不要只问“这项科技强不强”，应先问它由谁掌握、谁维护、谁付费、谁承担事故责任。
- 不要把技术存在等同于全民普及。军用、民用、职业许可、遗产隔离和黑市状态会导向完全不同的社会结构。
- 不要让高端科技无成本消除社会矛盾。若它提高生产力，应同时检查它是否制造新的权限、维护、认证、债务、污染或阶层壁垒。
- 涉及 AI 时，当前已确认强人工智能、超越旧 L4 等级的通用 AI 和数字人格技术可以存在。人工意识不再作为独立科技分类，后续统一归入人工智能。AI 多路线并存：人格上传至心智核心，心智之种置入心智核心并培养，传统 AI 计算路线与以太技术结合。旧 L1-L4 保留，L4 已与智慧个体无异；超 L4 可短期实现多人格/多线程，L5 可长期稳定维持多人格/多线程。AI 可调用算力受人格和思维稳定度限制，不是无限算力工具；传统计算路线最高可达 L5，但不能接入灵核或灵脉，也不能由生命结构承载。数字人格技术成熟，可以无损数字化生命体意识，但数字人格并非无限寿命，会受以太场影响而具有有限稳定周期，且无法进行数字人格繁殖。文明级 AI 当前不存在。生命结构承载后的权利按承载前身份处理，种族身份按承载生命结构判定，并标注数字转移来源；联邦个案评估繁殖权，环带禁止繁殖，裂谷不设额外限制。联邦中 L4 及以上 AI 具有完整公民权和法人地位；环带不允许 AI 接入灵核、灵脉，只允许接入算力，通常不享有完整公民权；裂谷允许 AI 享有完整公民权；商盟承认 L4 及以上 AI/数字人格为合约法人主体。
- 涉及生命工程、自动化和材料制造时，当前已确认克隆技术、人造子宫、胚胎非灵核设计、跨种族生育、体内纳米机器人、分子装配机、通用机器人、自修复材料和可编程物质均存在，其中克隆、分子装配机和通用机器人已高度成熟。完整智慧生命克隆的合法性和克隆体权利存在强文明差异；基础人造子宫服务对有生育权或繁殖权的个体免费开放；跨种族生育需要特制或定制人造子宫；胚胎设计不能稳定定制灵核六维、灵核倾向、能力天赋或出生职业。推导生产关系时，应优先检查所有权、授权密钥、材料/能源约束、设备维护、伦理许可、生育权、补贴制度和文明差异，而不是只用“生产力极高”一笔带过。
- 涉及能源、航行和远程作业时，当前已确认以太能不是近乎免费能源，惯性控制和小型传送技术不存在，自由跃迁存在但需要跃迁龙骨、锚定解、跃迁窗口、能源额度、航图、复检和目的地接纳，低成本星际航行已进入大众商业交通成本带但仍受航权、保险、排期、检疫和目的地资格限制。远程存在与远程驾驶依赖合适以太场环境，并可受到以太场/深层以太场清洁与屏蔽技术影响。
- 涉及高速载人机甲时，当前已确认惯性承载体系存在但不等于惯性控制。百倍重力级机动应写成军用机师、承压灵脉、承载舱、灵械自动响应、医疗预处理、战后复检和无人系统分工共同支撑的高代价能力。
- 涉及治理、传媒、身份和教育时，应区分技术能力与制度态度：联邦道德上排斥长期娱乐沉浸和长期虚拟身份生活，感官记录与共享受管控；联邦不采用单一社会信用体系，预测算法政策上只能监控预警、不应直接干预；教育灌注和技能临时租赁不存在，职业能力仍依赖学习、训练、认证和经验。
- 涉及复制人格、复活、自由跃迁、时间工程、近乎免费能源和万物制造时，应先确认是否会破坏星门、资源、死亡代价、职业分工和文明边界。
