# 以太纪元世界观项目

本目录用于维护“以太纪元”世界观资料。当前采用三层结构：世界观正文、展示发布层、协作工作台。根目录只保留项目入口、协作入口和少量工具配置。

## 目录结构

- `AGENTS.md`：协作入口说明书，保留在根目录供协作工具读取；详细规则位于 `90_协作工作台/20_写作指南与样稿库/协作规则/`。
- `10_世界观正文/`：权威设定正文。总纲、采用状态的正式专题、术语、能力系统和航行专题均放在这里。
- `20_展示发布层/`：面向 Matrees 和后续独立建站的中间发布层，承载背景、公开设定导览、时间线、地图、作品和关系图谱对象。
- `90_协作工作台/10_设定推导与蓝图/`：状态表、态度矩阵、设计蓝图、理论层级整理、推导辅助和原型工具。
- `90_协作工作台/20_写作指南与样稿库/`：协作规则、专题写作辅助、草稿与候选、Matrees 写文本技能目录、世界内文本风格样稿和候选稿。
- `90_协作工作台/30_素材与导出/`：图片素材、素材登记表、`.docx` 与海报等导出文件。
- `90_协作工作台/80_原始与废弃追溯/`：原始资料、废弃资料和已撤回稿，只用于追溯来源。
- `90_协作工作台/90_版本记录/`：版本索引、CHANGELOG 和版本管理规则。

## 维护原则

- 新增稳定设定先进入 `10_世界观正文/10_专题设定/`；如果影响核心规则，再同步更新总纲。
- 新增设定时必须执行协作规则检查；若产生新硬口径、可复用写法、禁用写法、错误教训、引用路径或维护流程变化，应同步更新 `90_协作工作台/20_写作指南与样稿库/协作规则/`。
- 正式专题保留设定口径、制度结构、技术边界、文明差异和示例对象；模板、检查清单、写作边界和风格样稿集中放入协作工作台。
- 新增或调整公开展示的设定、角色、事件、地图、作品或关系时，应同步检查 `20_展示发布层/90_发布记录/Matrees发布映射表.md`。
- 每次结构调整、总纲改写或核心设定变更，都要更新 `90_协作工作台/90_版本记录/CHANGELOG.md`。
- 形成稳定节点时，在 `90_协作工作台/90_版本记录/版本索引.md` 记录版本号、日期、主文件和变更摘要。
- 图片素材应进入 `90_协作工作台/30_素材与导出/图片素材/` 并登记来源、授权、用途和采用状态；图片本身不自动成为世界观正文设定。
- 原始资料、废弃资料和已撤回稿只用于追溯，不直接作为当前采用口径。
