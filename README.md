# 以太纪元世界观项目

本目录用于维护“以太纪元”世界观资料。当前采用轻量文件版本管理：正文按用途分区，重要变更记录在版本索引和变更记录中，原始资料只作追溯来源，不直接覆盖删除。

## 目录结构

- `AGENTS.md`：创作指南和维护规则，保留在根目录供协作工具读取。
- `00_项目管理/`：项目说明、版本规则、维护说明；设定工作台文件集中在 `00_项目管理/设定工作台/`。
- `01_原始资料/`：原始设定资料，只用于追溯来源。
- `02_总纲/`：当前权威总纲，后续创作优先参考这里。
- `03_专题设定/`：术语、角色、文明、剧情等专题扩写；能力系统专题集中在 `03_专题设定/能力系统/`，星际航行专题集中在 `03_专题设定/星际航行和开拓秩序/`。
- `04_版本记录/`：版本索引、变更记录、版本快照说明。
- `05_导出文件/`：`.docx` 等导出文件。
- `06_图片素材/`：图片素材、参考图、生成图和成品图集中管理区，素材采用状态登记在 `06_图片素材/素材登记表.md`。
- `07_网站发布结构/`：面向 Matrees 和后续独立建站的中间发布层，按背景、设定、时间线、地图、作品和关系图谱组织网站内容对象。

## 维护原则

- 原始资料不直接删除，不直接覆盖。
- 新增设定先进入对应专题文件；如果影响核心规则，再同步更新总纲。
- 世界内正文、模板和检查清单应分层维护；专题正文保留正式设定，写作辅助集中到 `00_项目管理/设定工作台/专题写作辅助与检查清单.md`。
- 新增或调整正式专题、角色、事件、地图、作品或公开设定时，应同步检查 `07_网站发布结构/90_发布记录/Matrees发布映射表.md` 是否需要更新。
- 每次结构调整、总纲改写或核心设定变更，都要更新 `04_版本记录/CHANGELOG.md`。
- 形成稳定节点时，在 `04_版本记录/版本索引.md` 记录版本号、日期、主文件和变更摘要。
- 图片素材应进入 `06_图片素材/` 并登记来源、授权、用途和采用状态；图片本身不自动成为世界观正文设定。
