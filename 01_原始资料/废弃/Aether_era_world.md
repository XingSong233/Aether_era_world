# 以太纪元：世界观与人物设定

## 世界概述

文明不是为了战争而扩张，而是为了生存与探索不得不掌握战争。

**时代背景：** 世界进入“第二星海时代”（联邦历 1487 年），各文明均基于以太技术踏上星际拓荒之路，却依然对深空未知保持敬畏。银河没有统一的版图，像一片被黑暗海洋分割的群岛，各文明通过星门和以太洋流联系，又彼此隔绝。

**宇宙观：** 宇宙由四层组成——物质层（传统物理）、能量层、信息层和以太层。以太同时是能量场、信息介质和意识共振场，是所有高科技和超感知能力的基础。

**基本理念：** 智慧生命通过灵核与以太交互，灵脉作为灵核中可成长的信息回路，使个体获得各种感知、认知、操控和生理强化功能。灵脉并非与生俱来，而是可设计、诱导甚至铭刻的科技产物。

## 宇宙历史概略

| 阶段 | 时间轴 | 事件与意义 |
| --- | --- | --- |
| **大寂灭** | 约 30 000 年前 | 多个古文明（先驱者联盟、永序议庭、灵升共同体等）因未知原因同时毁灭，留下大量断裂星门、自动防御系统和灵脉蓝图。 |

| **联邦成立** | 约 1500 年前 | 星海摇篮区的多个种族在经历战争与合作后，组建“星海联邦”，确立“开拓—共存—传承”三大信条。 |
| **当今时代** | 联邦历 1487 年 | 第二星海时代中期，各文明割据并持续扩张，古文明遗产、新生命形态和天灾事件频繁出现。 |

## 宇宙地理与星域

**稳定区（摇篮区）：** 星门密集、以太流平稳，适合孕育文明。每个摇篮区通常诞生多个原生种族和文明圈，例如联邦摇篮区、永恒环带、深渊裂谷、双旋走廊等。

**过渡区：** 航路不稳定、资源丰富，是远征军和开拓公司主要活动的区域。

**暗区：** 以太干扰强，航行与通讯极其困难，大部分区域仍未探索。

**星门：** 古文明遗留下的超空间航道设备，可以将数月航程压缩到几小时。现代文明只能修复和复制小型星门，故此星门枢纽成为战略要地。

**群岛式版图：** 银河的文明疆域并非连续，而是由星门串联的星系群组成。每个文明圈在地图上如同岛屿，周围是广阔的黑暗海洋和未测区域。

## 文明圈与主要势力

### 星海联邦文明圈

**理念：** 开拓未知、文明共存、遗产共享。

**核心星域：** 星海摇篮区——数十个宜居恒星系通过星门连成网状，被称为“摇篮”。

**主体种族：** 人类、狐人、龙裔、星灵、晶族、潮汐族及其混血后裔。

**主要国家与组织：** 星海联邦（多种族议会制国家）、龙庭帝国（以龙裔为主体的军事贵族国）、晶环学邦（以晶族和星灵组成的科研共同体）。

### 永恒环带文明圈

**理念：** 稳定与秩序，认为过度扩张会重演大寂灭。限制危险科技，优先维护自身的恒星生态圈。

**原生种族：** 苍裔（寿命长、情绪波动小）、守林者（生态型群体）、长歌者（语言共鸣型）。后天改造种族有守序者（基因优化后作为管理者）。

**主要政体：** 静滞联盟（秩序主义大国）、守望王庭（传统王政）、永序修会（宗教国家）。

### 深渊裂谷文明圈

**理念：** 进化与超越，主张主动改造灵核并探索深层以太。部分极端派认为应放弃传统生命形态。

**原生种族：** 夜民（超强感知）、灰裔（动力型强）、渊行者（空间异常适应）。改造种族有深裔（深潜议会标志性种群）。

**主要政体：** 深潜议会（改造主义大国）、灵升圣庭（极端进化派）、新生派联盟（温和改革派）。

### 商盟文明圈

**起源：** 各大文明早期授权的开拓公司在远征中脱离母国，最终形成商业协约体。主张契约与自由流通，认为文明的本质是交换。

**核心星域：** 双旋核心区——星门密集但无原生文明，被七大企业集团联合开发。

**主要种族：** 赫尼斯人（组织管理能力强）、风裔（导航专家）、岩民（工业专家），以及衍生的旅商族、航行者、碎星民等。

**权力结构：** 七大巨头董事会控制商盟——星门航运集团、深空资源财团、欧米伽工业联合、新地平线开拓集团、星穹金融联盟、遗产开发协会、天穹安保集团。

### 天灾势力与神秘力量

**共鸣虫群：** 一种文明级群居生命体，由母巢、工虫、战虫等组成，通过以太共享意识。兼具生物生命和以太网络的特性，数度侵袭已知星域。

**肃正协议：** 永序议庭遗留下的自动监管系统。检测到危险技术或以太灾害时，会无差别封锁星域并清除目标。现代文明既惧怕又利用其规则。

**失落星庭：** 只在遗迹记录中出现的神秘文明，疑为少数古文明幸存者，其真实存在与目的尚未确认。

**观星者、回响者、同调体** – 与古遗产有关的神秘种族，数量稀少，是遗迹研究重点对象。

## 科技体系概览

### 以太科学

现代科学框架将宇宙划分为物质层、能量层、信息层和以太层。以太层是能量与信息耦合的基础场，会受到意识影响。以太科学涉及共鸣学、以太场论、遗产学和深层以太学，是其他技术领域的根基。

### 能源与空间

**以太炉：** 现代文明核心能源设备，通过以太耦合从真空涨落中提取能量。分为固定式、舰船级、机甲级、战略级等等级。

**以太输能网络：** 取代传统电网的传输系统，城市和星舰通过以太脉管输送能量。

**跃迁技术：** 借助银河天然以太洋流进行超光速航行，类似顺流航行。不能任意方向跳跃，只能沿稳定航道进入、顺流、退出。

**星门：** 古文明制造的超空间通道，可将数月航程压缩为数小时。现代文明只能修复或复制小型星门，故星门枢纽成为战略关键。

**空间锚：** 用于稳定跃迁出口、固定航道和防止空间塌缩的装置，是远征舰队航行必备设施。

### 信息与工业

**共鸣网络：** 以太通讯网络，取代传统电磁网络。以太信号传播速度受以太流稳定度影响，因此远程通信存在延迟，需要中继站。

**AI ****等级：** 联邦将人工智能划分为 L1–L4，禁止发展自主文明级 AI（大寂灭时代可能出现过相关灾难）。

**纳米制造与轨道工业环：** 联邦具有成熟的纳米制造技术，可在轨道工业环批量生产战舰、机甲、城市模块。

### 生命与殖民

**灵核医学：** 结合生物工程和以太科技，平均寿命可达 150–200 岁，可修复器官和神经，但不能实现永生。

**行星塑造（Terraforming）：**

一级：大气调节、水循环恢复（20–50 年）。

二级：气候重构、生态重建（百年）。

三级：彻底改变行星环境（数百年）。

**生态工程：** 在太空站或小行星上构建封闭生态圈，让远征队在荒芜环境生存。

## 灵核与灵脉体系

### 灵核（Soul Core）

**性质：** 位于脑干和脊髓之间的真实器官，是灵核—神经—以太三者耦合的中心。所有智慧生命生而具有灵核。

**品质（容量）：** 决定灵脉承载上限。以单位表示，一般人约 100 单位，精英 150 单位，天赋者 200 单位以上。

**成长：** 灵核在幼年期形成，青春期觉醒第一条灵脉，成熟期快速成长，四十岁后趋于稳定，百岁后逐渐衰退。

**先天性与种族差异：** 不同种族的灵核天生结构不同，但灵脉数量和功能不受种族限制。

### 灵脉（Vein）

灵脉是灵核内通过训练或技术形成的以太信息回路，不是天赋能力。灵脉具有可组合性、可设计性，其尺度和结构决定了功能、消耗和稳定性。

**容量占用：** 不同灵脉占用灵核容量不同。例如红外视觉占用约 5 单位，空间定位占 15 单位，战术推演占 40 单位，舰队级指挥架构可能占到 120 单位。

**功能类别：**

**感知类：** 红外光谱视觉、低光视觉、运动预测、敌意感知、空间映射等。

**认知类：** 并行思维、高速计算、记忆索引、逻辑强化、战术推演等。

**操控类：** 远程共鸣（控制无人机/设备）、设备同步、武器校准、护盾调节、动力增幅等。

**生理类：** 高重力适应、疼痛抑制、高速恢复、低温耐受、高温耐受等。

**特殊类：** 深层感知（接触深层以太）、共鸣隐匿、灵核伪装、遗产解码等。

**品质等级：** 灵脉结构可分 V1（粗糙）到 V5（遗产级），等级越高功能越强、能耗越低，但铭刻难度和成本越高。

**形成方式：**

**自然生成：** 长期特定行为导致灵核自发形成简单回路，效率极低。

**诱导生成：** 通过训练、药剂、神经反馈形成，军校普遍采用，3–5 年内形成稳定灵脉。

**铭刻（Inscription）：** 使用以太刻印舱等高精度设备，将经过模拟验证的灵脉结构直接写入灵核，是现代灵脉工程学的核心。

### 灵脉铭刻流程

**灵核扫描：** 使用灵核断层扫描仪建立精细的灵核三维模型，评估容量、已有结构和潜在冲突区。

**架构设计：** 灵脉工程师根据职业需求及个体特点，设计灵脉组合与细节结构，类似软件架构师。

**模拟预演：** 超级计算机模拟灵脉写入后的共鸣效应，检测风险、效率和兼容性。

**刻印执行：** 在以太刻印舱内，通过精确共鸣场引导灵核神经生长和以太回路重组，完成铭刻。过程可持续数小时到数周，视灵脉复杂度而定。

**固化训练：** 铭刻完成后需要持续的共鸣训练以稳定结构，防止灵脉崩解或互相干扰。

### 灵脉组合与职业

灵脉不是直接对应该有的职业，而是根据需求组合不同功能模块。例如：

| 职业/定位 | 常见灵脉组合 |
| --- | --- |
| **舰队指挥官** | 战术推演 + 空间定位 + 并行思维 + 敌意感知 + 通讯同步 |
| **驱动师** | 动力增幅 + 高温耐受 + 疼痛抑制 |
| **主炮官** | 武器校准 + 高速计算 + 手眼协同灵脉 |
| **机甲机师** | 动力增幅 + 武器校准 + 运动预测 + 空间映射 + 疼痛抑制 + 设备同步 |
| **科研人员** | 并行思维 + 记忆索引 + 高速计算 + 逻辑强化 |
| **改造科学家（深潜）** | 深层感知 + 遗产解码 + 共鸣隐匿 |

灵脉的结构和组合不仅决定功能，还影响能耗、精神负担、共鸣稳定性等，因此灵脉工程是一门高风险、高价值的技术。

### 角色特色

**成长性：** 她的力量不是来自某条“最强灵脉”，而是来自于罕见的灵脉兼容能力，可以通过不断学习和铭刻，构建出一套覆盖观测、认知、操控的灵脉架构。

**故事驱动：** 心悚听见古代文明的“回响”，这或许和先驱者遗产有关。她的成长旅程会牵涉深潜议会的灵脉改造、静滞联盟对危险技术的戒备以及商盟对遗产蓝图的觊觎。她必须在拓荒、遗产争夺和文明冲突中寻找自己真正的道路。

**家族与使命：** 沧月氏族在联邦历史上立下赫赫战功，但家族近年来逐渐式微。家族将最后的希望寄托在心悚身上，希望她重振家声，同时守护联邦的开拓精神。

## 结语

“以太纪元”的世界是一片广袤的星海，充满了未解之谜、破碎的遗产、相互冲突的文明和不断进化的生命。灵核与灵脉体系为个体提供了无限的成长空间，也让科技与修炼融为一体。主人公沧月心悚将作为联邦远征舰队的一员，带领读者探寻这一切的终极答案。
