# 术语与舰船表

## 舰船与设施

| 名称 | 类型 | 功能 |
| --- | --- | --- |
| 衡霜号 | 联邦指挥舰 / 主力舰 | 林照野所在舰，负责舰队级授权、火控闭环和撤离线指挥 |
| 辰砂号 | 护航舰 | 点防、低功率烧蚀、协助锚标外壳清理 |
| 白潮号 | 工程救援舰 | 封锁泡投放、锚标保护、空间锚外层工程支援 |
| 潮汐三十二 | 医疗撤离舰 | 接收灵核过载、接口倒灌和红级污染人员 |
| 长缨号 | 机甲投送舰 | 第一机甲队母舰，负责突入舱投送 |
| 三号临时锚标 | 撤离锚标 | 为第三批撤离船提供返航边界和锚定解证明 |
| 霜背旧锚站 | 废弃锚站 | 旧锚链噪声和残骸潜伏巢风险来源 |
| 霜背三号殖民穹顶 | 撤离对象 | 边境殖民穹顶，第三批撤离为前五章主要压力源 |
| B-19 旧维护仓残段 | 旧锚链相位校准仓残骸 | 四十二年前事故遗留，含旧式维护 AI 接口壳，可能成为虫群相位参照 |

## 虫群对象

| 名称 | 类型 | 写作口径 |
| --- | --- | --- |
| 侦采预铺群 | 侦察虫、孢云、休眠孢囊等组合 | 不是全面进攻，而是路径测量、过滤器污染和锚标试探 |
| 残骸附着体 | 漂巢壳 / 临时壳层 | 伪装成旧船坞残骸，被清理能束唤醒 |
| 破障战体 | 战虫平台 | 高威胁突防工具，不等于指挥核心 |
| 迁徙核心体 | 移动巢港 / 迁徙统合层承载 | 不能按普通高威胁平台直接斩首 |
| 战区节律节点 | 局部指挥层 | 控制破障战体、工虫推进带和孢云节奏 |
| 幼巢舱 | 分巢风险平台 | 战略风险高，禁止直接接触，优先标记封存 |
| 分巢核搬运碎片 | 疑似幼巢/分巢相关移动残骸 | 小尺度高风险目标，禁止追逐和接触，优先坐标标记与封锁 |

## 流程术语

| 术语 | 含义 |
| --- | --- |
| 冷通道 | 低带宽、低风险的共鸣信息接入通道 |
| 封锁泡 | 工程舰投放的局部隔离屏障，可罩住锚标、拦截孢体和残屑 |
| 护盾短窗 | 舰船为射击或投送短暂打开局部护盾的时间窗口 |
| 返航证明 | 撤离船进入下一跃迁窗口所需的锚定证明链 |
| 红级污染战场 | 反虫群高污染等级战场，需要医疗、审计、封存和公众回放管制 |
| 医疗伴随口述 | 医疗监督下记录的主观残留内容 | 可辅助复核，但不得单独作为火控或战术证据 |
| 旧锚链噪声槽 | 断裂锚链在以太层留下的低频导向残响 | 文明航行视为不可靠航迹，虫群可能将其当作迁徙导向和巢场校准线 |
