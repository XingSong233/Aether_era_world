# 星海地理简易星图

本目录是按 `90_协作工作台/10_设定推导与蓝图/星海地理与星图节点设计蓝图.md` 重建的简易 Three.js 星图原型，不沿用已废弃的旧 `星海拓扑模拟器/` 程序目录、旧界面、旧依赖或旧样例数据。

## 文件

- `index.html`：浏览器入口。
- `styles.css`：深色调扁平 HUD 界面样式。
- `src/star-map-data.js`：按蓝图坐标范围、文明节点数量、显示层级、风险等级和节点字段生成样本数据。
- `src/main.js`：Three.js 渲染、正交相机、有限平移与小范围旋转 OrbitControls、Bloom 后处理、白色稀疏沸腾粒子点阵、晶体节点辉光、投影环与高度柱光效、光标准星、节点筛选和节点档案交互。

## 口径

- 当前样本生成 `450` 个主星图显示节点：星海联邦 `110`、永恒环带 `90`、深渊裂谷 `90`、商盟 `110`、中立/争议/遗产/暗区合计 `50`。
- 当前视图采用 `2.5D` 分布：`x/y` 作为主平面坐标，`z` 归一化为节点悬浮高度，并通过地面投影环、垂直高度柱、有限范围视角平移和小范围旋转显示高差；不再把节点展开为完整三维空间云。
- 当前版本只绘制节点，不绘制航线边；节点放弃平面辉光盘，改用透明菱形晶体、线框、内部核心、同形体边缘辉光、发光地面投影环和渐隐高度柱，以便观察文明分布、层级和风险。
- 背景粒子层采用深色背景下的白色稀疏沸腾水面式点阵：远看是更大尺度的随机起伏，近看是连续泡升、反弹和环形涟漪；不使用真实水面网格或水面材质，也不表现气泡爆裂过程。
- 当前采用深色扁平 HUD 视图，以便保留粒子、辉光、光标和 2.5D 高差的可读性；此前白色调界面参考 `90_协作工作台/30_素材与导出/图片素材/参考素材/1268.png` 的工程设定板语言，后续可按展示需要切回浅底版本。
- 画布内有轻量光标准星和短读数层，用于辅助鼠标定位和节点悬停观察；移动端保留静态 HUD 布局，不依赖光标元素。
- 节点名称统一为“样本”编号，不是正式采用的具名恒星系。
- 坐标、分布和层级字段用于验证地图展示口径；正式节点稳定后，应另建结构化数据文件。
