# 航行自由跃迁与开拓秩序

基准：《以太纪元世界观总纲》、《航行自由跃迁与开拓秩序设计蓝图》、《关键科技与生产关系状态表》、《经济体系》、《生产关系与职业结构》、《深层以太设定》、《文明圈对照表》、《星际定位技术原理》、《星海地理与星图节点》

本页是“以太纪元”航行、自由跃迁与开拓秩序的正式专题入口，用于说明星海社会如何组织星际交通、航权、星门窗口、空间锚、自由跃迁、边境救援、新航线测绘和殖民地承认。

以太纪元的星海不是可以任意穿越的连续平原。稳定文明圈依靠星门、以太洋流、空间锚和共鸣中继构成群岛式交通网络；自由跃迁扩展了舰船离开固定航道的能力，但它仍然受能源、航图、锚定、安检、保险、深层扰动和事故责任限制。开拓也不是插旗即主权，而是从测绘、作业、锚站、生态改造、定居到自治承认的长期秩序。

这里的“航图”需要分层理解：天体基准定位和物理星图负责确认舰船在物质星海中的坐标；以太航图和通行航图负责判断某条路线能否跃迁、能否救援、能否被制度承认。远征队即使在过渡区或暗区失去稳定共鸣中继，也通常不会完全失去空间坐标；真正稀缺的是可复测、可承保、可救援的航行条件。

天体基准定位的具体技术原理见：《星际定位技术原理》。主星图、文明疆域尺度、核心节点显示规则和跨星系航行时间基准见：《星海地理与星图节点》。

## 一、基本口径

| 口径 | 设定说明 | 社会后果 |
| --- | --- | --- |
| 星门仍是秩序骨架 | 古文明星门可把长距离航程压缩到稳定窗口内，但遗留星门普遍存在损伤、失准、污染残留或网络身份丢失；现代文明主要能修复、校准和接入遗留星门，低规格小型星门的复原或短支线扩建需要遗产核心、空间锚和枢纽能源网支持。 | 星门枢纽仍是军事、贸易、移民、金融和外交中心；未修复或未接入网络的星门只是高价值遗迹，不是公共交通设施。 |
| 星门通行依赖网络接入 | 单个星门不能凭空通往任意地点，必须与至少另一个可用星门完成网络握手、坐标校验、窗口同步和风险复核。 | 文明必须先用其他高成本航行方式发现、修复并接入两端星门，才能把两地转化为快速交通节点。 |
| 以太洋流塑造航道 | 天然以太洋流提供低成本超光速航行条件，但必须沿稳定航道进入、航行和退出。 | 航路测绘、潮汐窗口、空间锚维护和航线保险成为常规产业。 |
| 空间锚维持出口 | 空间锚用于稳定跃迁出口、固定航道、校准星门和维持锚链航行。 | 锚站维护者常掌握边境通行权，锚站失准会让殖民地孤立。 |
| 自由跃迁有限成立 | 自由跃迁不依赖固定星门或稳定以太洋流，而是由星舰在任务窗口内生成短时、一次性的以太航迹；它仍依赖舰载跃迁系统、锚定解、跃迁窗口、能源额度和航图。 | 它削弱部分节点垄断，却不能让小团体无视星门、航权、安检和救援链。 |
| 物理定位仍然成立 | 舰船可通过基准星表、背景星场、多谱段观测、脉冲源和天体运动模型确认自身物质层坐标。 | 远征队、科考队和探险队能绘制一定精度的物理星图，但这不等于掌握可通行航路。 |
| 航行成本已大众化 | 稳定区内的星门客运和成熟航线已进入大众商业交通成本带。 | 普通人可以跨星域求学、工作、探亲或迁徙，但仍受排期、补贴、检疫和目的地接纳限制。 |
| 边境不是无主空白 | 过渡区和暗区有资源、遗产、生态风险、旧灾害和航道不稳定性。 | 开拓必须处理测绘、样本、救援、基础服务权、居民自治和文明承认。 |
| 航行记录可追责 | 共鸣信标、跃迁黑匣、维护记录和航迹残留共同支持事故调查。 | 走私和非法跃迁可以规避部分记录，但会失去正规保险、医疗和救援。 |

## 二、星海地理与交通结构

### 稳定区

稳定区以星门枢纽、成熟以太洋流、密集共鸣中继和多级星港为核心。多数公民的跨星域生活发生在这里：客运、物流、教育、医疗转诊、旅游、职业流动和普通贸易都依赖公开航班、标准检疫和公共补贴。

稳定区的矛盾不在于能不能旅行，而在于窗口、速度、目的地资格和费用分层。普通旅客可以等待补贴航班；高端医疗、紧急合同和军用调动会购买或征用更高优先级窗口。

### 过渡区

过渡区航路不稳定，但资源、殖民空间和遗产线索更多。这里的交通通常依赖星门边缘节点、锚链航行、临时锚标、自由跃迁和护航舰队。过渡区常见灰色港、远征基地、企业开拓站和混合管辖星港。

过渡区的社会问题是机会与风险同时放大。基础供给可以建立，但高质量能源额度、医疗复健、职业训练、航线保险和事故复审不足，容易形成权限贫困和维护贫困。

### 暗区

暗区以太干扰强，通讯困难，以太航图和通行航图快速失效，深层以太、虫群、遗产和肃正协议风险更高。暗区航行通常不以高速通行为目标，而以低速、多段、强隔离和可撤离为目标。

暗区不是个人冒险者的常规舞台。合规行动通常需要舰队或基地指挥、深层以太探查队、灵脉工程师、医疗心理团队、法务伦理监管和遗产隔离技术员协同。

## 三、主要航行方式

| 航行方式 | 主要用途 | 优势 | 限制 |
| --- | --- | --- | --- |
| 星系内航行 | 行星、轨道环、船坞、星港和近星资源点之间。 | 成熟、可监管、适合日常交通和工业物流。 | 惯性控制不存在，高速机动仍需要结构强度、缓冲系统和航行纪律。 |
| 星门通行 | 稳定区大众客运、大宗物流、军队调动和跨文明外交。 | 容量大、时间可预期、保险成熟，通道由固定设施长期校准。 | 依赖星门窗口、枢纽政治、检疫、安检和古文明设施维护。 |
| 以太洋流航行 | 稳定区与过渡区贸易、远程补给和低成本航运。 | 成本低于自由跃迁，可支撑商运网络。 | 不能任意转向，受以太潮汐、深层扰动和空间锚状态影响。 |
| 锚链航行 | 新航线建设、边境殖民地、暗区边缘物流。 | 能把临时坐标逐渐转化为可维护航路。 | 空间锚建设和维护成本高，锚链失效会切断救援。 |
| 自由跃迁 | 紧急救援、高价值运输、军事机动、勘探测绘和绕开拥堵节点。 | 不必经过固定星门或稳定洋流主干，星舰可临时计算一次性出口。 | 能源、锚定解、冷却复检、出口误差、航迹追踪和管制压力都更高。 |
| 暗区潜航 | 遗产点、污染区、虫群边缘和失联殖民地。 | 能进入普通航线无法覆盖的区域。 | 通讯弱、污染高、人格回响和接口倒灌风险上升。 |

这些方式常常组合使用。一支边境远征队可能先通过星门抵达稳定区边缘，再沿以太洋流进入过渡区，投放临时锚标后执行短程自由跃迁，最后以暗区潜航方式靠近目标遗迹。

### 星门与自由跃迁的工程差异

星门通行和自由跃迁都能压缩星际距离，但工程逻辑相反。星门是固定基础设施先把两端稳定场域长期锁定成可重复通行的通道，再让舰船通过；自由跃迁是星舰在一次任务窗口内，用跃迁龙骨、以太炉、导航阵列和锚定解暂时包裹自身，生成随船消散的一次性以太航迹。

现代文明能够修复、校准、接入遗留星门，也能在遗产核心、低规格蓝图、可复用关键组件、空间锚网络和枢纽能源网支持下，完成小型星门的受限复原、改造或短支线扩建。但这不等于完整掌握星门制造；现代文明尚不能从零制造古文明级主干星门网络，也不能把星门技术缩小成个人、小艇或普通货柜级传送装置。

### 星门修复与网络接入

古代遗留星门通常不能在发现后直接启用。门体结构、能源接口、锚定阵列、网络身份、古代权限层、污染隔离和通道风险记录都需要分阶段检查。未经修复的星门可以成为遗产样本、禁航点、远征目标或军事争夺对象，但不能承担稳定客运、货运或舰队调动。

星门的可用性至少经过四层确认。

| 层级 | 核心问题 | 未完成后果 |
| --- | --- | --- |
| 结构修复 | 门体、遗产核心、能源接口和锚定阵列是否能稳定承压。 | 通行时门体失准、通道崩解或枢纽停摆。 |
| 本地校准 | 星门所在星系的空间锚、重力环境、以太场和星港调度是否可接入。 | 出入口漂移、窗口误判或污染回流。 |
| 网络握手 | 目标星门是否可识别、可授权、可同步窗口并完成双端校验。 | 单端星门无法形成稳定通道，只能作为待接入节点。 |
| 风险复核 | 通道中是否存在深层回流、肃正协议标记、虫群扰动或古代封锁。 | 航线不能进入公共排程，只能作为高危任务或封存对象。 |

### 星门网络分片与层级

古代星门网络由多个网络域构成，不是单一全银河公共网络。古代存在数个独立文明，每个文明拥有各自的私有星门网络；这些私网服务于本页明内部的殖民、军用、资源、迁徙和维护秩序。跨文明通行则依赖专门的跨文明星门网络，由相关古代文明共同构建、维护和制定使用规则。

跨文明星门网络与各文明私有网络存在物理隔离，并非只用权限系统限制通行。它们可以共享部分星区、星港或维护设施，但星门通道、枢纽承压结构、网络身份和中继链不直接混同。现代文明修复一个古代私网星门，最多说明它接入了某个私网残片；若没有对应跨文明网络接口、桥接枢纽或物理中继链，就不能把它当成通往所有古文明网络的入口。

各文明私有星门网络内部也存在层级。低等级星门通常只允许直接联通一定区域内的同网星门；中等级星门承担星区间中继；高等级星门和枢纽星门承担远距离跨区、主干调度和高流量通行。一次从低等级星门发起的远距离通行，表面上可以由权限和调度系统自动完成，工程上却可能经过多个中等级、高等级或枢纽星门中继，逐段完成窗口同步、锚定转发、通道承压和风险复核。

| 网络或层级 | 古代功能 | 现代修复后果 |
| --- | --- | --- |
| 文明私有星门网络 | 单一古文明内部的殖民、资源、军用和行政交通。 | 可恢复局部私网交通，但不自动获得跨文明通行能力。 |
| 跨文明星门网络 | 多个古文明共同建设和维护的外交、贸易、迁徙或协防通道。 | 需要识别共同协议、桥接枢纽和物理隔离边界，审查等级更高。 |
| 低等级星门 | 区域内短程或低流量通行。 | 适合恢复本地支线，不能直接承担超远距离主干交通。 |
| 中等级星门 | 星区内外中继和支线汇流。 | 常成为现代边境枢纽、军管站或商运节点。 |
| 高等级/枢纽星门 | 主干中继、远距离通行和大流量调度。 | 战略价值极高，也最容易触发肃正协议、遗产权限和跨文明争夺。 |

因此，星门网络扩张不是在地图上点亮一个遗迹，而是先通过自由跃迁、以太洋流、锚链航行、慢速远征或其他高成本方式跨越区域，找到第二端或多端星门，再完成修复、校准和接入。一个文明可以先知道某处存在星门，却因为无法安全抵达、无法修复、无法完成网络握手或无法通过风险复核，而长期不能把它纳入常规交通。

自由跃迁不需要两端星门，但它也不是便携星门。它不形成可长期停留、可排队复用的公共通道，只是在一次跃迁中让一艘舰船或小编队承担结构应力、能源消耗、出口误差、航迹暴露和复检责任。

| 对比项 | 星门通行 | 自由跃迁 |
| --- | --- | --- |
| 工程主体 | 星门门体、遗产核心、枢纽能源网、空间锚和星港调度共同维持。 | 星舰自身的跃迁龙骨、以太炉、导航阵列、灵算核心和舰载灵械共同承担。 |
| 通道性质 | 双端长期校准，可重复通行，可纳入公共排程。 | 短时生成，一次性使用，跃迁后只留下航迹残留。 |
| 坐标与航图 | 目的地星门已接入网络，出口由固定设施和调度系统预先校准。 | 舰船必须把物理星图、即时观测、以太航图和通行条件综合成锚定解。 |
| 能源与维护 | 依赖枢纽级能源、星门维护、空间锚校准和检疫安检体系。 | 依赖舰载能源额度、跃迁系统适航、冷却复检和目的地接纳。 |
| 船体负担 | 舰船通过稳定通道，结构压力较低，适合大客流和大宗物流。 | 舰体承受跃迁剪切和出口扰动，连续短跳会积累隐伤。 |
| 失败形态 | 星门失准、通道封锁、枢纽停摆、错窗通行或遗产核心异常。 | 出口偏移、舰体裂伤、人员神经冲击、误入扰动区或失联。 |
| 交通地位 | 固定主干线，决定星门枢纽、贸易走廊和军事投送。 | 从已知锚点向外扩展的高成本机动能力，适合救援、军用、急件和勘探。 |

因此，星门解决的是“把一条固定通道长期维护成公共基础设施”，自由跃迁解决的是“在没有固定通道或需要越过拥堵节点时，由舰船临时承担一次高风险跨越”。两者可以互补，但不能互相替代。

## 四、自由跃迁

自由跃迁是星舰在舰载跃迁系统、以太炉、跃迁龙骨、导航阵列和锚定解共同作用下，短时切入非固定航段并在另一个稳定场域退出的航行方式。它不需要固定星门，也不要求沿完整以太洋流主干航行，但它仍需要可计算的场域条件。

### 关键要素

| 要素 | 说明 | 失效后果 |
| --- | --- | --- |
| 跃迁龙骨 | 分配跃迁剪切、承受舰体应力并维持舰内以太场稳定的主结构。 | 舰体裂伤、局部失压、接口烧蚀、人员神经冲击。 |
| 锚定解 | 由导航员、灵算核心和航图系统共同计算的出口方案。 | 出口偏移、进入禁航区、撞入强扰动场或失联。 |
| 跃迁窗口 | 适合进入和退出自由跃迁的时间段与场域状态。 | 能耗暴涨、航迹污染、空间锚失准或被迫等待。 |
| 临时锚标 | 远征或救援时投放的短期导航与出口稳定装置。 | 舰队无法形成回程路线，后续救援困难。 |
| 航迹残留 | 跃迁后留在以太场中的可读扰动。 | 可能被追踪、复盘，也可能暴露秘密航线。 |
| 冷却复检 | 跃迁后对舰体、接口、导航、护盾和人员进行检查。 | 连续短跳会积累隐伤，黑市船常在这里出事故。 |

### 使用边界

自由跃迁不能在所有地方启动或退出。强重力梯度、星门拥堵区、巨构密集区、深层以太回流、虫群扰动、肃正协议警戒区和封存航道都会显著提高风险。稳定文明通常要求自由跃迁在申报窗口内进入，在指定空间锚附近退出，并保留黑匣和航迹记录。

在过渡区，自由跃迁是开拓工具；在稳定区，它是高价快速服务和紧急通道；在军事行动中，它是机动能力；在黑市中，它是走私和逃亡手段。不同用途的合法性取决于授权、锚定、保险、检疫和事故后复检。

### 等级与用途

| 类型 | 典型使用者 | 制度口径 |
| --- | --- | --- |
| 许可商业跃迁 | 客运公司、医疗转运、急件物流、高价值商队。 | 需要适航认证、航线保险、目的地锚定许可和污染筛查。 |
| 勘探跃迁 | 远征舰队、科研机构、开拓公司、商盟合约勘探队。 | 需要测绘授权、临时锚标、救援预案和航图上报。 |
| 军用跃迁 | 舰队、授权执法、灾害救援和反虫群部队。 | 可临时越过民用窗口，但必须接受军用审计、战后复检和污染隔离。 |
| 高危实验跃迁 | 深层以太站、裂谷实验派、遗产隔离项目。 | 通常进入高危任务、遗产隔离或环带封存审查。 |
| 黑市跃迁 | 走私者、非法采样队、逃亡者、灰色承包商。 | 失去正规保险与救援，事故后常牵连污染、伪黑匣和非法锚标。 |

## 五、航图、星舰与职业链

航行能力不是只有船和引擎。成熟星际交通依赖航图、测绘、锚站、调度、维护、保险、检疫、救援和事故复盘。

星际测绘通常分为四层。

| 层级 | 内容 | 作用 |
| --- | --- | --- |
| 基准星表 | 稳定恒星、星团、脉冲源、背景星场和长期运动模型。 | 为舰船提供物质层坐标参照。 |
| 即时观测 | 抵达后用光学、红外、射电和引力观测校准本地星空。 | 修正遮蔽、星体运动、引力透镜和遗产干扰造成的误差。 |
| 以太航图 | 以太潮汐、深层扰动、空间锚状态、临时锚标、污染区和航迹残留。 | 判断能否跃迁、能否退出、能否返航。 |
| 通行航图 | 航权、保险、检疫、救援承诺、目的地接纳和事故责任。 | 决定一条路线能否成为合法航线或开拓依据。 |

因此，一个坐标可以在物理星图上被标清楚，却仍然是航行上不可达、不可保或不可救援的风险点。航图垄断也常常发生在以太模型、锚定解和通行条件上，而不是单纯隐藏恒星坐标。

| 职业或机构 | 核心职责 | 社会位置 |
| --- | --- | --- |
| 跃迁领航员 | 计算锚定解，判断跃迁窗口，处理出口误差和紧急改道。 | 高端航运、舰队和远征队的关键职业。 |
| 天体测绘师 | 维护基准星表，校准背景星场、脉冲源和本地天体运动模型。 | 远征、科考、救援和开拓测绘的坐标基础。 |
| 航道测绘师 | 更新以太潮汐表、暗区边界、深层扰动和空间锚状态。 | 以太航图权、保险费率和开拓许可的基础。 |
| 空间锚工程师 | 建设、校准、维护和回收空间锚、临时锚标与锚链中继。 | 边境殖民地和新航线能否长期存在的关键。 |
| 星门枢纽调度员 | 分配星门窗口、泊位、检疫、安检和应急通道。 | 稳定区交通秩序和枢纽政治的日常执行者。 |
| 航线保险评估师 | 评估舰船、航线、货物、人员和样本风险。 | 把航行风险转化为可审查、可赔付、可争议的合约结构。 |
| 检疫与遗产隔离人员 | 检查人员、货物、样本、灵械和舰体以太残留。 | 阻止污染、虫群、遗产回响和非法军标模块进入稳定区。 |
| 边境救援队 | 负责失联搜索、锚站抢修、人员撤离和灾害封锁。 | 决定开拓秩序是否具有公共信用。 |
| 航权仲裁师 | 处理窗口违约、锚站事故、救援拒付和跨文明通行争议。 | 连接经济体系、生产关系和文明外交。 |

这些职业与能力系统相互嵌合。跃迁领航员常需要空间映射、风险建模、反馈压缩和接口带宽训练；空间锚工程师依赖结构诊断、污染识别和大型灵械接口；检疫人员需要以太场清洁、样本隔离和遗产隔离规格。

## 六、航权与星港治理

航权配额是星际交通中的组合性通行权利，包括星门窗口、航路进入、空间锚使用、泊位、检疫优先级、救援响应和能源补给。航权可以交易、租赁、补贴、征用或冻结，但不能越过军事、检疫、灾害预警和文明监管。

### 星港的核心功能

| 功能 | 说明 |
| --- | --- |
| 交通调度 | 管理星门窗口、航道排队、自由跃迁出口和星系内交通。 |
| 检疫安检 | 检查人员身份、舰体污染、货物样本、灵械规格和军标装备。 |
| 能源与维护 | 分配能源额度、检修跃迁龙骨、校准导航阵列和空间锚接口。 |
| 保险与仲裁 | 接入航线保险、事故记录、货损赔付和跨文明仲裁。 |
| 救援与封锁 | 在事故、虫群、深层扰动或肃正协议警戒时启动撤离和禁航。 |

星港不是普通车站，而是交通、能源、检疫、金融、军事和外交的交汇点。商盟星港常把这些功能拆成服务包和保险条款；联邦星港强调公共许可和救援链；环带星港更重视封存和长期安全；裂谷星港则常与实验站、改造医疗和深层探查设施相连。

## 七、开拓秩序

开拓秩序把未知星域从危险坐标转化为可居住、可维护、可审计、可救援和可自治的社会空间。合法开拓通常分为多个阶段。

| 阶段 | 权利内容 | 必要责任 |
| --- | --- | --- |
| 远程测绘 | 标注航道、资源、生态、遗产和危险信号。 | 上报重大风险，不能伪造或故意压低航线危险。 |
| 临时作业 | 进行短期采样、维修、科研、救援或补给。 | 保持撤离能力，隔离样本，限制生态扰动。 |
| 锚站建设 | 投放临时锚标、建设空间锚、中继和补给点。 | 维护锚站，公开必要安全参数，承担失准事故责任。 |
| 资源开发 | 建设矿区、生态穹顶、自动工厂、补给农场和工业站。 | 处理能源、污染、劳工、保险和样本处置。 |
| 生态改造 | 调整大气、水循环、土壤、微生物或居住环境。 | 接受长期评估，保护先存生命和共鸣生态。 |
| 定居承认 | 允许居民长期居住，建立基础服务和地方组织。 | 保障基础服务权、医疗教育、撤离方案和居民代表机制。 |
| 自治承认 | 新殖民地获得稳定地方治理权和外部关系资格。 | 接受航权责任、事故审计、跨文明协议和公民权保障。 |

自动殖民种子系统可以让机器人、生态包和自动工厂先行建设，但它只是开拓工具。没有测绘、生态评估、救援、基础服务、居民权利和文明承认，自动设施不能单独构成完整殖民法理。

若新星域存在智慧生命、疑似古文明封存、共鸣虫群生态、深层以太回响或肃正协议警戒，开拓权会被暂停或降级，相关地点转入接触规程、遗产隔离、封存审查或军事管制。

## 八、四文明差异

| 文明圈 | 航行秩序 | 自由跃迁 | 开拓秩序 | 主要矛盾 |
| --- | --- | --- | --- | --- |
| 星海联邦 | 公共航图、舰队救援、星门标准化、边境补贴和跨种族迁徙制度较完善。 | 允许商业、勘探和军用跃迁，但要求黑匣、复检、救援预案和污染隔离。 | 强调开拓许可、基础服务权、居民自治、遗产共享和公共审计。 | 前线希望快速授权，内星区要求审查、复检和救援链。 |
| 永恒环带 | 航道治理服从长期阈值、生态安全、封存和灾害预警。 | 高风险跃迁通常只用于防灾、撤离、守望和封存任务。 | 强调生态阈值、长期维护和封存权，主动限制激进殖民。 | 其他文明质疑其垄断航道安全解释权，环带则认为开放会扩大灾害。 |
| 深渊裂谷 | 暗区、深层扰动和极端航线被视为适应性试验场。 | 更愿意使用高危锚定、改造船员和实验跃迁。 | 允许实验共同体、改造个体和极端环境定居，容忍更高失败率。 | 进化权与污染扩散、人格损伤、实验劳工伦理冲突。 |
| 商盟 | 航权、窗口、泊位、空间锚、保险和救援被高度服务化、合约化。 | 商业化最成熟，提供包船、急件、护航、高价救援和风险外包。 | 倾向特许开拓、企业领、承包自治和债务融资。 | 市场扩展快，也易制造锚站垄断、债务殖民、合约弃救和灰色跃迁。 |

## 九、主要冲突

| 冲突 | 触发条件 | 表现 |
| --- | --- | --- |
| 星门窗口争夺 | 枢纽拥堵、战争、灾害、检疫或高峰航运。 | 普通旅客等待补贴航班，高价值货物和军用行动争夺优先级。 |
| 自由跃迁越权 | 救援、走私、逃亡、抢采或军事行动绕开航权。 | 争议焦点落在是否有锚定解、救援预案、黑匣和污染复检。 |
| 航图垄断 | 新航线数据被军方、企业或测绘机构独占。 | 边境殖民地依赖单一航图和保险供应商，形成长期平台依附。 |
| 锚站主权争议 | 临时锚标发展为补给站、军港、灰色港或殖民枢纽。 | 维护者、投资者、居民和母文明争夺实际通行控制权。 |
| 自动殖民先占 | 自动殖民种子抢先建设设施。 | 设施存在不等于合法产权，生态评估和居民权利可能推翻企业主张。 |
| 救援承诺破裂 | 边境事故成本过高或样本风险超出保险。 | 保险方、承包方、舰队、居民和监管机构围绕弃救责任争执。 |
| 黑市跃迁链 | 非法锚标、伪黑匣和解除限制的旧船进入航运。 | 短期绕开管制，长期带来污染、失联、诈骗和事故追责断裂。 |
| 开拓自治冲突 | 殖民地从企业前哨成长为稳定社会。 | 居民要求自治，企业保留航权、能源、医疗和债务控制。 |
| 高危发现封锁 | 航路发现遗产、深层回响、虫群活动或肃正协议警戒。 | 联邦受控研究、环带封存、裂谷继续接触、商盟确认采样权之间发生冲突。 |
