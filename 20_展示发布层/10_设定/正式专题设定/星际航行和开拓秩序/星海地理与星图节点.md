# 星海地理与星图节点

基准：《以太纪元世界观总纲》、《星海地理与星图节点设计蓝图》、《星海地理理论层级整理》、《航行自由跃迁与开拓秩序》、《星际定位技术原理》、《文明圈对照表》、《深层以太设定》

本页是“以太纪元”星海地理、文明疆域尺度和主星图显示节点的正式专题入口。它用于说明四文明处在银河中的哪一类区域、主星图应显示哪些核心星系、文明实际控制范围与公共星图节点之间如何分层，以及跨星系航行时间应如何估算。

本专题不替代航行、自由跃迁、空间锚和开拓秩序专题。航行制度、航权和殖民承认见 《航行自由跃迁与开拓秩序》；星际定位、物理星图和测绘误差见 《星际定位技术原理》。本专题只冻结宏观星海尺度、文明分布和地图节点口径。

在本专题的规则层之下，若要推导后世四大文明的前身共同体在刚步入星际时代时如何扩张，参考 《前身文明扩张策略推导》；该文件不作为正式地图节点表。

当前星海地理规则也可按“公理、定义、引理、定理、推论”的层级辅助推导，整理见 《星海地理理论层级整理》。其中稳定采用的关键引理包括“中央控制衰减引理”：刚步入星际时代的文明若航图、锚链、共鸣中继、星门修复、救援、事故审计和跨组织仲裁尚未成熟，中央政权对边境地区的有效控制力会随着综合距离增加而显著衰减。这里的距离不只是光年，还包括航行距离、通信距离、维护距离和制度距离。该引理用于解释早期边境势力为何容易私自试验遗产、军事化星门、隐瞒航图风险或把空间锚和船坞转化为地方权力，直到事故外溢后才被核心区发现。

## 一、基本口径

| 口径 | 设定说明 | 写作后果 |
| --- | --- | --- |
| 银河是正常大型螺旋星系 | 银河整体可按棒旋星系或多悬臂螺旋星系理解，银心存在超大质量黑洞。 | 银心、主悬臂、支臂、悬臂间隙和稀疏星区都可以成为远期扩展方向。 |
| 当前主星图不是整片银河 | 当前四文明活动区只是某条主悬臂外缘与一条支臂交汇处的一段文明星海。 | 不需要在主星图中画出完整银河，也不需要把银心黑洞放进当前地图。 |
| 文明版图是群岛式 | 星门、以太洋流、空间锚、暗区和深层扰动共同决定可达性。 | 文明疆域不应画成均匀实心色块，而应表现为核心星区、航道、节点和影响带。 |
| 古代星海更连通 | 古文明时代的深层以太区远没有现代广泛，稳定区和过渡区占比更高，更像大陆上分布着地中海、湖泊和河流，而不是现代“大海中的群岛”。 | 古代星门、锚链和遗产设施可以形成远比现代顺畅的网络；现代文明面对的是灾后群岛化、断裂化后的星海。 |
| 主星图显示核心节点 | 星图上的点是具名核心恒星系、星门枢纽、星区首府、空间锚、港口、前哨或事件地点。 | 一个地图节点不等于一个普通恒星系，也不代表文明实际恒星系总量。 |
| 实际疆域保留更大背景 | 四文明实际定居、登记、保护、开拓或影响的恒星系数量远多于地图显示节点。 | 后续剧情可以持续新增星系，不会破坏既有地图尺度。 |
| 航行时间按航路等级估算 | 距离只是基础变量，星门窗口、锚链、以太洋流、自由跃迁、排期、复检和许可都会改变实际耗时。 | 不应把“相邻恒星系”固定写成统一时间，也不应让自由跃迁抹平所有交通差异。 |

### 星门网络与深层以太区规则

星门不是被发现后即可直接使用的天然传送门。古代遗留星门普遍存在不同程度损伤，常见问题包括门体结构断裂、锚定失准、能源接口失效、网络身份丢失、污染残留、古代权限锁或肃正协议警戒标记。未经修复和校准的星门只能作为遗迹、危险节点、导航异常点或待审查基础设施存在，不能直接承担公共交通。

星门也必须接入星门网络才能形成稳定通道。单个星门不能把舰船送往任意地点；它需要至少与另一个可用星门完成网络握手、身份校验、坐标锚定、窗口同步和通道风险复核，才能实现可重复通行。现代文明掌握的是识别、修复、校准和局部接入古代星门网络的技术，不等于完整掌握古文明级星门网络建造能力。

古代星门网络不是单一的全银河公共网络。古代存在数个独立文明，每个文明都有各自的私有星门网络；跨文明通行则依赖各古代文明共同构建、共同维护的跨文明星门网络，其使用规则由参与文明共同商议。跨文明星门网络与各文明私有网络存在物理隔离，不是只靠权限控制阻止任意通行；因此，修复某个文明私网中的星门，并不等于可以直接接入其他古文明私网或跨文明公约网。

各文明私有星门网络也不是完全分布式结构。星门存在层级：低等级星门通常只允许直接连通一定区域内的同网星门；跨越更远距离时，需要通过中等级、高等级星门或枢纽星门中继。一次看似从低等级星门发起的远距离传送，实际可能经过数个中继星门完成窗口同步、锚定转发和通道承压，再抵达另一个低、中或高等级星门。这个中继流程可以由权限、调度和风险复核控制，但其成立前提仍然是对应层级星门、网络域和中继链物理存在且可用。

因此，星门交通的本质不是发现一个点，而是打通一条可维护的边。文明必须先依靠自由跃迁、以太洋流、锚链航行、慢速远征或其他高成本方式抵达另一个区域，发现并修复当地星门，再把两端并入同一星门网络。只有当两个非深层以太高扰动区域都拥有已修复星门，且两端通过网络接入和风险复核后，它们之间才会出现稳定快速交通。

深层以太区不是物质意义上的海洋、空洞或不可穿越墙面。“海”只是一种地图隐喻。深层以太区的恒星、行星、小行星带、宜居星球、古代遗迹、星门、空间锚、矿产和普通物质环境可以与其他区域没有本质区别；真正危险的是以太场强、信号密度、反馈速度、污染概率、深层回流和灵核、灵械、共鸣网络、AI 载体、数字人格稳定性受到的持续压力。

深层以太区也可能存在以太洋流和古代文明遗留锚链。古代文明可能曾经利用空间锚、锚链、星门和封存设施穿越、固定、隔离或研究这些高扰动区域。现代文明可以发现这些设施残段，但它们往往存在断裂、漂移、失准、污染、权限锁和古代风险标记。修复深层区星门或锚链不是单纯工程任务，而是航行测绘、遗产审查、污染隔离、救援预案和文明监管共同参与的高危任务。

深层以太区不是一成不变的固定边界。它会随着以太潮汐、深层回流、恒星活动、古代设施启停、肃正协议残留、虫群活动和文明工程干预缓慢变动。常规变动的时间尺度通常以数十年、数百年甚至更长周期计算，不会像天气一样每天重画星图；但对星门校准、锚链维护、殖民地安全、航线保险和长期定居资格来说，这种缓慢漂移已经足以改变历史。

深层以太区也不是随机生成的噪声斑块。它们遵从一定地理和历史规律形成：恒星密集带、悬臂断裂、强引力结构、古代星门和锚链残网、深层以太回流口、大寂灭灾害残留、肃正协议封锁带、虫群长期活动区和文明长期清洁或封存工程，都会改变局部以太环境。公共星图上的深层区边界是这些因素长期叠加后的结果，通常表现为裂隙、带状走廊、封存环、节点周边晕区、旧航道断裂带或被稳定工程切开的局部窗口，而不是均匀随机散布。

当前深层以太区中的古文明遗迹、星门、空间锚、锚链、封存站和殖民设施，大多数不应默认理解为古代文明主动建在深层危险区里。更常见的情况是：它们在古代运行时位于稳定区、过渡区或经过长期维护的局部稳定窗口，后来因大寂灭灾害、星门网络崩解、锚链断裂、封锁设施失控、深层回流、虫群活动或长期漂移而被卷入现代深层以太区。少数遗迹可以本来就是古代深层观测站、封存站、实验站或隔离枢纽，但这类设施应标明其深层建设目的和防护体系。

古文明时代的宏观以太地理也不应按现代群岛式星海倒推。那一时代的深层以太区远没有现在广泛，高压深层区更像大陆内部的地中海、湖泊、河流、峡湾和局部险滩：它们会塑造航路、边境和禁限区，但不会像现代这样把大部分星海切割成孤立岛链。大寂灭、星门网络崩解、锚链断裂、肃正协议封锁、深层回流扩张和后续长期漂移，才使现代星海呈现“深层以太大海中的文明群岛”格局。因此，古代文明拥有更连通的星门网络和更广泛的设施布局是合理的；现代文明只能在其残骸上修复少量边、稳定局部窗口，并不等于能恢复古代整体地理条件。

现代文明也并非无法影响或改造深层以太区。以太场清洁、屏蔽、空间锚、锚链稳定、星门复检、共鸣中继、隔离穹顶、封存阵列和长期观测站都能改变局部环境，使某些航道、设施圈、殖民地或星门枢纽从不可用变成短时可用、从高危变成可监管、从失准变成可复核。限制在于成本高、周期长、维护负担重，并可能把扰动转移到邻近区域；因此文明通常只能稳定局部通道和节点，不能轻易把整片深层以太区改造成普通稳定区。

这组规则意味着星海地理应表现为“稳定场域、深层高扰动区、可修复星门、天然洋流、古代锚链和现代航权秩序”共同塑造的群岛式网络。深层以太区不是没有价值的空白地带，反而可能拥有宜居世界、遗产节点和高价值航道；但其治理成本、维护成本和事故代价远高于稳定区。

当前主星图建议范围为约 `6500 x 4200 x 800` 光年。常规文明节点大多位于银河盘面上下 `250` 光年内；深层异常、暗区边缘、古文明遗产点和裂谷前哨可以扩展到盘面上下约 `800` 光年。

银心黑洞不在当前主星图中。它可以在地图边缘以方向标记呈现，例如“银心方向：约 18000 光年外”。银心方向可作为远期设定接口，用于古文明核心遗迹、肃正协议源头、禁限航区或深层以太异常高密度区。

## 二、整体星海尺度

| 层级 | 建议规模 | 说明 |
| --- | ---: | --- |
| 当前主星图显示范围 | 约 6500 光年宽、4200 光年高、800 光年厚 | 用于公共星海图、静态地图和创作时的尺度参照。 |
| 已知文明活动星域 | 直径约 5000-7000 光年 | 不是连续占领区，而是航道、核心星区、边境斑块和高危隔离区交织而成。 |
| 四文明合计稳定定居恒星系 | 约 15000-25000 个 | 包括成熟殖民地、自治星系、核心生态星系、贸易自治港和长期实验定居点。 |
| 有正式航图和救援备案的恒星系 | 约 80000-120000 个 | 不一定常住人口多，但可被定位、登记、承保、救援或限制通行。 |
| 暗区、深层边缘和遗产隔离区 | 数量不稳定，远多于显示节点 | 可作为后续剧情、探查任务和地图扩展的主要来源。 |

这个尺度下，星海足够大，边境和暗区仍然遥远；但核心文明之间仍然能发生贸易、战争、移民、外交和连续事件，不会变成彼此完全隔绝的孤岛。

## 三、四文明疆域与显示节点

主星图建议初始显示约 `440-460` 个节点。其中四文明各显示约百个核心节点，中立、争议、暗区和遗产相关节点显示 `40-60` 个。这个数量适合公共星海图和静态地图表达，也足以表现文明格局。

| 文明或区域 | 主星图显示节点 | 实际成熟定居恒星系 | 其他登记或影响范围 | 疆域形态 |
| --- | ---: | ---: | ---: | --- |
| 星海联邦 | 100-120 个，建议初始 110 个 | 约 3500-5000 个 | 边境殖民地、军管站、科研站和资源站约 8000-12000 个；航图登记恒星系约 50000-70000 个 | 最大连续片区，核心密集，边境向外伸出多条开拓臂。 |
| 永恒环带 | 80-100 个，建议初始 90 个 | 约 1200-2200 个 | 高度维护的生态核心约 150-250 个；封存、观察和限制开发恒星系约 5000-9000 个 | 沿稳定星群和生态带展开的弧带或环带，不是均匀圆形领土。 |
| 深渊裂谷 | 80-100 个，建议初始 90 个 | 约 700-1300 个 | 实验站、改造殖民地和深潜前哨约 2500-5000 个；高风险登记点约 20000-40000 个 | 沿深层以太异常带、暗区边缘和危险航道分布，稀疏但纵深大。 |
| 商盟 | 100-130 个，建议初始 110 个 | 约 2500-4000 个 | 核心合约恒星系约 300-500 个；服务覆盖恒星系约 10000-20000 个；商业影响可达登记市场 50000 个以上 | 不表现为连续国土，而是贸易港、结算枢纽、护航节点和契约自治港网络。 |
| 中立、争议、暗区和遗产节点 | 40-60 个 | 不按文明统计 | 作为事件、禁航、遗产隔离、虫群或深层异常入口 | 分布在文明接壤处、悬臂间隙、暗区边界和旧航道断裂区。 |

显示节点应优先选择具有政治、航行、经济、军事、生态或剧情意义的恒星系。普通农业星系、资源星系、居住星系和小型研究站可以存在于设定背景中，不必全部出现在主星图上。

## 四、空间分布

当前主星图应表现为“悬臂段文明区”，而不是平面棋盘。

| 区域 | 星海位置感 | 主要文明或节点 |
| --- | --- | --- |
| 悬臂内侧密集带 | 恒星密度高、星门修复较早、以太洋流和共鸣中继成熟。 | 星海联邦核心区、部分商盟枢纽和早期星门城市。 |
| 稳定弧带 | 以太场平缓、生态维护价值高、灾害阈值清晰。 | 永恒环带的生态核心、封存门户和长期维护节点。 |
| 支臂交汇区 | 航道交叉、商贸频繁、政治边界复杂。 | 商盟港口网络、联邦边境枢纽、中立仲裁港和跨文明市场。 |
| 悬臂断裂与暗区边缘 | 以太扰动强、深层信号活跃、旧航道断裂。 | 深渊裂谷实验城邦、深潜前哨、遗产隔离区和虫群风险点。 |
| 悬臂外缘稀疏区 | 恒星密度下降、救援链变薄、开拓成本增加。 | 联邦远征线、商盟特许开拓地、失联殖民地和新剧情节点。 |

四文明不应按简单四象限切开。联邦可以拥有最大连续核心区；环带应呈弧形或带状；裂谷应像沿异常带生长的长裂隙；商盟应穿插在交通交汇处和跨文明边界上。

## 五、星图节点分层

| 层级 | 显示规则 | 典型对象 |
| --- | --- | --- |
| `display_tier: 1` | 主星图常驻显示。 | 文明首府、星区首府、主星门枢纽、一级空间锚、舰队总港、核心生态星系、商盟结算总港。 |
| `display_tier: 2` | 缩放或筛选时显示。 | 区域港、二级空间锚、舰队基地、封存门户、深潜站、重要贸易港、跨文明仲裁港。 |
| `display_tier: 3` | 剧情、事件或专题需要时显示。 | 新开拓星系、事故地点、失联殖民地、遗产样本点、临时锚标、局部战场。 |
| `display_tier: 4` | 背景存在但默认不显示。 | 普通居住星系、资源点、小型研究站、农业和工业附属星系。 |

新开拓星系、事故地点、失联殖民地、遗产样本点、临时锚标和局部战场通常属于 `display_tier: 3`。长期政治、航行或经济中心属于 `display_tier: 2`；文明核心、主枢纽或反复影响全局的地点才属于 `display_tier: 1`。

## 六、航行时间基准

单次自由跃迁或星门通行的物理过程可能只有数十分钟到数小时，但星际旅行的实际时间还包括离港排期、锚定解计算、能源额度申请、航权许可、检疫安检、跃迁窗口、抵达复检、保险确认和目的地接纳。

| 航线类型 | 典型距离 | 客运时间 | 货运时间 | 说明 |
| --- | ---: | ---: | ---: | --- |
| 同一核心星群内 | 10-80 光年 | 12 小时-3 天 | 2-6 天 | 航权成熟、救援覆盖强，普通人可视为远程公共交通。 |
| 稳定锚链航道 | 80-300 光年 | 2-7 天 | 5-14 天 | 需要空间锚、星港排程和常规复检。 |
| 跨星区航线 | 300-1000 光年 | 1-3 周 | 3-6 周 | 常见于工作迁移、军事调动、区域贸易和跨星区医疗转运。 |
| 核心到边境 | 1000-2500 光年 | 3-8 周 | 2-4 个月 | 需要多段航行、补给、锚链复核和边境检疫。 |
| 跨文明远程航线 | 1500-5000 光年 | 1-3 个月 | 3-8 个月 | 时间受跨文明许可、仲裁、保险和检疫影响很大。 |
| 暗区或深层边缘探索 | 不稳定 | 3 个月-1 年以上 | 不适合常规货运 | 以可撤离、可隔离和可复测为目标，不以速度为目标。 |

普通人跨邻近成熟恒星系通常需要 `1-3` 天；从联邦核心去边境通常需要 `1-2` 个月；商队跨文明贸易通常按数月一轮计算；军用快速调动可以明显更快，但连续行动会受维护、补给、政治许可和污染复检限制。

## 七、主星图显示边界

主星图只是具名核心节点、航行节点和事件节点的可视化层，不等于文明全部恒星系清单。四文明实际控制、登记、保护、定居或影响的恒星系数量远多于主星图默认显示节点。

文明疆域在星图上表现为星门枢纽、空间锚、航道、核心星区、边境斑块和影响范围，不是均匀实心边界。当前文明星海只是一段悬臂文明区，银心超大质量黑洞存在，但不占据当前主星图中心。

星图距离不直接等于旅行时间。航权、窗口、锚链、保险、复检、深层扰动和目的地接纳都会改变实际耗时。

地图节点不只有政治属性。核心节点通常同时具备航行、产业、生态、军事、风险或事件功能中的至少一种；普通居住星系、资源点、小型研究站、农业和工业附属星系可以作为背景存在，不必进入主星图常驻显示层。
