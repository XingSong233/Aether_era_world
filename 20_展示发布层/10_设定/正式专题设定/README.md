# 正式专题设定

本目录对应 Matrees“设定”栏目中的 `正式专题设定` 设定集，用于承载 `10_世界观正文/10_专题设定/` 下已经稳定的正式专题整理稿。

## 目录分工

- `能力系统/`：对应 `10_世界观正文/10_专题设定/能力系统/`。
- `星际航行和开拓秩序/`：对应 `10_世界观正文/10_专题设定/星际航行和开拓秩序/`。
- 本目录根部文件：对应 `10_世界观正文/10_专题设定/` 根部的跨领域正式专题。

系统分类目录 `角色/`、`物品/`、`组织/`、`地理/`、`能力/`、`其他/` 只用于独立对象、公开导读或暂未专题化的展示稿；不再承载正式专题正文。
