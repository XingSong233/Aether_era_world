# 联邦思潮状态与制度底线

基准：《以太纪元世界观总纲》、《文明圈对照表》、《文明思潮、道德风向与社会共识》、《航行自由跃迁与开拓秩序》

本专题用于说明星海联邦在当前时代的思潮状态、制度底线、内部张力和设定边界。它不解释联邦文明的具体历史起源，也不保留某个单一战争、霸权体系、遗迹事件或战后公约作为联邦形成原因。

联邦当前的核心理念是开拓未知、文明共存、遗产共享。它把星海生活理解为一套需要公共航图、救援链、职业认证、遗产审计、舰队协同、成员自治和跨种族公共服务共同维持的制度空间。联邦赞赏开拓和远征，但要求风险能够被记录、救援、复盘和追责。

## 一、当前思潮

联邦现在的思潮可以概括为“程序化开拓主义”：未知星域应被探索，古文明遗产应被研究，边境社会应被支持，但开拓不能只靠发现、占有或个人勇气成立。

在联邦语境中，一次合格的开拓至少应留下可复核航图、危险标注、救援信标、样本链、事故预案和责任记录。一个新殖民地只有具备基础服务、居民代表、医疗教育、航权责任、撤离方案和自治承认路径，才会被视为逐步进入文明秩序。

这种思潮不是保守主义。联邦允许远征、军用跃迁、高危遗产研究、边境殖民和前线临时授权。它真正反对的是把风险从公共记录中抹除，把事故推给单个前线个体，把遗产变成无责任战利品，或让某一群体因为航图、身份、救援和责任链缺失而被制度性遗忘。

## 二、制度底线

| 底线 | 当前制度表现 |
| --- | --- |
| 开拓必须可救援 | 航图、信标、空间锚状态、边境救援预案和撤离能力是开拓许可的重要条件。 |
| 风险必须可审计 | 舰船黑匣、维护记录、样本链、共鸣审计账本和任务编号共同构成责任链。 |
| 遗产必须可复核 | 古文明遗产不按发现者直接占有处理，高危样本需隔离、转译、复检和跨机构审查。 |
| 标准化服务于协同 | 舰队接口、职业执照、军标灵械、医疗记录和救援格式需要跨种族、跨星区可读。 |
| 公民权不得被能力化替代 | 联邦不把灵核六维、职业能力、血统、AI 载体或数字人格状态作为唯一身份根基。 |
| 竞争必须保留救济出口 | 航权、资源、遗产样本、军工采购和边境财政可以争夺，但应进入仲裁、议会、法院、审计或赔偿程序。 |

联邦的制度语言常显得繁琐：授权窗口、复检编号、救援优先级、赔付责任、污染隔离、样本托管、职业许可和事故复盘会反复出现。这些内容在联邦日常中不是额外宣言，而是嵌入星港排程、任务记录、停机动作、人员习惯、审计签名和救援流程的制度现象。

## 三、复合文明圈

联邦不是单一中央帝国，而是由多种族公民授权、成员星区协议、盟友政体、舰队开拓功绩、公共标准和遗产共享承诺共同维持的复合文明圈。

联邦系主体族群包括泰拉人族、龙裔、星灵、晶族、潮汐族及其混血后裔。泰拉人族是复合人族谱系，包含多个亚种族；狐人族并入泰拉人族，作为青丘狐人亚种处理。各主体族群和泰拉人族内部各亚种族不应写成天然互补、各自缺少完整文明能力的设定拼图，也不应写成没有资源竞争的理想共同体。它们都可以拥有完整的母星传统、政治传统、职业文化和地方利益；现代联邦身份的重点在于共享一套星海公共操作系统：航图、救援、审计、公民登记、职业认证、遗产隔离和跨族群公共服务。

龙庭帝国、晶环学邦、边境自治星区、舰队驻区、混合星港和成员地方政体都可以保留明显差异。联邦制度的作用不是抹平差异，而是在跨星区行动、公共救援、军用协同、职业标准和公民权争议中提供共同底线。

## 四、内部张力

联邦的主要矛盾不是“开拓者对保守者”这么简单，而是多组制度压力同时存在。

| 张力 | 表现 |
| --- | --- |
| 前线效率与后方审计 | 前线舰队、边境殖民地和救援队希望快速授权；核心星区、法院和专业委员会要求保留复检、样本链和事故责任。 |
| 公共标准与地方传统 | 联邦标准提高协同和救援效率，也可能压低边境小工坊、地方职业路线和成员政体传统的认证空间。 |
| 遗产共享与安全封存 | 联邦支持遗产研究和技术共享，但高危遗产、污染样本、古代权限锁和肃正协议标记会触发隔离审查。 |
| 公共救济与资源分层 | 基础医疗、教育、身份和网络接入应被保障，但高端训练、航权配额、职业套件维护和风险保险仍会制造阶层差距。 |
| AI 公民权与维护责任 | 联邦承认高等级 AI 和数字人格主体权利，但载体维护、备份激活、人工灵核风险和平台劳动仍会制造争议。 |
| 临时授权与常态管制 | 军事、救援和暗区任务需要临时越权，但联邦担心临时授权变成长期不受审计的前线惯例。 |

这些张力应被写成日常制度摩擦，而不是单纯道德口号。联邦角色可以真诚相信公共救援，也可能被表格、复检、排期和责任划分拖住；边境角色可以怨恨核心区迟疑，也可能依赖核心区医院、舰队救援和标准接口活下来。

## 五、制度边界

联邦文明没有单一、具体、决定性的起源事件或起源战争；已撤回的具体历史专名、霸权体系名、战争名、遗迹名和战后公约名不再解释联邦现状。

联邦也不是几个天然互补种族的自动合并。主体族群和泰拉人族亚种族差异不构成文明短板拼装、固定职业命运或天然等级；它们共同进入联邦身份，是因为共享航图、救援、审计、公民登记、职业认证、遗产隔离和跨族群公共服务。

联邦不是没有资源争抢、阶层分化或制度拖延的理想国。“遗产共享”不等于无成本公开所有古文明技术，高危样本、污染风险、军事用途和主体稳定风险仍需隔离审查。“开拓”也不等于插旗即主权，联邦开拓强调测绘、作业、锚站、基础服务、居民权利、救援和自治承认。

联邦公共审计不是绝对正确的无人裁判。法院、仲裁、专业委员会、舰队审计和地方机构都可能冲突、延误或犯错，公共救济和责任链因此同时构成联邦的优势与摩擦来源。

## 六、当前稳定口径

联邦当前时代可稳定保留以下现状口径：

- 联邦重视开拓、共存、遗产共享。
- 联邦赞赏可救援、可审计、可共同承担的开拓。
- 联邦强调公共航图、边境救援、职业认证、舰队协同和事故复盘。
- 联邦承认跨种族公民权，混血后裔与纯血后裔无本质差异。
- 联邦承认高等级 AI 和数字人格的主体权利，并将维护救济纳入制度争议。
- 联邦由中央议会、成员星区、舰队司令部、专业委员会、盟友政体和边境自治组织共同构成。
- 联邦内部长期存在前线授权、核心审计、边境资源、成员自治、遗产隔离和公共救济之间的张力。
