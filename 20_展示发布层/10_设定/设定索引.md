# 设定发布索引

本文件对应 Matrees“设定”栏目，用于说明本地专题如何映射到网站设定集树和网站分类，并说明正式正文整理稿在展示发布层中的读取方式。

## 当前网站结构

Matrees“设定”栏目当前按以下结构维护：

- 系统分类：`角色`、`物品`、`组织`、`地理`、`能力`、`其他`。
- `正式专题设定`
  - `能力系统`
  - `星际航行和开拓秩序`
  - `文明思潮、道德风向与社会共识`
- `90_协作工作台`
  - `设定工作台`
    - `关键科技与生产关系状态表`
  - `素材与导出`

发布正式专题时，先挂入 `正式专题设定` 或其子设定集，再选择合适的网站分类。系统分类节点 `角色`、`物品`、`组织`、`地理`、`能力`、`其他` 只承载独立对象、公开导读或暂未专题化的展示稿。工作台、蓝图、状态表和素材登记只进入 `90_协作工作台`，不作为正式设定正文发布。

## 正式正文整理稿

当前已为 `10_世界观正文/` 下的总纲、正文 README、专题设定、能力系统和星际航行专题建立展示层整理稿。整理稿保留源正文的半世界观说明文口吻，只补充展示层 frontmatter、栏目分类、来源追踪和必要的路径清理，不改写为世界内报道、档案或叙事文本。

整理稿主要分布如下：

| 展示分类 | 对应内容 |
| --- | --- |
| `00_世界背景/` | 世界观正文阅读入口、以太纪元世界观总纲。 |
| `10_设定/正式专题设定/` | 术语、文明思潮、深层以太、以太能与能源体系、天灾、共鸣虫群、AI 与数字人格、信息治理、经济体系、生产关系、生命工程、文明、军事、机甲等跨领域正式专题。 |
| `10_设定/正式专题设定/能力系统/` | 能力系统索引、个体能力运行、灵脉结构、架构演变、标准化、谱系和职业模板。 |
| `10_设定/正式专题设定/星际航行和开拓秩序/` | 星际航行和开拓秩序、星际定位技术、星海地理与星图节点。 |
| `10_设定/正式专题设定/文明思潮、道德风向与社会共识/` | 文明思潮专题按自然章节拆分后的发布条目。 |
| `10_设定/能力/` | 面向读者的能力系统导读和可对象化能力条目，不承载正式能力专题全文。 |
| `10_设定/地理/` | 面向读者的航行、星域、暗区或地点导读，不承载正式航行专题全文。 |
| `10_设定/组织/` | 文明、机构、舰队等独立对象或公开简报，不承载正式文明专题全文。 |
| `10_设定/物品/` | 灵械、舰船、机甲、遗产样本、工具装备等独立对象或公开导读。 |
| `10_设定/其他/` | 暂未适合拆入角色、组织、地理、能力、物品的公开导读和过渡展示稿。 |

## 分类口径

| 网站分类 | 本地内容类型 | 说明 |
| --- | --- | --- |
| 设定集 | 总体制度、社会结构、经济、航行、科技状态、术语总表 | 适合承载跨领域专题和说明性资料；文件优先放入 `正式专题设定/`。 |
| 能力 | 灵核、灵脉、灵械、能力成长、灵脉结构原理、灵脉谱系、标准化、职业能力模板 | 正式能力专题放入 `正式专题设定/能力系统/`，读者导读和独立能力对象才放入 `能力/`。 |
| 角色 | 主角、候选角色、人物设定卡、视觉设定卡 | 后续建议一人一页。 |
| 组织 | 文明圈、机构、舰队、商盟组织、议会、军工体系 | 当前文明圈对照表可先归入组织或设定集。 |
| 地理 | 星域、航道、空间锚、星港、暗区、深层以太边界 | 航行专题挂入 `正式专题设定 > 星际航行和开拓秩序`，后续地点和航路对象再拆到地理或地图。 |
| 物品 | 灵械、舰船、机甲、遗产样本、工具装备 | 当前载人机甲可先归入设定集或物品。 |
| 其他 | 公开导读、过渡展示稿、发布说明和非正文辅助资料 | 正式专题不放入 `其他/`；工作台索引应优先进入 `90_协作工作台`。 |

详细文件级映射见 `../90_发布记录/Matrees发布映射表.md`。
