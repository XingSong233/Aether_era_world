# 联邦基层资源九维整理

状态：工作台采用
基准：`10_世界观正文/10_专题设定/经济体系.md`、`10_世界观正文/10_专题设定/生产关系与职业结构.md`、`10_世界观正文/10_专题设定/信息治理与社会制度.md`、`10_世界观正文/10_专题设定/联邦思潮状态与制度底线.md`

本文件整理当前讨论中已经确认的“联邦基层人口资源状态”。它只保存世界观内部结论，不展开外部理论模型；分析方法另见 `Max-Neef模型与资源分析方法.md`。

## 一、总口径

联邦基层人口通常不是基础生存贫困。稳定星区和多数合法殖民地能够提供基础食物、住房、基础医疗、基础教育、基础共鸣网络和公共身份入口。

基层资源问题主要表现为：基础服务存在，但满足质量、可选择性、个体化程度、风险兜底、迁徙能力、制度兑现能力和生活主体性不足。

换言之，联邦基层的贫困更常是机会贫困、维护贫困、权限贫困、时间贫困、身份贫困和自由贫困，而不是单纯饥饿或无家可归。

## 二、九维资源表

| 维度 | 基础保障 | 基层常见缺口 | 可写表现 |
| --- | --- | --- | --- |
| 生存 | 基础营养、公共住房、空气水循环、温控重力、基础医疗、灾害避难资格。 | 居住环境拥挤、共鸣底噪高、维护响应慢、生态景观少、低污染生活场域不足。 | 居民吃得饱、住得下，却长期生活在标准舱、维护区、交通噪声区或低景观层。 |
| 保护 | 基础医保、事故记录、公共救援、基础安全巡检、职业许可救济入口。 | 高质量保险、长期复健、职业密钥临时恢复、记录纠错、事故复审、灾害撤离优先级不足。 | 一次训练事故、灵械断连或维护记录错误，就可能长期影响职业资格和保险费率。 |
| 情感 | 家庭登记、社区服务、基础心理咨询、公共照护入口。 | 长期照护、伤后陪护、心理回响治疗、儿童照护、跨星区家庭团聚和稳定社区网络不足。 | 家庭成员若进入灵核稳定观察或职业复健，其他成员的训练、迁徙和休息时间被持续压缩。 |
| 理解 | 基础教育、以太安全、低功率接口训练、公共职业咨询、基础共鸣课程。 | 个体化灵脉设计、灵核六维细分评估、职业改道建议、事故案例库、高质量导师和训练沙箱不足。 | 学生有课上、有考试，却只能套标准模板；上层能获得适配个人的灵脉路线和多次安全试错。 |
| 参与 | 公民身份、居民代表、公共听证、职业许可申诉、殖民地自治入口。 | 议程设置、听证参与成本、公共数据解释、工会/互助组织、边境发声渠道不足。 | 基层问题常被处理成个案失败，难以变成星区预算、学校分流或维护制度改革议题。 |
| 闲暇 | 公共娱乐、低价沉浸内容、节庆活动、基础休息日和公共休闲场所。 | 连续闲暇、高带宽沉浸、真实旅行、安全刺激、优质景观、非平台化休闲不足。 | 大家都“看过星海”，但基层多是压缩沉浸版本，很少真正离开本地星区。 |
| 创造 | 公共工坊、基础艺术教育、低功率民用灵械、社区活动、共鸣发布平台。 | 非功利创作时间、材料额度、低风险改装权限、导师反馈、展示渠道和失败后不被记录的空间不足。 | 基层青年可以做公共模板作品，却很难长期试验个人灵械、原创灵脉思路或高质量艺术项目。 |
| 身份 | 联邦公民权、基础身份凭证、职业证书、族群文化登记和公共服务记录。 | 职业尊严、边境学历认可、维护记录声誉、被核心机构信任、地方文化被平等看见不足。 | 边境学校毕业生到核心区仍被要求额外复核；同等能力下，出身和记录来源影响信任。 |
| 自由 | 法定迁徙权、职业选择权、最小披露原则、退出高危任务的制度底线。 | 航权票补、核心区居住资格、职业改道、平台退出、隐私服务、拒绝高风险训练后的替代路径不足。 | 个体理论上能换城市、换职业、换平台，实际被航权、记录、密钥、债务和家庭照护锁住。 |

## 三、已讨论资源类型归并

| 已讨论类型 | 对应九维 | 资源内容 |
| --- | --- | --- |
| 提升资源 | 理解、创造、身份、自由 | 个体化教育、灵脉设计、导师、训练沙箱、职业模板、维护链、认证担保、职业改道。 |
| 享乐资源 | 闲暇、情感、身份、自由 | 高带宽沉浸、真实旅行、优质环境、真实食材、私人空间、安全刺激、审美定制、完整闲暇。 |
| 保障资源 | 保护、生存、情感 | 保险、事故救济、职业复健、密钥临时恢复、灾害撤离、长期照护、家庭应急额度。 |
| 主体性资源 | 参与、身份、自由、保护 | 申诉、迁徙、隐私、记录纠错、共同体组织、政治表达、拒绝权、平台退出权。 |

这些类型不是互斥分类。一个资源常同时满足多个维度。例如高质量共鸣网络既是理解资源，也是参与、闲暇、身份和自由资源；稳定维护记录既是保护资源，也是身份和职业自由资源。

## 四、基层资源状态样例

### 边境职业学校学生

| 维度 | 状态 |
| --- | --- |
| 生存 | 有公共宿舍、餐食和基础医疗。 |
| 保护 | 训练事故后复审慢，灵核复健名额少。 |
| 理解 | 有标准课程，缺少个人灵脉路线设计。 |
| 闲暇 | 有廉价沉浸娱乐，缺少真实旅行和连续假期。 |
| 身份 | 边境学校证书进入核心区时需要额外复核。 |
| 自由 | 想转入核心星区学院，但缺星门票补和临时居住资格。 |

### 城市低阶灵械维护员

| 维度 | 状态 |
| --- | --- |
| 生存 | 生活稳定，有基础住房和医疗。 |
| 保护 | 工伤保险覆盖基础治疗，不覆盖长期职业复健。 |
| 理解 | 能完成标准维护，缺高等级模块训练名额。 |
| 参与 | 很少进入标准修订听证，只能被动执行新规。 |
| 创造 | 不能轻易做非登记改装，社区工坊资源有限。 |
| 自由 | 被租赁平台排班、密钥续费和维护记录绑定。 |

### 核心区公共服务家庭

| 维度 | 状态 |
| --- | --- |
| 生存 | 基础服务质量高于边境。 |
| 情感 | 儿童照护名额紧张，家庭时间被证书续期和排班切碎。 |
| 理解 | 孩子有较好基础教育，但个体化灵脉设计仍需额外支付或排队。 |
| 闲暇 | 可获得高质量公共娱乐，但私人空间和真实旅行仍昂贵。 |
| 身份 | 核心区公共学校记录更容易被机构信任。 |
| 自由 | 比边境居民迁徙更容易，但仍受居住资格、训练名额和职业记录限制。 |

## 五、写作口径

- 不要把联邦基层写成普遍饥荒、无医疗或无教育；除非具体地点处于灾害、战争、边境断供或非法控制状态。
- 写基层困境时，优先写基础服务与高质量资源之间的落差。
- “有权利”和“能兑现权利”要分开写。联邦有救济链，但基层可能缺时间、代理人、证据解释权和跨星区资源。
- “有娱乐”和“有高质量闲暇”要分开写。廉价沉浸娱乐可以覆盖很广，但真实旅行、私人空间、优质环境和连续休息仍然稀缺。
- “有教育”和“有适合自身的能力成长方案”要分开写。基础教育普及不等于个体化灵脉设计、长期导师和安全试错普及。
- 基层资源问题应同时落在票据、记录、密钥、排班、许可、复检、维护周期、航权窗口、居住资格和申诉入口上。
