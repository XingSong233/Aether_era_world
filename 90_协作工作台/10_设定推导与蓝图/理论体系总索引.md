## 四、理论依赖关系

| 依赖关系 | 说明 |
| --- | --- |
| 地理 -> 经济 | 群岛式星海、可达性、星门、空间锚和深层边界决定运输成本、购买力断层和贸易利润结构。 |
| 经济 -> 生产关系 | 多结算制、购买力差异、维护资格、能源额度和航权配额决定阶层、职业机会和劳动组织。 |
| 经济/生产关系/信息治理 -> 个体资源 | 基础服务权、维护链、身份凭证、救济程序、航权、记录纠错和制度信任共同决定个体九维资源状态。 |
| 能力 -> 职业结构 | 灵核、灵脉、灵械、接口带宽和维护链决定职业套件、训练路径和能力资本。 |
| 信息治理 -> 制度执行 | 身份凭证、审计账本、预测算法、自动执法和救济程序决定制度能否真实落地。 |
| 风险治理 -> 文明分化 | 对遗产、深层以太、污染、封存、实验和军用化的不同态度会长期固化为文明差异。 |
| 主体身份 -> 经济与法律 | AI、数字人格、克隆体和生命载体的主体地位会影响劳动合同、赔付、继承和公民权。 |
| 地理/经济/风险治理 -> 文明分化 | 群岛式星海、购买力断层、风险解释和高危遗产共同提供多稳定收益地形；具体文明差异由历史微态成核和亲和策略放大。 |

简化调用顺序：

```text
星海地理 -> 星海经济 -> 生产关系
能力工程 -> 职业结构 -> 生产关系
信息治理 -> 制度执行 -> 文明分化
风险治理与遗产 -> 战争/封存/实验 -> 文明分化
主体身份 -> 经济合同/法律权利/社会制度
星海地理 + 星海经济 + 前身路线 -> 文明分化
```

## 六、使用规则

- 理论文件负责推导，正式专题负责世界内稳定叙述。
- 理论总索引只做导航和依赖关系，不合并各理论文件正文。
- 新增理论体系前，应先判断该领域是否有跨专题复用价值。
- 若某条理论会改变核心规则，应同步更新相关正式专题、术语表、AGENTS.md、版本索引和 CHANGELOG。
- 理论编号是维护工具，不应作为世界内术语直接出现在报道、档案、人物口述或小说正文中。
