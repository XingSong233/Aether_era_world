# 设定推导与蓝图

本目录保存用于辅助正文扩写的状态表、态度矩阵、设计蓝图、理论层级整理和原型工具。它们可以作为推导依据，但不是世界观正文专题。

## 文件分工

- `理论体系总索引.md`：理论体系导航、依赖关系、编号规范和后续理论化优先级。
- `关键科技与生产关系状态表.md`：确认关键科技存在状态、成熟度、掌握主体和生产关系边界。
- `四文明关键科技态度矩阵.md`：推导四大文明对关键科技簇的态度、管控方式和冲突点。
- `经济体系设计蓝图.md`、`星海经济理论层级整理.md`：辅助经济体系、贸易、购买力和区域生活成本推导。
- `航行自由跃迁与开拓秩序设计蓝图.md`、`星海地理与星图节点设计蓝图.md`、`星海地理理论层级整理.md`、`前身文明扩张策略推导.md`：辅助航行、地图、疆域、前身文明和联邦制度根系推导。
- `文明分化理论层级整理.md`：辅助四大文明起源、亲和型扩张策略、历史微态成核和整体意识形态推导。
- `联邦诞生过程多模型推导草稿.md`、`联邦早期历史事件链草案.md`：辅助联邦起源模型、公共责任链成核、工具性文明、候选遗迹、深层危机和早期历史事件链推导。
- `信息治理与社会制度设计蓝图.md`：辅助共鸣网络、身份、审计、预测算法、自动执法和深层污染信号治理扩写。
- `星海疆域生成器/`：用于地图布点和疆域形态推演的工作台原型。

## 使用原则

- 稳定设定应整理进 `10_世界观正文/10_专题设定/`。
- 推导规则如成为硬口径，应同步更新总纲、协作规则、版本索引和 CHANGELOG。
- 原型工具输出只作为辅助，不自动成为正式星图或正式历史。
