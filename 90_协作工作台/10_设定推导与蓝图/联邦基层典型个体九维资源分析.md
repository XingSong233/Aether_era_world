# 联邦基层典型个体九维资源分析

状态：工作台分析稿
基准：`联邦基层资源九维整理.md`、`Max-Neef模型与资源分析方法.md`

本文件用于把“联邦基层资源九维”落实到几个可写的典型个体切面。它不是世界内官方评分表，也不应在报道、档案或人物口述中直接提及 Max-Neef 模型；正式写作时应只吸收资源状态、制度入口和剧情表现。

## 一、总口径

联邦基层人口通常不是基础生存贫困。稳定星区和多数合法殖民地拥有食物、住房、基础医疗、基础教育、基础共鸣网络和公共身份入口。

基层资源困境主要表现为：基础服务存在，但高质量资源、可选择性、个体化方案、风险兜底、制度兑现能力、迁徙能力和生活主体性不足。

因此，分析个体时不要只问“他有没有资源”，而要进一步问：

- 这些资源是否高质量、可持续、可自由选择。
- 他能否进入真正有效的制度入口。
- 他是否有时间、担保、记录可信度和解释权。
- 一次失败、事故、复检或密钥冻结是否会长期压低未来机会。
- 某项资源是否同时满足一项需要、压制其他需要，或只是伪满足物。

## 二、典型个体切面

### 1. 边境职业学校学生

该切面适合表现边境教育、能力成长、迁徙机会和身份可信度之间的张力。学生通常有学校、课程、宿舍和职业入口，但缺少个体化路线、优质训练沙箱、核心区认可和迁徙资源。

| 维度 | 可调用资源 | 主要缺口 | 可写表现 |
| --- | --- | --- | --- |
| 生存 | 公共宿舍、学校餐食、校医、基础空气水循环、训练区避难规程。 | 宿舍拥挤，噪声和共鸣底噪偏高，训练区与生活区维护优先级低。 | 吃得饱、能上课，但长期住在低景观层；训练后头痛，校医判断未达事故等级，只能排队复查。 |
| 保护 | 基础医保、训练事故登记、实训安全巡检、职业许可救济入口。 | 灵核复健名额少，事故复审慢，实训密钥恢复需要多方确认。 | 低功率接口事故后身体恢复了，记录仍挂着“共鸣不稳定观察”，错过核心区学院复核窗口。 |
| 情感 | 家庭登记、学生事务、基础心理咨询、校内社区服务。 | 跨星区陪护和团聚受航权、票补、假期窗口和居住资格限制。 | 父母只能低带宽共鸣通话陪伴；家属排到星门窗口时，关键复审已经结束。 |
| 理解 | 基础课程、以太安全课、低功率接口训练、公共职业咨询。 | 缺灵核细分评估、个体化灵脉路线、导师和完整事故案例库。 | 考试合格，但标准灵脉路线不适配；系统自动推荐低风险岗位，把上升路径提前封窄。 |
| 参与 | 学生代表、校内听证、事故复审、殖民地自治青年席位。 | 议程设置弱，边境教育资源不足常被拆成个人成绩或适配问题。 | 学生拿出同届事故数据，却被要求提供个人损害证明；申诉还可能延迟毕业许可。 |
| 闲暇 | 学校公共娱乐、低价沉浸内容、节庆活动、基础休息日。 | 缺真实旅行、连续假期、私人空间和不计分的安全刺激。 | “看过星海”其实是低带宽沉浸舱十分钟压缩景观；假期被补测、说明会和票补等待切碎。 |
| 创造 | 学校工坊、低功率教学套件、基础材料包、校内发布平台。 | 材料额度低，改装审批保守，导师担保弱，失败容易进训练记录。 | 工坊就在楼下，但原创改装只能批教材限定参数；申请核心区展示还要补安全复核。 |
| 身份 | 联邦公民身份、边境职校学籍、实训评价、地方共同体推荐。 | 边境学历信任折损，核心区机构要求额外复核和延长观察。 | 合格毕业证进入核心区面试仍需重跑基础共鸣稳定测试；边境现场经验被当作背景故事。 |
| 自由 | 法定迁徙权、升学申请权、职业分流选择权、拒绝超标训练底线。 | 票补排队、临时居住资格短、边境证书复核多，拒绝高危训练会留下保守标签。 | 拿到核心区学院初审，却因票补没批、住宿资格不覆盖等待期而放弃。 |

核心资源链：边境记录可信度不足，会同时压低理解、身份和自由；一次训练事故又会牵动保护、情感和迁徙机会。

### 2. 城市低阶灵械维护员

该切面适合表现维护贫困、平台绑定、密钥控制和职业声誉脆弱性。维护员不是无技能者，而是掌握一线经验，却常被困在执行端。

| 维度 | 可调用资源 | 主要缺口 | 可写表现 |
| --- | --- | --- | --- |
| 生存 | 基础住房、基础医疗、工伤初诊、公共交通、维护区安全规程。 | 居住和工作常靠近维护区、交通噪声区或低景观层，恢复余量不足。 | 下班回到设备井旁的窄舱；空气指标合规，却总有冷却液和金属尘味。 |
| 保护 | 工伤基础保险、维护事故记录、职业体检、许可救济、平台最低事故处理。 | 长期复健不足，记录纠错成本高，密钥冻结会直接中断收入。 | 平台日志缺失上下文，记录变成“响应延迟”；密钥临时恢复只给低风险区权限。 |
| 情感 | 家庭登记、社区服务、公共心理咨询、工友互助、家庭照护证明。 | 情感资源被排班、密钥续费和收入波动切割，长期职业耗竭难承接。 | 伴侣陪护假只能覆盖几天；孩子照护名额排不上；心理建议降夜班，平台却要求补足时长。 |
| 理解 | 标准维护培训、岗位手册、低阶模块权限、规程更新。 | 缺高阶模块训练、完整事故链解释权和个体化能力开发。 | 知道临时补丁能救场，却没有权限查看完整事故链；训练窗口排在休息日，失败影响密钥续费。 |
| 参与 | 工会、行会、平台投诉、标准修订意见征集、公共听证。 | 一线经验难被承认为制度知识，数据解释权弱。 | 他最清楚旧城区接口问题，却没被邀请进标准修订会，只能提交低优先级意见。 |
| 闲暇 | 城市公共娱乐、廉价沉浸、维护员休息舱、平台积分娱乐。 | 连续休息被抢修、排班和密钥续费打断，休闲高度平台化。 | 买了夜间沉浸票，临时抢修通知弹出，只能折成平台点数。 |
| 创造 | 社区工坊、废旧模块、标准工具、同事经验网络。 | 缺合法时间、改装权限、署名担保和失败后不进记录的空间。 | 做出低成本稳定件，社区试用有效，但平台要求先提交完整责任链和复检费用。 |
| 身份 | 公民身份、低阶维护证书、平台工单履历、客户或社区评价。 | 职业声誉依赖平台记录，误标、密钥冻结、坏设备接手都可能污染履历。 | 居民口头信任他，机构审核只看事故率、返修率和密钥异常。 |
| 自由 | 职业选择权、平台退出权、职业许可申诉、隐私和记录纠错服务。 | 被租赁债务、密钥续费、排班和维护记录绑定。 | 想退出平台，却发现外置灵械套件还有三年分期；拒绝高风险抢修后被降到低收益排班。 |

核心资源链：平台职业套件是典型抑制性满足物。它提供工作机会，同时用债务、密钥、排班和记录限制退出、创造和上升。

### 3. 核心区公共服务家庭成员

该切面可以写基层公共服务排班家长，也可以写其青年子女。它适合表现“资源质量较高但时间贫困严重”的生活状态：入口更多、记录更可信，但排班、照护、证书续期和居住资格把家庭成员绑定在高度制度化的生活轨道里。

| 维度 | 可调用资源 | 主要缺口 | 可写表现 |
| --- | --- | --- | --- |
| 生存 | 质量较高的公共住房、稳定供应、社区医疗、核心区避难系统。 | 时间紧张，高质量医疗、营养、安静空间和恢复资源不总能持续调用。 | 窗外是干净公共景观，家里却长期缺整块休息时间。 |
| 保护 | 高质量基础医保、公共救援、完整服务记录、家庭应急额度。 | 长期复健、心理回响治疗、儿童照护和替班额度不足。 | 单位确认事故无重大责任，但权限恢复仍要等复审；请假记录被归为履约不稳定。 |
| 情感 | 家庭登记、社区服务、儿童照护、公共心理咨询、家庭紧急申报。 | 入口不缺，但名额和连续时间不足。 | 家长白天值班、晚上抢照护名额；每个人都有入口，却没有完整共同时间。 |
| 理解 | 优质公共学校、稳定共鸣网络、职业咨询、灵核评估、训练沙箱预约。 | 个体化灵脉设计仍需排队、推荐或额外支付，家庭排班打断连续评估。 | 老师建议细分评估，但预约排到数月后，家长正好进入续证和轮班高峰。 |
| 参与 | 居民代表、单位申诉、家长委员会、青年议事、公共数据查询。 | 理论上能参加听证，实际上常被排班、证书续期和照护切碎。 | 家长想参加照护名额听证，却临时被调班；青年发现公开数据不区分夜班家庭。 |
| 闲暇 | 高质量公共娱乐、核心区文化设施、公共沉浸馆、城市节庆。 | 私人空间、真实旅行、家庭包厢和连续休息仍稀缺。 | 能去一流展演，却只能卡在换班后两小时窗口；旅行计划常变成城市沉浸替代。 |
| 创造 | 公共工坊、学校创作课程、社区展示厅、城市发布平台。 | 入口更多但筛选更强，青年缺不被评估的试错空间，家长缺连续创作时间。 | 孩子想参展，老师建议选更稳妥的评审标准作品；家里缺失败不留标注的时间。 |
| 身份 | 核心区居民身份、公共服务家庭记录、可信教育医疗档案。 | 公共服务记录既是信誉也是约束，青年可能被默认适合稳定公共路线。 | 家庭记录带来初始信任；一次照护冲突导致临时缺勤，系统从稳定可靠改为需观察。 |
| 自由 | 排班调整、儿童照护、隐私服务、职业转轨、核心区内迁居或跨区调任。 | 居住资格、岗位、子女学校、照护名额和公共服务记录精密绑定。 | 青年拿到外星区训练机会，但家庭若迁出会失去核心区住房资格和学校连续记录。 |

核心资源链：核心区身份资源更强，但也更容易转化为精密约束。公共服务记录能换来信任、担保和入口，也会把排班、照护、住房、学校和职业声誉绑在一起。

## 三、横向结论

| 资源问题 | 边境职业学校学生 | 城市低阶灵械维护员 | 核心区公共服务家庭成员 |
| --- | --- | --- | --- |
| 主要贫困形态 | 机会贫困、身份贫困、迁徙贫困。 | 维护贫困、权限贫困、平台锁定。 | 时间贫困、照护贫困、制度化自由贫困。 |
| 最关键记录 | 学籍、训练记录、事故复审、边境学历。 | 维护工单、密钥状态、事故记录、平台排班。 | 公共服务履历、证书续期、家庭照护记录、核心区学校记录。 |
| 最关键制度入口 | 转学复核、票补、灵核评估、导师推荐。 | 职业许可、密钥恢复、记录纠错、行会和工会。 | 排班调整、照护名额、公共服务转岗、学校与社区推荐。 |
| 常见剧情触发点 | 票补失败、实训事故、证书额外复核、导师担保不足。 | 密钥冻结、工单误标、非登记改装、抢修打断休假。 | 临时调班、照护名额失败、证书续期冲突、居住资格绑定。 |
| 最容易被误写成 | 没饭吃、没学上、无医疗。 | 纯低技能、纯穷、无组织。 | 完全体面稳定、资源充足无困境。 |
| 正确写法重点 | 有基础服务，但缺个体化成长和可迁徙机会。 | 有技能和工作，但被记录、密钥和债务锁在执行层。 | 有高质量入口，但缺连续时间和退出成本低的选择。 |

## 四、写作抓手

- “有权利”和“能兑现权利”分开写：迁徙、转学、申诉、退出平台、拒绝高危训练都可能合法存在，却被票补、居住资格、记录、密钥、债务和照护责任拦住。
- “有服务”和“有高质量服务”分开写：基础医疗、教育、娱乐、心理咨询、公共工坊都存在，但高质量、连续、个体化和可失败的资源稀缺。
- “有记录”和“记录被信任”分开写：边境学校、低阶维护平台和核心区公共服务系统都有记录，但它们的信任权重、迁移能力和纠错成本完全不同。
- “有娱乐”和“有闲暇”分开写：低价沉浸能覆盖很多人，但真实旅行、私人空间、连续休息、高带宽体验和非平台化休闲仍是关键差异。
- “有创造入口”和“允许失败”分开写：工坊、课程、发布平台并不稀缺，真正稀缺的是材料余量、低风险改装权限、导师担保和失败不进履历或维护记录的空间。

## 五、满足物检查

| 满足物类型 | 可用例子 |
| --- | --- |
| 协同满足物 | 高质量公共职业学院同时提供理解、身份、保护、参与和自由。 |
| 单一满足物 | 临时票补只解决一次通行，不解决核心区居住资格、导师担保或长期改道。 |
| 伪满足物 | 廉价沉浸娱乐缓解无聊，却遮蔽真实旅行、连续休息和主体性缺口。 |
| 抑制性满足物 | 平台职业套件提供工作机会，同时用债务、密钥和排班限制退出。 |
| 破坏性满足物 | 黑市改装短期提供职业机会，却破坏保护、身份、自由和长期生存安全。 |
