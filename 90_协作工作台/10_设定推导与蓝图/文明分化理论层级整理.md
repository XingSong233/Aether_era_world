# 文明分化理论层级整理

状态：采用，设定推导工作台
基准：`10_世界观正文/00_总纲/以太纪元世界观总纲.md`、`10_世界观正文/10_专题设定/文明圈对照表.md`、`90_协作工作台/10_设定推导与蓝图/前身文明扩张策略推导.md`、`90_协作工作台/10_设定推导与蓝图/星海地理理论层级整理.md`、`90_协作工作台/10_设定推导与蓝图/星海经济理论层级整理.md`
总索引见：`理论体系总索引.md`

本文件整理四大文明起源、扩张策略和整体意识形态的推导框架。它不是正式历史年表，也不是世界内学术论文；稳定世界内叙述仍需后续整理进正式专题。本文件优先解决一个底层问题：在相同星海底层环境中，四大文明为什么能形成四种长期稳定、收益无明显优劣、彼此亲和条件不同的扩张策略。

这里的“公理”“定义”“引理”“定理”沿用工作台理论文件格式：

- 公理：本层不再继续证明的底层文明分化规则。
- 定义：用于避免“环境”“扰动”“策略”“亲和”等概念混用。
- 引理：由公理和定义推出、可反复调用的中间规律。
- 定理：用于解释四大文明起源、扩张和意识形态的高层结论。
- 推论：面向写作、历史整理和设定修订的直接应用。

## 二、公理层

### V-A1 底层环境一致公理

当前四大文明起源于相同底层星海环境。这里的底层环境包括自然规律、以太规律、星海尺度、星际交通基本约束、资源统计结构、智慧生命基础条件和技术可行性边界。

底层环境一致不等于每个文明拥有完全相同的历史微态。随机扰动、起源顺序、早期接触网络、局部战争、技术标准先发、航图事故和组织记忆属于历史微态，不属于本公理要排除的底层差异。

使用后果：四大文明不能被写成“因为起源星区环境本质不同，所以天然形成不同文明”。若出现起源地差异，应解释为历史微态扰动、早期组织选择和后续策略成核的结果。

### V-A2 收益地形公理

底层环境决定扩张策略的可行集合、成本结构和收益地形，但不唯一决定具体策略。文明会在收益地形中通过历史微态、组织选择和早期正反馈落入某种稳定扩张策略。

使用后果：环境不是直接给出一个答案，而是给出若干可持续答案的空间。

### V-A3 多稳定解公理

某些底层环境只允许单一压倒性最优策略，某些底层环境允许多个长期稳定、收益无明显优劣的局部最优策略。当前星海底层环境采用后一种口径：它允许多种稳定扩张策略并存。

使用后果：四大文明不是先进与落后的线性序列，而是同一问题的不同稳定解。

### V-A4 历史微态锁定公理

在多稳定解环境中，随机扰动、先发规模、接触顺序、技术标准、战争结果、灾害记忆、组织利益和制度兼容，会使某一策略在局部区域率先成核，并通过正反馈扩大为区域性吸引盆地。

使用后果：一个起源星区看起来“天然适合某文明”，往往是该策略先成核后重塑周边制度、航图、贸易和教育体系的结果。

### V-A5 无普世亲和公理

不存在无条件亲和所有其他文明的扩张策略。任何亲和都依赖识别条件、同构条件、可转译条件、利益边界或责任边界。

使用后果：四大文明都可以是亲和策略，但都不是无条件和平策略。

### V-A6 扩张策略性质公理

本理论至少讨论两类主导扩张策略：

| 策略性质 | 核心特征 |
| --- | --- |
| 排他策略 | 倾向排除自身以外的所有文明，即使对方扩张策略与自身一致，也仍会被视为竞争者。 |
| 亲和策略 | 对相同策略或可转译为相同策略的文明呈现亲和倾向，对不同策略文明呈现排斥、改造或压缩倾向。 |

本分类是当前理论使用的主轴，不宣称穷尽所有可能策略。隔离、寄生、伪装、附庸化和中立避世等策略可以作为局部或失败策略存在。

### V-A7 四文明亲和策略公理

当前世界观的四大文明均采用亲和策略。它们会吸纳同策略或可转译为同策略的文明共同体，但亲和条件不同：

| 文明 | 亲和识别条件 |
| --- | --- |
| 星海联邦 | 是否愿意进入公共航图、救援、审计、权利和责任链。 |
| 永恒环带 | 是否愿意服从长期阈值、生态稳定、封存和人格连续原则。 |
| 深渊裂谷 | 是否愿意把风险、改造、失败和深层接触转化为可记录、可继承的适应能力。 |
| 商盟 | 是否愿意把关系纳入可履约、可仲裁、可维护、可定价的合约网络。 |

## 三、定义层

| 定义 | 说明 |
| --- | --- |
| 底层环境 | 自然规律、以太规律、星海尺度、交通约束、资源统计结构、技术可行性边界等长期背景条件。 |
| 历史微态 | 具体起源顺序、早期接触网络、局部战争、灾害、组织记忆、技术标准先发和偶然事件。 |
| 扩张策略 | 文明把人口、制度、航路、资源、风险、技术和身份边界向外复制或延伸的稳定方式。 |
| 策略吸引子 | 一种能自我强化、吸纳同类、压缩竞争策略并长期稳定的扩张逻辑。 |
| 策略成核区 | 某个策略因历史微态扰动率先跨过临界规模的局部区域。 |
| 吸引盆地 | 成核后被该策略持续吸纳、改造、标准化和制度化的区域或文明网络。 |
| 亲和条件 | 一个亲和策略用来识别“同类”或“可转译同类”的制度条件、技术条件或伦理条件。 |
| 转译成本 | 一个异策略文明被吸纳时，为了接入主导策略而必须改变航图、法律、教育、身份、维护链和风险解释方式的成本。 |
| 主导制度生态 | 一个文明内部多数机构、职业、家庭、企业、军队、学校和边境组织共同默认的扩张逻辑。它不等于每个个体都自觉接受同一意识形态。 |

## 四、引理层

### V-L1 环境非直接决定引理

由 V-A1、V-A2 和 V-A3 可得：在当前星海中，底层环境不直接决定唯一文明形态，而是提供多个可持续吸引子的收益地形。

使用后果：写四大文明起源时，不应把文明差异归因于自然环境天然分区。更稳的写法是：同一底层环境允许多种稳定解，不同历史微态让不同策略率先成核。

### V-L2 局部优势非全局优越引理

由 V-A3 和 V-A4 可得：某策略在局部区域压倒其他策略，不代表它在全星海绝对更优。局部胜出可能来自先发规模、网络效应、组织兼容、标准锁定、灾害记忆或早期战争结果。

使用后果：被吞并或被转译的文明策略不应被写成理论上低劣，只能说明它们在该局部历史窗口中没有跨过临界规模。

### V-L3 亲和网络放大引理

由 V-A5、V-A6 和 V-A7 可得：亲和策略一旦识别出同策略或可转译同策略的共同体，就能通过联盟、标准兼容、共同教育、贸易网络、航图共享、救援互认或风险担保放大扩张速度。

使用后果：四大文明的早期吞并不一定主要表现为军事征服，也可以表现为标准接入、救援依赖、封存托管、实验担保或合约债务。

### V-L4 亲和不等于融合引理

亲和策略降低同策略文明的合作和吸纳成本，但不消灭主权、历史身份、地方利益、组织惯性和资源分配争议。

使用后果：同属联邦式公共责任链的成员可以保留自治；同属环带式长期阈值体系的共同体可以争夺封存解释权；同属裂谷式进化共同体的派系可以内斗；同属商盟式合约网络的企业可以激烈竞争。

### V-L5 排他策略规模瓶颈引理

排他策略在早期可能具备很强的军事效率和行动一致性，但它难以低成本吸纳同类，也难以把被征服者转化为稳定共同扩张网络。面对多个亲和策略形成的标准、救援、封存、实验或合约网络时，排他策略更容易在治理成本、维护成本和合法性成本上被压制。

使用后果：排他型文明可以作为失败成核、短命霸权、边境军阀、封闭残余或局部威胁存在，但不应自动成为当前四大文明之一。

### V-L6 起源星区结果化引理

由 V-A4 和 V-L3 可得：四大文明的起源星区不是预先给定某策略的天然苗床，而是在某策略成核后被回溯性定义出来的吸引盆地中心。

使用后果：可以写“联邦起源星区”“环带核心弧带”“裂谷成核带”“商盟港链核心”，但这些区域应被解释为策略胜出后的制度地理，而不是策略产生前的环境命定。

### V-L7 内部多样性保留引理

由“主导制度生态”的定义可得：一个文明采用某扩张策略，只表示该文明多数机构围绕该策略组织，不表示内部不存在反对派、异策略残留、边境例外、地下网络和未完全转译的地方传统。

使用后果：成熟四大文明应保留内部摩擦。否则文明会变成单一理性主体，失去社会学可信度。

### V-L8 四大数量历史性引理

理论能推出“多种稳定策略可并存”，但不能单独推出“必然正好四种”。当前“四大文明”应解释为在早期历史窗口中，有四种亲和型策略成功跨过文明级临界规模；其他策略可能存在，但没有形成同等级文明吸引盆地。

使用后果：四大文明是当前主星海的历史结果，不是宇宙法则规定的固定数量。

### V-L9 残余痕迹必要引理

由 V-L2、V-L6 和 V-L7 可得：如果早期起源区曾均匀分布多种策略，那么被吞并、转译或压缩的策略应留下制度残余、考古遗迹、边境少数共同体、旧航图、禁用标准、口述争议或内部非主流学派。

使用后果：写文明起源时，应留下失败策略的痕迹，而不是让胜出策略抹掉所有历史层次。

## 五、定理层

### V-T1 四文明多吸引子定理

由 V-A1、V-A3、V-A4 和 V-A7 可得：当前四大文明不是四种底层环境的产物，而是同一底层环境中四个成功跨过临界规模的亲和型扩张策略吸引子。

四大文明对应吸引子如下：

| 文明 | 策略吸引子 |
| --- | --- |
| 星海联邦 | 公共责任吸纳型扩张。 |
| 永恒环带 | 长期阈值维护型扩张。 |
| 深渊裂谷 | 适应性进化型扩张。 |
| 商盟 | 可结算流通型扩张。 |

### V-T2 等价稳定共存定理

由 V-A3 和 V-L2 可得：四大文明的长期并存说明它们在当前星海底层环境中收益无明显绝对优劣。它们的冲突来自边界识别、风险解释、转译成本和扩张逻辑不兼容，不来自某一方天然正确、另一方天然错误。

### V-T3 意识形态策略化定理

由扩张策略定义和 V-L3 可得：四大文明的整体意识形态不是装饰性价值口号，而是为各自扩张策略提供合法性、教育语言、制度组织和亲和识别的上层结构。

使用后果：意识形态必须落到谁被视为同类、谁可被吸纳、谁必须被改造、谁承担代价、谁有权拒绝。

### V-T4 亲和边界定理

由 V-A5、V-A6 和 V-A7 可得：四大文明都会亲和“同类”，但它们对“同类”的定义不同，因此同一第三方文明可能同时被一个文明视为可吸纳对象，被另一个文明视为危险异策略对象。

使用后果：跨文明冲突应围绕“这个共同体应该进入哪种亲和协议”展开，而不是只围绕资源占有。

### V-T5 前身路线重解释定理

由 V-T1 和 `前身文明扩张策略推导.md` 可得：公共开拓型、长期维护型、深层接触型和航运合约型前身，不是四种天然起源环境的产物，而是同一底层环境中四类早期任务组织率先成核后的历史表现。

使用后果：该文件中的“公共开拓型前身”“长期维护型前身”“深层接触型前身”“航运合约型前身”仍可保留，但应理解为历史微态和组织任务的成核结果。

## 六、四文明扩张策略与意识形态

### 1. 星海联邦：公共责任吸纳型扩张

联邦式扩张不是单纯占领新星系，而是把新区域、新族群、新遗产和新风险纳入可记录、可救援、可审计、可追责、可共享的公共责任链。

| 维度 | 内容 |
| --- | --- |
| 扩张对象 | 航图、救援链、公共锚点、居民登记、职业认证、遗产样本链、跨族群公民权。 |
| 亲和对象 | 接受公共航图、救援义务、主体权利、职业认证、遗产审计和事故复盘的共同体。 |
| 排斥对象 | 私占航图、隐瞒危险、选择性弃救、私刻遗产、把 AI/数字人格/克隆体工具化、把前线特例常态化的共同体。 |
| 吸纳方式 | 公共航图接入、救援互认、成员星区协议、职业标准互认、舰队保护、遗产共享审查、公民权承认。 |
| 意识形态 | 程序化共存主义，或公共开拓主义。 |
| 核心口号式命题 | 未知可以被探索，但必须留下能让后来者获救、复盘和申诉的记录。 |

联邦的“亲和”不是无条件接纳，而是要求对方承认公共责任链。一个文明可以保留地方传统、族群身份和内部自治，但如果它拒绝危险记录、救援义务和主体权利，联邦就会把它视为异策略或危险策略。

联邦的优势：

- 容易吸纳多种族、多政体和多技术路线。
- 能积累长期信任、公共航图和跨星区救援能力。
- 对高风险遗产的处理更容易形成公开复核和跨机构协作。

联邦的风险：

- 程序成本高，扩张速度可能慢于前线需求。
- 公共标准可能压低地方传统和边境小工坊。
- 审计和救济链可能被资源雄厚者利用，形成庇护型精英。

### 2. 永恒环带：长期阈值维护型扩张

环带式扩张不是快速占有，而是把区域转化为可长期维护、低扰动、可跨代继承、可封存复查的稳定圈。

| 维度 | 内容 |
| --- | --- |
| 扩张对象 | 生态阈值、恒星生态维护、低扰动航段、封存线、长期观测站、人格连续医疗、居住资格。 |
| 亲和对象 | 接受长期阈值、生态稳定、封存程序、低波动发展和人格连续原则的共同体。 |
| 排斥对象 | 不可逆扩张、炫技式改造、绕过封存令、传播未净化深层信号、把短期收益置于跨代稳定之上的共同体。 |
| 吸纳方式 | 守望协议、封存托管、生态配额、长期居住资格、巨构维护席位、阈值教育和低风险技术共享。 |
| 意识形态 | 守望延续主义，或稳态秩序主义。 |
| 核心口号式命题 | 文明的第一义务不是增长，而是让未来仍然可居住、可继承、可复原。 |

环带的“亲和”指向可长期维护者。它可以与外部共同体合作，但合作前会要求对方接受阈值、观察期和封存边界。

环带的优势：

- 能减少不可逆事故，保留生态和人格连续。
- 擅长长期工程、恒星生态维护、灾害预警和封存体系。
- 能在其他文明高波动扩张后提供稳定避难与修复秩序。

环带的风险：

- 封存权可能被制度精英垄断。
- 长期稳定可能滑向迁徙限制、身份固化和人生路径锁死。
- 技术更新慢，面对突发外部压力时反应可能不足。

### 3. 深渊裂谷：适应性进化型扩张

裂谷式扩张不是把稳定区复制到新区域，而是把高风险环境、异常样本、改造失败和深层接触转化为可记录、可继承、可继续试验的适应能力。

| 维度 | 内容 |
| --- | --- |
| 扩张对象 | 深层边界、异常航图、改造共同体、样本库、适应性工坊、失败记录、实验医疗和高危前哨。 |
| 亲和对象 | 接受改造、风险承受、失败记录、深层接触、实验担保和适应性继承的共同体。 |
| 排斥对象 | 把封存当作唯一安全答案、否认进化权、伪造同意、隐瞒污染扩散、拒绝记录风险却占用成果的共同体。 |
| 吸纳方式 | 派系担保、导师项目、试验资格、样本共享、改造医疗、失败者安置、深层站共同驻留。 |
| 意识形态 | 记录化进化主义，或实验共同体主义。 |
| 核心口号式命题 | 文明不是保存当前形态，而是制造能承受更深未知的新形态。 |

裂谷的“亲和”不是对所有冒险者友好，而是对愿意把风险、失败和变形纳入可继承记录的人友好。它排斥的不是谨慎本身，而是把封存变成唯一答案、把失败数据抹除、把风险外包给无记录个体的做法。

裂谷的优势：

- 擅长异常知识、深层以太适应、灵核边界和极端环境技术。
- 能吸纳被其他文明排除的改造者、失败者和高风险探索者。
- 在未知边界上最容易产生突破。

裂谷的风险：

- 真实同意容易被派系、债务、导师关系和医疗依赖污染。
- 失败者安置可能被突破叙事挤压。
- 污染扩散、人格损伤和样本武装化会让外部文明难以信任。

### 4. 商盟：可结算流通型扩张

商盟式扩张不是先占领土地，而是通过港口、航运、合约、保险、授权密钥、维护包、信用记录和仲裁，把区域并入可交换、可履约、可追责的流通网络。

| 维度 | 内容 |
| --- | --- |
| 扩张对象 | 港链、维修链、授权航图、保险评级、能源额度市场、灵械模块、合约仲裁、债务和信用网络。 |
| 亲和对象 | 接受合约、信用、风险定价、授权密钥、接口标准、维护责任和仲裁执行的共同体。 |
| 排斥对象 | 破坏授权密钥、逃避赔付、伪造质保、制造不可定价风险、拒绝履约又要求网络收益的共同体。 |
| 吸纳方式 | 港口服务、维护外包、保险承保、贸易信用、企业学院、行会证书、星门窗口合约和平台接入。 |
| 意识形态 | 合约流通主义，或信用网络主义。 |
| 核心口号式命题 | 文明不是领土，而是持续交换、履约、结算、维修和再分配风险的网络。 |

商盟的“亲和”表现为接入而非归化。只要对方愿意履约、接受仲裁和维护责任切分，就可以被视为合作对象；但一旦对方破坏信用基础设施，商盟会高度排斥。

商盟的优势：

- 扩张速度快，能让边境迅速获得补给、维修、教育、医疗和航运服务。
- 能跨文明平滑价格、组织保险和拆分复杂责任。
- 模块化能力和服务网络最成熟。

商盟的风险：

- 合约自由容易滑向平台依附和维护债务。
- 风险可定价不等于风险合法，商盟容易包装高危技术。
- 密钥冻结、保单拒付和债务绑定可能直接影响主体稳定。

## 八、对现有设定的修订口径

1. 旧有“前身路线”可以保留，但应被解释为同一底层环境中的历史微态成核结果，不是四种起源环境天然分化。
2. “起源星区”应写成策略胜出后形成的吸引盆地中心，而不是策略形成前的环境原因。
3. 四大文明都采用亲和策略，但亲和不等于和平。它们亲和的是可转译为自身策略者，排斥的是无法转译或拒绝转译的异策略者。
4. 被吞并、被转译或被压缩的早期文明不代表策略天然低劣，应在历史、制度和边境中留下残余痕迹。
5. 排他策略可以作为失败霸权、边境威胁、短命军国体、封闭残余或古文明遗留样本存在，但不是当前四大文明主干。
6. 成熟文明内部不能写成单一理性主体。四大文明的策略是主导制度生态，不是每个成员个体的绝对信念。

## 九、写作使用规则

- 写四大文明起源时，先写多策略混杂分布，再写某策略在局部成核，最后写吸引盆地扩大和异策略被吞并、转译或压缩。
- 写四大文明意识形态时，必须落到亲和条件和排斥条件，避免只写“联邦共存、环带稳定、裂谷进化、商盟契约”的口号。
- 写跨文明冲突时，优先问“这个人、星港、样本、殖民地或共同体应该进入哪种亲和协议”，再决定资源、战争和法律冲突。
- 写起源星区时，不要让地理环境承担决定性解释。可以写局部事故、战争、先发技术、组织记忆、航图网络和制度标准如何把星区重塑为某文明的吸引盆地。
- 写被吞并文明时，保留遗迹、旧制度、旧航图、非主流派系、方言、禁用标准、地下市场或考古争议。
- 写正式世界内文本时，不必出现“吸引子”“收益地形”“历史微态”等工作台术语；可转译为“早期路线”“公共责任链”“守望阈值”“适应记录”“信用网络”等世界内表达。
