# 协作工作台

本目录用于存放辅助世界观扩写、口径冻结、状态追踪、推导蓝图、写作指南、素材、导出、版本记录和追溯资料。这里的文件可以具有权威参考价值，但它们不是世界观正文专题。

稳定正文应整理进 `10_世界观正文/00_总纲/` 或 `10_世界观正文/10_专题设定/`；面向网站或读者展示的导览、事件、地图和作品应整理进 `20_展示发布层/`。

## 目录分工

- `10_设定推导与蓝图/`：关键科技状态表、四文明态度矩阵、经济/航行/地理/信息治理蓝图、理论层级整理、前身文明推导和星海疆域生成器。
- `20_写作指南与样稿库/`：协作规则、专题写作辅助与检查清单、草稿与候选、Matrees 写文本技能目录、世界内文本风格样稿和候选稿。
- `30_素材与导出/`：图片素材、素材登记表、导出文档和海报文件。
- `80_原始与废弃追溯/`：原始资料、废弃资料和已撤回稿。只用于追溯来源，不直接作为当前采用口径。
- `90_版本记录/`：版本索引、CHANGELOG 和版本管理规则。

## 使用原则

- 不把状态表、矩阵、蓝图、检查清单或样稿直接当作世界观正文。
- 不把“保守推导”写成已经生效的硬制度；稳定后再整理进正文专题。
- 正式专题应优先保留世界内设定、制度结构、技术边界和示例对象；写作辅助、检查清单、模板和风格样稿统一放入本工作台。
- 每次新增设定时必须执行协作规则检查；若产生新硬口径、可复用写法、禁用写法、错误教训、引用路径或维护流程变化，应同步更新 `20_写作指南与样稿库/协作规则/` 中对应文件。
- 若工作台判断改变核心规则，应同步更新总纲、相关专题文件、版本索引和 CHANGELOG。
