# 写作指南与样稿库

本目录保存协作规则、写作辅助、检查清单、技能目录和风格样稿。它服务于创作和维护，不是世界观正文，也不是发布层内容对象。

## 目录分工

- `协作规则/`：文件入口、操作维护、创作原则、硬性口径、专题写作与误读检查。
- `专题写作辅助与检查清单.md`：模板、检查清单、写作用途、旧口径修正和辅助规则。
- `舰队战战术体系写作指南.md`：舰队战位置变化、资源窗口、小型舰艇生存、机甲协同和武器破坏力平衡的工作台指南。
- `草稿与候选/`：角色草稿、剧情年表草稿和视觉候选资料，不作为世界观正文。
- `Matrees写文本技能目录.md`：Matrees 写文本与富文本发布相关技能目录和发布检查规则。
- `世界内文本样稿/`：世界内报道、档案体、游戏文本、叙事风格和 Matrees 文本类型候选样稿。

## 使用原则

- 风格样稿只用于选型和训练写法；其中出现的事件、角色或细节不自动成为权威设定。
- 草稿与候选只作为工作底稿；稳定后再整理进正式专题或展示发布层对象。
- 若样稿中的内容后续被采用，应先整理进 `10_世界观正文/` 或 `20_展示发布层/` 的对应文件。
- 用户指出误读或口径错误时，先修正文档正文，再把可复用教训整理进协作规则或检查清单。
