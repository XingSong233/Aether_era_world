# 世界内叙事风格候选：第四批

本文件用于测试更接近小说正文的世界内叙事风格。以下八种风格已同步加入 `aether-world-report` 技能风格目录，可直接用对应 style-id 调用。

共同任务设定：恒历 472.18.07，岚桥二号锚站外段出现低功率回流异常。夜班小组需要在自由跃迁窗口开放前，完成临时锚标复检、锚链外段降级确认、旧锚标核心蓝封转运和人员接口复查。

## 候选风格二十三：贴身第三人称场景 / close-third-person-scene

适合写单个角色在任务现场的操作、判断和情绪，不写成报告，也不让旁白跳出去解释整个世界。

### 样例

安棠进外段廊桥时，先看脚边的隔离灯。

灯还亮着黄，不是故障黄，是允许靠近但不许接主功率的黄。她把手套腕口压紧，接口灵脉只开到最低识别，左手贴上检修柜的冷面板。柜体迟了半拍才回她，像夜班里被人叫醒的旧同事。

“3-17 端口延迟 0.6。”她说。

身后的值守员把数字写进板端，没有催。岚桥二号今晚有一艘医疗转运船要从这里出窗口，大家都知道时间不宽，但没人会在黄灯下说“加快”。安棠换了角度，指尖沿着面板边缘摸到一条细小的热痕。它没有烧穿，只是把手套里的触觉提示烫出一圈浅白。

她退开一步，断连，等自己的指尖先冷下来。

## 候选风格二十四：第一人称任务叙事 / first-person-task-scene

适合写某个任务参与者的个人口吻。它比日记更有现场动作，比档案更有人声。

### 样例

我那晚只负责看蓝封箱。

说起来不体面，但锚站夜班里最容易出事的不是看起来危险的东西，而是大家以为已经封好的东西。旧锚标核心被转运车推过来时，箱体外壳干净，蓝封也齐。我的任务是确认它没有在锚链复检时自己翻灯。

19:42，外段廊桥降到低功率。安棠他们在前面接检修柜，我站在后面，听箱体里的冷却片一层层安静下去。那声音很薄，像有人把碎纸夹在金属里慢慢抽出来。

19:47，第三只蓝封箱亮了一次白。

我没有碰它。我把手从推车把上拿开，按流程报位置、箱号后两位和灯色。后来调查员问我为什么不报完整编号。我说我们在检疫课上背过，异常箱先报位置，编号留给封存端读。调查员点了点头，把我的夜班记录盖成了有效。

那是我那周唯一一次没被老师挑错。

## 候选风格二十五：群像任务场景 / ensemble-mission-scene

适合写多人协同的任务现场，重点是角色分工、互相制衡和任务节奏。

### 样例

外段廊桥开放后，四个人同时动了起来。

安棠接检修柜，周启守蓝封箱，林月桥站在廊桥中线读航迹残留，医疗员傅澈没有进灯区，只把复检贴片按号码排在折叠台上。锚站广播每隔三十秒报一次窗口倒计时，声音被低功率墙滤得很平。

“临时锚标回流偏左。”林月桥说。

“能不能过医疗船？”调度端问。

安棠没有抬头。“先别问过不过，3-17 端口还没让我们看完。”

周启举手，没有插话。他面前第三只蓝封箱亮了白，持续一秒，又灭了。他把箱子推离主线半米，脚尖卡住轮锁，再把手背给傅澈看。傅澈扫了一眼他的接口贴片，确认没有冷痛反应，才朝调度端点头。

林月桥把航迹读数降采样保存，说：“窗口不取消，改低带宽接纳。医疗船先过，货船等二轮复检。”

广播停了两秒，随后把倒计时改成了新的颜色。

## 候选风格二十六：对话推进现场 / dialogue-driven-field-scene

适合靠对话表现关系、权限、专业习惯和现场压力。设定不由角色讲课，而是从他们怎么说、什么时候不说里露出来。

### 样例

“你别碰那个箱。”

周启的手停在半空。“我只是想把它推正。”

“它自己亮过白灯，你推正了也不算正。”安棠没回头，左手还贴在检修柜上，“报位置。”

“外段廊桥，蓝封三号，靠西侧轮锁。”

“别报三号。”

“为什么？”

林月桥从中线抬眼看他。“因为三号是我们叫它的名字，不是封存端认它的名字。”

周启闭嘴，重新报了一遍位置。傅澈把复检贴片递给他，示意他贴在腕侧，不要贴掌心。锚站广播报出医疗船接纳申请，倒计时从八分钟跳到七分钟。

“我是不是拖慢了？”周启问。

“你没碰它。”安棠说，“今晚这就算快。”

## 候选风格二十七：感官氛围场景 / sensory-atmosphere-scene

适合写地点、气氛和基础设施的存在感。它可以慢，但每个细节都应该和运行状态、维护或风险有关。

### 样例

岚桥二号的外段廊桥在低功率时会变得很长。

顶灯收成一线淡白，脚下的维护刻度只照亮最近三步。空气里有冷却剂的甜味，混着旧金属被反复清洁后的干涩。廊桥尽头的临时锚标没有声音，只有每次回流通过时，墙面会浮起一层极轻的细纹，像有人在透明板后面呼吸。

安棠不喜欢这个时段。不是害怕，是低功率会把太多东西留给人的身体去判断。检修柜没有把全部曲线送上主屏，只在她指尖下方给出断续的冷意。她数了三次，每次数到第十七下，那点冷都会迟到。

外港方向传来医疗船的接纳铃。铃声被中继压扁后失去尾音，只剩一个短促的开头。

安棠把手收回来，说：“3-17 需要旁路读数。”

廊桥上的白线随即暗了一格，像整座锚站也把呼吸放轻了。

## 候选风格二十八：高压行动场景 / high-pressure-action-scene

适合写事故边缘、抢修、救援和行动任务。紧张感来自限制，而不是角色无视限制。

### 样例

黄灯跳红时，安棠已经断连。

她的手离开检修柜，周启把蓝封箱轮锁踩死，傅澈拉开复检贴片盒。三个人没有等广播说完。外段廊桥的低功率墙先落下来，把他们和主锚链隔成两段白色空间。

“人员状态。”林月桥说。

“四人清醒，一人接口冷。”傅澈看周启。

“不是冷痛。”周启抬起手，“像被水冲了一下。”

“记录原话，不要改成无异常。”林月桥说。

调度端要求确认医疗船能否继续入窗。安棠盯着旁路表，数字在 0.4 和 0.9 之间来回跳。她知道再等两分钟，医疗船就要改道；她也知道这组读数如果被忽略，下一次亮红可能就在船体穿过锚标之后。

“降级接纳。”她说，“不挂货柜，不接主网，只给医疗船过。”

林月桥重复她的话，上传黑匣标记。红灯没有灭，但倒计时恢复了。

## 候选风格二十九：日常劳动场景 / ordinary-work-scene

适合写低烈度的工作、换班、社区服务和普通生活，让世界感从小事里出现。

### 样例

夜班结束时，食堂还没开热餐。

安棠把检修手套挂进消毒柜，柜门弹出一张小票：外段廊桥任务后，二十四小时内不建议接入高功率诊断灵械。她把小票折起来，塞进胸前口袋，排在她前面的周启正在和医疗员争一件事。

“只是一下白灯。”周启说。

“你当时在异常箱旁边。”傅澈把营养包推给他，“吃完再签离站。”

食堂屏幕上滚过晨间排班。医疗转运船已通过，货船延后六小时，三号候车廊的儿童课程改成录播。安棠看到 3-17 端口被标了橙色复检，明天下午还是乙班去。

她端起温水时，广播提醒旧线回收点九点开放。隔壁桌的厨工骂了一句，说又要有人拿十年前的转接线来问能不能抵扣。

安棠笑了笑。她希望他们都来问。

## 候选风格三十：事后余波场景 / reflective-aftermath-scene

适合写任务结束后的医疗、审计、疲惫、关系变化和未完全消化的后果。

### 样例

周启的腕侧没有留下伤口，只留下一块贴片形状的浅印。

复检室的灯比外段廊桥亮，他反而觉得不真实。傅澈让他把手放进低温环里，环内的水样介质没过指节，一点点带走接口残留的冷意。屏幕上跳出他的夜班记录：异常箱旁值守，未接触，报告有效，建议十二小时内暂停蓝封转运岗位。

“这算处分吗？”他问。

“算排班建议。”傅澈说，“处分不会给你热水喝。”

周启低头看水面。安棠隔着玻璃在外面签维护记录，林月桥把航迹降采样包交给封存端。每个人都在做自己的收尾，像刚才那盏红灯只是夜班里多出来的一道工序。

可他还记得蓝封箱亮白的一秒。那一秒短到不该留下声音，却在他耳后反复响。

他把另一只手伸进口袋，摸到没发出去的讯息草稿。等水温提示跳绿，他还是没有删。
