# 协作规则文件组

状态：采用
定位：工作台辅助资料，不是世界观正文专题。
入口：`AGENTS.md`

本目录承接原 `AGENTS.md` 中较长的创作规则、维护规则、引用路线和误读修正。根目录 `AGENTS.md` 只保留协作者启动时必须先读的入口说明；需要细节时，再按本目录文件分工查阅。

## 文件分工

- `文件入口与引用规则.md`：说明总纲、专题、工作台、发布层、图片素材和版本记录的权威层级，以及不同主题应优先参考哪些文件。
- `操作维护规则.md`：说明目录操作禁令、UTF-8 编码、图片素材、发布层、版本记录、冲突处理和新增设定时的同步检查。
- `创作原则与硬性口径.md`：说明世界观创作总原则、统一术语和当前不可违背的关键设定口径。
- `专题写作与误读检查.md`：说明能力、文明、角色、剧情的写作检查项，以及历史误读、旧口径和用户指出错误时的修正流程。

## 推荐阅读顺序

- 开始一次普通设定扩写：先读 `AGENTS.md`，再按主题进入 `文件入口与引用规则.md`。
- 涉及核心科技、文明制度、经济、航行、生命工程或 AI：同时检查 `创作原则与硬性口径.md`。
- 涉及文件移动、发布层、图片、版本记录或编码：检查 `操作维护规则.md`。
- 涉及能力设计、职业模板、世界内报道、误读修正或旧口径清理：检查 `专题写作与误读检查.md`。
- 需要详细模板时，继续查 `../专题写作辅助与检查清单.md`，不要把模板全文重复写进本目录。

## 新增设定时的协作规则检查

每次新增设定都必须执行协作规则检查。检查结果分两类处理：

- 如果只是普通细节扩写，不产生新硬口径、新禁忌、新引用路线或新维护流程，可不改本目录文件，但应确认没有违反现有规则。
- 如果新增设定产生可复用口径、禁用写法、误读修正、引用路径变化或维护流程变化，必须同步更新本目录中对应文件，并按 `操作维护规则.md` 检查 `CHANGELOG.md` 和 `版本索引.md`。
