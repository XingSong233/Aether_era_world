# 草稿与候选

本目录用于存放尚未整理为权威正文的角色草稿、剧情年表草稿、命名规则候选和视觉候选资料。候选资料可以作为后续创作、发布层拆分或正式专题扩写的参考，但不自动成为世界观正文设定。

## 目录分工

- `角色/`：主角草稿、候选角色设定卡和角色视觉候选。
- `剧情/`：主线阶段、剧情年表草稿和事件拆分前的工作底稿。
- `联邦人族命名规则候选.md`：泰拉星、泰拉人族及其个体姓名重命名前的三轨候选规则。

## 使用原则

- 稳定角色设定应整理进正式角色专题或展示发布层角色页。
- 稳定历史事件或剧情节点应拆成 `20_展示发布层/20_时间线与事件/` 下的独立事件文件。
- 图片路径、视觉转译要求和素材状态仍以 `90_协作工作台/30_素材与导出/图片素材/` 及素材登记表为准。
