# 个体能力体系设定综述

本章节概述了基于以太→灵核→灵脉的个体能力体系。该体系试图用工程化的方式解释一个生命体如何从感知以太到构建、运行、升级自己的超凡能力。本设定并非来自现实科学，而是世界观架构中的一套技术框架，但在设计时借鉴了魔法系统工程学中的原则，例如限制和代价比能力本身更重要[1]。

## 能力体系总览

- 以太：宇宙的基础介质，既承载能量，又承载信息。所有超凡现象和灵脉运作都建立在对以太的操控之上。以太的浓度、纯度和流动方式直接决定灵脉效率和灵核成长。
- 灵核：生命与以太交互的核心器官。它类似于生物的处理器和能量池，负责储存、调度以太并运行灵脉。
- 灵脉：封装具体功能的以太回路，是能力的工程化实现。灵脉可以刻在体内（内铭），也可以外置在器具中（灵械）。
- 接口灵脉：一种通用型灵脉，负责连接灵核与外置灵脉（灵械）。它本身不提供显性能力，但允许个体根据需要接入不同外置模块。
- 灵械：外置的人工灵脉装置，用人工材料和以太结构构成，承担高功率、危险或体积庞大的能力模块。其背后是灵械工程学的兴起。

## 灵核：能力的硬件基础

灵核是生命的核心，在能力体系中扮演着储存与运算以太的角色。为了便于刻画个体差异，可以将灵核属性拆分为以下六个维度：

- 容量：能储存多少以太，决定续航时间；
- 算力：单位时间内能处理多少信息和运算，决定可以同时运行的灵脉数量和复杂度；
- 吞吐量：每秒可输出的以太总量，决定爆发力；
- 稳定性：长时间运行灵脉而不发生紊乱的能力；
- 亲和度：吸收和感知以太的效率，影响修炼和环境适应；
- 可塑性：灵核结构可改造程度，决定未来成长空间。

不同灵核会表现出某些倾向性，例如感知倾向（适合感知类灵脉）或输出倾向（适合攻击和强化类灵脉）。灵核不仅由先天决定，也可以通过训练和资源投入进行成长，包括容量扩张、算力开发、吞吐强化和稳定性淬炼等。正如魔法系统设计强调限制和代价更能带来故事张力[1]，“低容量高稳定”“高算力低容量”等不同组合都会限制个体能力的构建路线。

## 灵脉：能力的工程模块

灵脉是对以太流和信息处理的精细雕刻。它不是单一的管路，而是一组有序的微结构。可以按层级将其拆分为：

- 灵纹：最小的功能单位，类似电路中的晶体管或代码中的指令。每个灵纹只承担极小功能（如吸收特定波段、偏转以太、稳定局部流）。
- 灵环：由多个灵纹组成的小型回路，具备基本功能，如感应环、放大环、滤波环、稳定环等。
- 灵节：几个灵环组合形成的功能节点，例如感知节点、运算节点、转换节点。它们能完成复杂一点的任务，如“热源捕捉”或“信号降噪”。
- 灵段：多个灵节串联成的局部功能链，如捕捉链、放大链、转译链等。
- 灵脉：完整能力模块，由多个灵段组成并经由接口接入神经或灵核。不同灵脉的功能、功耗、品质和风险差异巨大。

### 灵脉的基础功能

灵脉可以拆解为几类基础结构，这些结构被排列组合以产生不同的能力：

- 感应结构：捕捉特定类型的信息或能量，如光、声、热、以太波动、生命电信号等。
- 筛选结构：过滤噪声和无用信息，提高信号质量。
- 放大结构：增强微弱信号或输出，但过度放大会带来噪声和负载增加。
- 转换结构：将信息从一种形式转换为另一种形式，如将红外信号转换成可视化图像、将以太流转化为力场。
- 接口结构：将灵脉输出接入生物器官或神经，如视网膜、视神经、肌肉神经、灵核意识层等。
- 稳定结构：防止过载、反馈或神经伤害，设置安全阈值。
- 调控结构：允许使用者开启/关闭灵脉、调整参数、切换模式。

### 灵脉实例：红外视觉

以红外视觉灵脉为例，可以分成若干功能段：

- 赤谱感应段：在眼球或周围形成对红外波段敏感的以太场，捕捉不同强度和方向的红外辐射。
- 滤波与降噪段：识别背景温差、运动热源，剔除杂散热流，使有效信号突出。
- 放大段：利用赤谱增幅环和边缘强化纹增强热源强度和轮廓。
- 转译段：将红外信息映射为可理解的视觉信号（颜色、明暗、轮廓），并与正常视觉融合。没有这一段，使用者只能感到热源却无法形成图像。
- 神经接口段：根据设计将处理后的信号接入视网膜、视神经或视觉皮层。高阶版本可直接在脑内投射图像，不依赖眼睛。
- 调控段：允许使用者调节灵敏度、开启或关闭红外视觉，切换普通夜视/残热追踪等模式。

这一例子展示了灵脉工程的复杂性：同一功能可以有多种结构实现，带来不同的距离、分辨率、耗能与风险；合理的工程设计（例如加装稳定环、降噪环）才能使能力可用。

### 灵脉的铭刻位置

灵脉不一定刻在灵核内部。根据安全性和效率，可以分为：

- 灵核内铭：灵脉刻在灵核内部，反应快但风险高，通常用于核心或基础能力。
- 灵核外壳铭刻：刻在灵核表层或边缘，适合接口和稳定类灵脉，较安全但效率稍低。
- 神经并联铭刻：沿着神经系统构筑，适合感知、反射和运动控制类能力，但对神经适配要求高。
- 器官附着铭刻：附着在器官周围，例如眼、耳、心脏，适合局部强化能力如红外视觉、呼吸过滤、肌肉增幅。
- 体表/外场铭刻：刻在皮肤或生成外部以太场，常用于护盾、隐匿、外放能力。
- 外置灵脉（灵械）：高功率或大型结构外置成灵械，由接口灵脉连接。

### 灵脉兼容性与组合

并非所有灵脉都可以任意组合。兼容性包括以下方面：

- 同源兼容：同一设计体系的灵脉更易协同。
- 功能冲突：不同灵脉可能互相干扰，如高灵敏感知与情绪放大并用可能导致过载。
- 频率冲突：灵脉运行频率若不一致可能产生延迟或失控，需要调频或协调灵脉师校准。
- 认知冲突：信息过载会造成意识混乱，需要意识滤波或训练。

常见的组合模式包括：

- 串联组合：一个灵脉的输出成为另一个的输入，例如红外捕捉 → 目标识别 → 弹道预测。
- 并联组合：多个灵脉同时运行，如夜视+红外+声波定位，但负载较高。
- 反馈组合：灵脉的输出反过来调整自身或其他灵脉，提高精度或反应速度。
- 冗余组合：为重要能力设置备用灵脉，例如红外视觉失效时自动启用声波定位。

## 灵械与接口灵脉

随着技术发展，研究者发现不仅可以外置感知器，还可以将任何高功率、高风险或大型灵脉外置化，形成“灵械”。在传统修炼社会，只有资质高、灵核强者能体内铭刻强大灵脉，而资质差者无法承载多个能力。灵械的出现改变了这种状况：

- 灵械定义：外置的人工灵脉装置，由人工材料、以太回路和灵脉结构组成，是“离体化的能力模块”。它本质是外脉+物质载体+控制接口。
- 接口灵脉：通用型灵脉，用于识别、同步、转译和保护灵核与灵械之间的连接。当灵械受损或污染时，接口负责断连保护，防止反噬。
- 灵械分类：
- 感知型灵械：红外目镜、声波定位、生命扫描等；
- 运算型灵械：灵算核心、战术推演器，分担灵核算力；
- 强化型灵械：外骨骼、肌肉牵引系统；
- 输出型灵械：以太炮、冰凝枪、屏障发生器；
- 防御型灵械：护盾发生器、污染过滤器；
- 医疗型灵械：生命维持、灵核修补、污染清除等；
- 工程型灵械：结构分析、材料雕刻、灵脉刻录工具。
- 灵械优势：节省体内灵核空间，降低铭刻风险，易于升级和维护，能承载极端结构，支持团队协作和工业生产。
- 灵械限制：依赖外部供能，有延迟与带宽限制，易受干扰或劫持，维护成本高。使用者需要进行接口灵脉适配和断连训练。

灵械的普及催生了接口时代。强者不再是体内刻满灵脉的人，而是能够高效协调灵核、接口与灵械系统的人。社会也因此从个体修炼转向以灵械工业、接口标准和模块化装备为基础的超凡文明。

## 能力构建与资源模型

为了描述能力构建的限制，可以将灵核资源抽象为多个槽位：

| 资源 | 说明 |
| --- | --- |
| 结构空间 | 可以刻录多少灵脉结构，决定可装载能力的种类与数量。 |
| 算力线程 | 可以同时运行多少灵脉，决定并发能力。 |
| 吞吐通道 | 可以同时提供多少以太输出来维持能力运转。 |
| 注意力窗口 | 使用者意识能处理多少反馈信息，防止认知超载。 |
| 稳定冗余 | 发生错误时的缓冲余地，决定灵脉运行安全性。 |
| 接口带宽 | 能与灵械交换多少数据和能量，影响外置模块表现。 |

当构建能力时，需要在这些资源之间做取舍。例如，一个侦察者可以选择在体内铭刻高稳定性的通用接口灵脉，并将红外视觉、声波定位、残热追踪等感知能力外置成灵械；而一个传统武者可能优先分配结构空间和吞吐通道，在体内铭刻多种强化和攻击灵脉。

## 成长与训练路径

个体能力的形成不是一步完成，而是经历训练与适配：

- 灵核检测与基础训练：评估灵核参数，进行以太感知和冥想训练，提升容量、算力与稳定性；
- 第一轮铭刻：根据目标职业和灵核条件，选择安全的基础灵脉，如通用接口、基础感知、防护类灵脉；
- 适配训练：学习调控灵脉、处理反馈信息，适应身体和神经的变化；
- 架构优化：根据实际使用情况调整灵脉组合，增加调频灵脉、意识滤波灵脉，避免冲突；
- 灵械接入：通过接口灵脉连接外置灵械，训练同步率和延迟补偿；
- 职业定型与进阶：通过长期实战、实验或研究，选择专业路线，并不断升级灵脉或更换灵械模块。

不同职业（如战斗者、研究员、医生、侦察者、工程师）会有不同的训练重点。研究员更重视算力开发和接口操控，战斗者更重视吞吐强化和反应训练，医生更看重稳定性和精密控制。

## 副作用、损伤与心理影响

能力使用的代价是体系内的重要设计原则[1]。常见问题包括：

- 灵核类损伤：过度吞吐导致灵核裂纹，污染导致浑浊，长时间高算力运行导致过热等。
- 灵脉类损伤：灵脉错位、堵塞、烧蚀或串扰，造成能力失准、疼痛或永久性能下降。
- 神经类副作用：感知类灵脉可能导致幻视、幻听，动态预测可能导致肢体误动或反射失控。
- 灵械副作用：接口磨损、共鸣依赖、断连眩晕，或因灵械损坏而产生“能力空洞感”。
- 心理与人格影响：长期依赖特定灵脉会反向塑造使用者性格。例如，长期依赖危险感知灵脉可能使人警觉而焦虑，长期使用思维加速灵脉可能使人与普通交流脱节，长期使用情绪抑制灵脉则可能减弱共情能力。

因此，能力体系的设计不仅要提供强大的功能，还必须考虑限制、代价与风险，使角色在成长中面对选择和牺牲，这也呼应了魔法系统设计中“限制比能力更重要”的理念[1]。

## 总结

本章梳理了一个以 以太→灵核→灵脉→灵械 为核心的个体能力体系。灵核提供硬件基础，灵脉通过工程化的微结构构建具体能力，接口灵脉和灵械扩展了能力的边界，而资源模型、兼容规则、训练路径和副作用构成了体系的约束。类似于一些修真设定中蕴含灵气的“灵脉”是一种能量富集的地脉[2], 本体系也强调以太流动和存储的重要性；同时借鉴了魔法系统设计中注重限制与代价的思想[1], 使能力系统更具逻辑性和戏剧张力。通过这种分层设计，世界观中的个人能力可以不再是神秘的“天赋”，而是可以学习、设计、维护和升级的工程产物。
