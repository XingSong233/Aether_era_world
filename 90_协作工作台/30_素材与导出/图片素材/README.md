# 图片素材管理

本目录用于集中管理“以太纪元”世界观相关图片素材，包括参考图、生成图、概念草图、成品图和废弃追溯素材。图片素材只作为视觉参考或后续美术生产资料，不自动成为世界观正文设定。

## 目录结构

- `素材登记表.md`：所有进入本目录的重要图片素材登记入口，记录来源、用途、授权状态、关联设定和采用状态。
- `原始素材/`：用户提供、历史留存或外部取得的原始图片；不得直接覆盖，修订时另存到其他目录。
- `参考素材/`：用于风格、结构、构图、服装、器物、场景等参考的图片；必须登记来源和使用边界。
- `生成素材/`：由生成工具产出的图片，包括概念草图、角色图、场景图、器物图等。
- `成品素材/`：已整理、命名稳定、可用于展示、文档插图或后续导出的图片。
- `废弃/`：不再采用但需要追溯的图片素材；不得批量删除。

## 使用原则

- 图片素材不直接决定设定正文。只有当相关设定写入 `10_世界观正文/00_总纲/` 或 `10_世界观正文/10_专题设定/` 后，才视为正式世界观内容。
- 新增图片时，应优先登记到 `素材登记表.md`，至少记录文件路径、素材类型、来源、授权状态、关联设定、采用状态和备注。
- 外部图片只能作为参考素材使用，除非明确具有可用授权；不得把未授权图片整理为成品素材。
- AI 生成图片应记录生成日期、生成工具或模型、提示词摘要、后期处理情况和对应设定用途。
- 图片文件命名应尽量稳定，推荐使用 `主题_对象_状态_日期` 格式，例如 `联邦_边境星港_概念草图_20260531.png`。
- 替换或重绘图片时，不直接覆盖原文件；保留旧图并在登记表中标注“废弃”“替代版本”或“待复核”。
- `.docx`、网页、设定集或展示稿引用图片时，应优先引用 `成品素材/` 中的稳定文件。
