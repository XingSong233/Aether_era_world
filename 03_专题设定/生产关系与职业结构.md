# 生产关系与职业结构

状态：采用
基准：`../02_总纲/以太纪元世界观总纲.md`、`../00_项目管理/设定工作台/关键科技与生产关系状态表.md`、`../00_项目管理/设定工作台/经济体系设计蓝图.md`、`航行自由跃迁与开拓秩序.md`、`经济体系.md`、`能力系统/职业与灵脉模板库.md`、`文明圈对照表.md`

本文件是“以太纪元”生产关系与职业结构的正式专题入口。它回答：关键科技成熟后，生产资料由谁掌握，劳动如何组织，职业如何分层，事故责任如何追溯，以及不同文明如何把同一技术纳入不同制度。

职业不是技能标签，而是生产资料、训练路径、接口授权、维护链、风险承保、法律责任和文明制度共同形成的社会位置。能力系统说明个体和灵械如何工作；本专题说明这些能力如何进入产业、机构、劳动关系和阶层结构。

## 一、生产资料重心

| 生产资料 | 依托科技 | 主要控制主体 | 社会后果 |
| --- | --- | --- | --- |
| 以太场与能源基础设施 | 以太炉、以太输能网络、以太场清洁与屏蔽、护盾发生器。 | 城市能源部门、舰队、殖民地、环带封存机构、商盟能源承包商。 | 能源权与场域治理权成为基础权力；城市、舰船、殖民地和高危工区都依赖运维、熔断、屏蔽和事故复核。 |
| 接口与权限系统 | 接口灵脉、授权密钥、权限审计、标准数据库与维护记录。 | 标准署、行会、军方、商盟平台、保险机构。 | 能力社会的关键生产资料不是单件装备，而是合法接入、授权续期、维护记录和事故责任链。 |
| 高端制造设备 | 分子装配机、纳米制造、轨道工业环、材料相位工程、可编程物质。 | 工业联合体、军工船坞、商盟巨头、材料行会。 | 普通物品稀缺性下降，设计权、材料配方、能源额度、装配设备和授权密钥更重要。 |
| 星门、航道与空间锚 | 星门、以太洋流航行、空间锚、自由跃迁、星港调度和锚链中继。 | 星门枢纽政体、舰队、航运集团、殖民地工程机构。 | 星际流通受节点、航权、保险、安检、救援和航道窗口限制；航道维护者拥有战略地位。 |
| 灵械与职业套件 | 灵械、职业套件标准、民用规格、军用规格、商盟商品规格。 | 职业学校、军方、行会、灵械企业、租赁平台。 | 劳动者掌握技能，但工具、维护、授权和事故保险常由机构或平台控制。 |
| 数据、蓝图与风险模型 | 共鸣网络、灵算核心、风险建模与事故数据库、遗产蓝图转译。 | 科研机构、金融保险、遗产隔离、军方、商盟仲裁机构。 | 数据是航线、保险、职业准入、遗产复原和事故追责的依据；解释权本身成为权力。 |
| 生命与身体维护能力 | 灵核医学、完全再生医学、体内纳米机器人、克隆器官、人造子宫。 | 医疗体系、军医机构、保险医疗、生命工程机构。 | 身体伤残不再是多数劳动风险的终点，灵核损伤、医疗许可、再生资源和生育治理更关键。 |
| 智能主体与载体维护能力 | 人工智能、数字人格、心智核心、心智之种、人工灵核、生命结构承载。 | AI 公民、医疗机构、载体维护企业、公共登记机构、遗产隔离机构、商盟平台。 | 高等级 AI 和数字人格不是可任意复制的工具；生产关系围绕人格稳定、载体产权、人工灵核维护和劳动合同展开。 |
| 金融合约与审计基础设施 | 共鸣审计账本、智能合约、数字货币、能源额度、授权密钥、风险建模。 | 商盟金融机构、仲裁庭、银行、保险联盟、公共审计机构。 | 产权和合约可高度自动化、可追踪和可冻结，但仍需仲裁、司法和文明监管。 |
| 恒星生态与轨道级巨构 | 恒星工程、轨道工业环、星港、生态穹顶、星舰船坞、重力控制。 | 永恒环带、工业星区、船坞联合体、殖民地政府、商盟巨头。 | 大型基础设施是长期治理对象；维护权、居住资格、能源配额和防护优先级会形成稳定阶层。 |
| 生育与人口工程 | 胚胎设计、人造子宫、跨种族生育技术、克隆技术、基因优化。 | 医疗体系、生育服务机构、殖民地人口部门、伦理委员会、家族与种族治理机构。 | 生育不再被自然条件完全限制，但灵核无法被出生前稳定定制；高端优化和跨种族定制服务仍会制造准入差异。 |
| 高危样本与封存库 | 深层以太采样、污染隔离、沙箱测试、单向转译、污染熔断。 | 联邦远征、环带封存、裂谷实验体系、商盟合约勘探。 | 样本价值由隔离、转译、继续研究权、封存令和事故责任共同决定。 |

## 二、劳动组织变化

| 领域 | 被替代或压缩的劳动 | 新增或强化的职业 | 关系变化 |
| --- | --- | --- | --- |
| 第一产业 | 大量传统种植、牧养和露天采掘体力劳动。 | 生态种植师、星壤调理员、矿脉探测员、采掘机群调度员、殖民生态包维护员。 | 土地本身不再是唯一核心资产，生态设备、矿区数据、环境许可和机群维护权上升。 |
| 第二产业 | 重复装配、低精度加工、危险现场操作。 | 分子装配配方师、材料相位工、灵械装配工、工业安全复核员、轨道环调度员。 | 工业权力集中在设备、配方、能源和标准接口上；工人承担异常处理和安全复核责任。 |
| 第三产业 | 基础客服、低端物流搬运、部分常规文书。 | 共鸣网络基础设施人员、远程存在调度员、能力法律顾问、航线保险评估师、灵械租赁经纪人。 | 服务业围绕身份、权限、风险和体验展开；劳动者价值来自判断、同意管理和信任维护。 |
| 军事与公共安全 | 部分前线侦察、巡检和低风险执法。 | 机甲机师、承载舱调校师、高过载事故调查员、自动执法监督员、舰队接口官、污染隔离技术员、军事伦理审查员。 | 自动系统可执行高风险动作，但责任仍回到智慧个体和授权链；高速载人机甲还会把医疗复检、轨迹审计和过载授权变成军事生产关系的一部分。 |
| 教育与职业训练 | 单纯知识传授和一次性资格考试。 | 职业学校导师、训练诱导师、接口带宽教练、维护记录审查员、事故复盘教官。 | 教育灌注不存在，职业能力仍需训练和经验；学校、军校和企业学院成为能力生产机构。 |
| 医疗与照护 | 大量传统伤残照护和单一器官治疗。 | 灵核医生、再生医学师、体内纳米维护师、心理回响隔离师、职业复健评估员。 | 医疗从修复身体扩展为维护职业能力与人格连续性。 |
| 智能主体协作 | 把 AI 或数字人格当作低责任调度工具。 | AI 劳动权益协调员、人工灵核维护师、心智核心维护师、人格稳定评估员、载体登记员、L5 多线程任务审计员。 | 高等级 AI 是智能主体；协作关系需要处理合同、医疗、人格稳定、劳动权和责任主体。 |
| 金融、合约与审计 | 人工核对票据、部分常规合同执行和低风险赔付流程。 | 共鸣账本审计员、智能合约中止官、能源额度经纪人、授权密钥合规员、跨文明仲裁师。 | 金融岗位从记账转向解释、纠错、仲裁和风险拆分。 |
| 生育与人口治理 | 自然生育条件限制下的部分家庭照护与殖民人口规划。 | 胚胎非灵核设计师、人造子宫监管员、跨种族生育顾问、亲属身份法务、克隆体权益审查员。 | 社会分工围绕医疗安全、身份归属、克隆体权利、殖民适应和公平准入展开。 |
| 恒星生态与巨构运维 | 分散式设施维护、短周期建筑维修和单一城市后勤。 | 恒星生态维护员、轨道环调度员、巨构安全复核员、居住资格管理员、能源配额调度员。 | 大型工程需要跨代维护和持续审计，劳动者价值来自长期稳定和危机优先级判断。 |
| 遗产与深层以太 | 个人探险式寻宝和非组织化采样。 | 深层潜航员、单向转译员、封存库管理员、污染熔断工程师、样本产权法务。 | 高危探索必须团队化、制度化；样本流向由授权、隔离、复检、封存和文明协议决定。 |

## 三、职业结构层级

| 层级 | 典型职业 | 进入门槛 | 社会位置 |
| --- | --- | --- | --- |
| 基础服务岗位 | 家用灵械服务员、公共安全巡检员、基础教育导师、社区医疗技术员。 | 基础职业训练、低功率接口许可、民用维护记录。 | 与普通居民接触最频繁，决定基础服务权的实际质量。 |
| 标准职业岗位 | 生态种植师、灵械装配工、星港物流调度员、职业学校导师、以太炉运维员。 | 职业套件训练、标准认证、维护周期、事故保险。 | 是星海社会日常运转的主体，数量大、责任清晰、可跨地区流动。 |
| 高端专业岗位 | 灵核医生、灵脉工程师、材料相位工、航线保险评估师、共鸣网络基础设施人员。 | 长期教育、高等级接口训练、专项许可、职业信用和复检。 | 掌握复杂系统的解释权与维护权，是技术阶层核心。 |
| 授权阀门岗位 | 标准审查官、事故调查员、跨文明仲裁师、能源额度调度员、居住资格管理员。 | 反利益冲突审查、法律和技术双重认证、公共或行会授权。 | 决定能力、设备、航权、能源和巨构资源能否进入合法运行。 |
| 高危接触岗位 | 深层以太探查队、遗产隔离技术员、污染熔断工程师、军用接口官、肃正协议预警员。 | 高危任务许可、隔离训练、心理复检、战后复检和任务担保。 | 稀缺且受敬畏，同时被监管机构警惕；退役后常需长期复健。 |
| 灰色与非法岗位 | 黑市改装师、走私维护商、伪密钥刻录者、污染样本中介、证据清洗师。 | 非正规经验、非法数据库、地下维护网络。 | 填补正规制度无法、过慢或过贵的需求，也制造后门、污染和追责断裂。 |

职业上升不只取决于天赋。灵核六维会影响适配路径，但真正决定长期职业位置的是训练机会、接口带宽、维护记录、认证等级、事故历史、担保机构和文明身份。

## 四、阶层与冲突来源

| 冲突来源 | 推导口径 | 可写表现 |
| --- | --- | --- |
| 权限阶层 | 普遍基本供给存在，基础生存不是主要贫困来源；差异来自接口权限、航权、教育机会、医疗许可、居住场域和风险暴露。 | 边境居民有基本生活，却缺少合法接口升级、稳定中继和高质量职业训练。 |
| 维护阶层 | 高端设备和能力模块需要持续维护、校准、复检和责任记录。 | 维护员、标准审查官、事故调查员和保险评估师能决定生产资料是否继续合法运行。 |
| 平台依附 | 灵械租赁、授权密钥、商盟商品规格和风险定价降低接入门槛，也可能制造债务和冻结风险。 | 个体可以租到高端模块，却长期依赖质保、仲裁、密钥续费和事故赔付条款。 |
| 安全阈值冲突 | 四文明对危险技术容忍度不同。 | 同一深层样本在联邦是受控研究，在环带是封存对象，在裂谷是进化材料，在商盟是高风险资产。 |
| 身份与人格边界 | AI、数字人格、克隆、人造子宫和跨种族生育改变身份来源，但复活和记忆编辑不存在。 | 数字人格备份、非法克隆体、军用克隆体和生命结构承载 AI 的权利问题需要制度处理。 |
| 自动化后的劳动价值 | 通用机器人和分子装配机压缩体力劳动，但不能替代职业经验、伦理责任和事故判断。 | 低技能岗位减少，认证岗位增多，公共供给兜底与机会竞争并存。 |
| 信息治理边界 | 共鸣网络、预测算法、感官共享和监控存在，但不同文明对干预权和隐私边界态度不同。 | 地方机构、保险公司或企业可能越权使用预测结果，造成准入歧视和救济困难。 |
| 高危职业外包 | 私人军事公司、商盟安保、合约勘探和遗产开发使风险可被市场分配。 | 合同拆分责任后，事故仍会回到谁授权、谁复检、谁允许样本进入正式系统。 |
| 生育设计边界 | 胚胎设计已成体系但无法定制灵核，非灵核优化和跨种族定制仍会制造公平问题。 | 基础疾病筛除可普及，高端优化、特殊环境适应和跨种族定制服务受费用、补贴和权限影响。 |
| 自动合约救济权 | 智能合约可以自动执行冻结、赔付、租赁到期和授权撤回，但必须保留救济入口。 | 密钥冻结后，争议焦点是职业生存权、比例审查和临时功能恢复。 |
| 能源额度与航权阶层 | 能源额度、航权、星门窗口和中继稳定性会成为比货币更具体的机会门槛。 | 小企业能买设备，却拿不到稳定生产时段和合理航线保险。 |
| AI 公民劳动边界 | 高等级 AI 与数字人格可参与劳动和治理，但有寿命、人格稳定和载体维护限制。 | 联邦 AI 公民拒绝被平台拆成廉价工作副本；商盟平台以维护债务压低报酬。 |
| 巨构维护权争议 | 轨道级和行星级巨构成熟后，维护权、居住资格、能源配额和防护优先级成为核心资源。 | 一次护盾或生态系统检修会暴露管理者、能源部门、居民组织和保险方的冲突。 |

## 五、四文明生产关系倾向

| 文明圈 | 生产关系倾向 | 容易产生的矛盾 |
| --- | --- | --- |
| 星海联邦 | 把灵械、AI 公民权、职业训练、高危探索和公共审计纳入标准化、授权制和救济链。 | 开拓效率与公共审查冲突；边境机会贫困、AI 劳动权、遗产审查和非法克隆体救济容易引发争议。 |
| 永恒环带 | 把关键生产资料视为生态与安全阈值的一部分，重视封存、长期观测、恒星生态维护和巨构稳定。 | 稳定优先会压制技术扩散、AI 公民权、个体改造自由和外来模块准入。 |
| 深渊裂谷 | 把灵核改造、深层以太接触、生命工程和 AI/数字人格载体实验视为进化路径。 | 技术突破快，但实验劳工伦理、污染扩散、克隆体权利、改造失败者安置和遗产依赖更突出。 |
| 商盟 | 把灵械、航权、授权密钥、能源额度、保险、样本、心智核心和维护服务商品化。 | 市场降低接入门槛，也制造债务、密钥冻结、责任拆分、自动合约滥用和私人武装越界。 |

## 六、与职业模板库的关系

本专题处理宏观生产关系和跨产业结构；`能力系统/职业与灵脉模板库.md` 处理具体职业模板、常见能力配置、训练路径和岗位写法。

新增职业时，建议先在本专题中判断它属于哪种生产关系节点，再进入职业模板库填写能力与训练细节：

- 它控制或维护哪类生产资料：能源、接口、航权、制造设备、医疗资源、数据模型、样本、巨构还是合约。
- 它属于第一产业、第二产业、第三产业、跨产业岗位、高危岗位还是灰色岗位。
- 它依赖哪些许可、维护记录、保险、授权密钥、职业信用或文明身份。
- 它失败时造成个人事故、职业事故、城市事故、舰队事故、样本污染还是文明级风险预警。
- 它在四文明中的合法性、地位和救济渠道是否不同。

## 七、写作检查清单

- 不要把职业写成单个角色的技能栏；先说明它在生产链、维护链、授权链和事故责任中的位置。
- 不要把自动化写成劳动完全消失；应说明哪些劳动被压缩，哪些判断、维护、认证和追责岗位被强化。
- 不要把灵核六维写成完整劳动能力评级；联邦尤其不能把灵核六维当作唯一身份或职业根基。
- 不要把灵械租赁写成纯粹便利服务；同时检查密钥冻结、维护断供、债务和救济权。
- 不要把高危样本、遗产蓝图或军标模块写成普通商品；先确认隔离、转译、封存、承保和文明监管。
- 不要把 AI、数字人格或 L5 多线程写成无责任工具；必须处理人格稳定、载体维护、权利地位和任务审计。
- 不要把胚胎设计写成能力职业预设；它只能处理非灵核层面的疾病、体质、环境适应和跨种族兼容。
- 不要让某个文明制度成为唯一正确答案；联邦、环带、裂谷和商盟都应有优势、代价和内部矛盾。
