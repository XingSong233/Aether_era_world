# 星际航行和开拓秩序

状态：采用

本目录承载“以太纪元”星际航行、自由跃迁、空间锚、航图、星际定位、航权治理、边境救援和开拓秩序相关正式专题。

当前文件：

- `航行自由跃迁与开拓秩序.md`：星际交通结构、星门、以太洋流、空间锚、自由跃迁、航权、星港治理、开拓权链和四文明航行差异的总入口。
- `星际定位技术原理.md`：说明星舰在共鸣中继不稳定、以太航图失效或暗区行动中，如何依靠物质层天文观测确认自身坐标并绘制物理星图。

维护原则：

- 物理星图只说明天体坐标、结构和观测置信度，不自动等于可通行航路。
- 以太航图必须继续记录以太潮汐、深层扰动、空间锚状态、临时锚标、污染区和航迹残留。
- 通行航图必须叠加航权、保险、检疫、救援承诺、目的地接纳和事故责任。
- 新增航行、定位、测绘、锚链或开拓专题时，应优先放入本目录，并同步更新总纲、术语表、发布映射和版本记录。
