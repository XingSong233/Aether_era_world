# 星海地理与星图节点

状态：采用
基准：`../../02_总纲/以太纪元世界观总纲.md`、`../../00_项目管理/设定工作台/星海地理与星图节点设计蓝图.md`、`航行自由跃迁与开拓秩序.md`、`星际定位技术原理.md`、`../文明圈对照表.md`、`../深层以太设定.md`

本文件是“以太纪元”星海地理、文明疆域尺度和主星图显示节点的正式专题入口。它用于说明四文明处在银河中的哪一类区域、主星图应显示哪些核心星系、文明实际控制范围与公共星图节点之间如何分层，以及跨星系航行时间应如何估算。

本专题不替代航行、自由跃迁、空间锚和开拓秩序专题。航行制度、航权和殖民承认见 `航行自由跃迁与开拓秩序.md`；星际定位、物理星图和测绘误差见 `星际定位技术原理.md`。本专题只冻结宏观星海尺度、文明分布和地图节点口径。

## 一、基本口径

| 口径 | 设定说明 | 写作后果 |
| --- | --- | --- |
| 银河是正常大型螺旋星系 | 银河整体可按棒旋星系或多悬臂螺旋星系理解，银心存在超大质量黑洞。 | 银心、主悬臂、支臂、悬臂间隙和稀疏星区都可以成为远期扩展方向。 |
| 当前主星图不是整片银河 | 当前四文明活动区只是某条主悬臂外缘与一条支臂交汇处的一段文明星海。 | 不需要在主星图中画出完整银河，也不需要把银心黑洞放进当前地图。 |
| 文明版图是群岛式 | 星门、以太洋流、空间锚、暗区和深层扰动共同决定可达性。 | 文明疆域不应画成均匀实心色块，而应表现为核心星区、航道、节点和影响带。 |
| 主星图显示核心节点 | 星图上的点是具名核心恒星系、星门枢纽、星区首府、空间锚、港口、前哨或事件地点。 | 一个地图节点不等于一个普通恒星系，也不代表文明实际恒星系总量。 |
| 实际疆域保留更大背景 | 四文明实际定居、登记、保护、开拓或影响的恒星系数量远多于地图显示节点。 | 后续剧情可以持续新增星系，不会破坏既有地图尺度。 |
| 航行时间按航路等级估算 | 距离只是基础变量，星门窗口、锚链、以太洋流、自由跃迁、排期、复检和许可都会改变实际耗时。 | 不应把“相邻恒星系”固定写成统一时间，也不应让自由跃迁抹平所有交通差异。 |

当前主星图建议范围为约 `6500 x 4200 x 800` 光年。常规文明节点大多位于银河盘面上下 `250` 光年内；深层异常、暗区边缘、古文明遗产点和裂谷前哨可以扩展到盘面上下约 `800` 光年。

银心黑洞不在当前主星图中。它可以在地图边缘以方向标记呈现，例如“银心方向：约 18000 光年外”。银心方向可作为远期设定接口，用于古文明核心遗迹、肃正协议源头、禁限航区或深层以太异常高密度区。

## 二、整体星海尺度

| 层级 | 建议规模 | 说明 |
| --- | ---: | --- |
| 当前主星图显示范围 | 约 6500 光年宽、4200 光年高、800 光年厚 | 用于公共星海图、静态地图和创作时的尺度参照。 |
| 已知文明活动星域 | 直径约 5000-7000 光年 | 不是连续占领区，而是航道、核心星区、边境斑块和高危隔离区交织而成。 |
| 四文明合计稳定定居恒星系 | 约 15000-25000 个 | 包括成熟殖民地、自治星系、核心生态星系、贸易自治港和长期实验定居点。 |
| 有正式航图和救援备案的恒星系 | 约 80000-120000 个 | 不一定常住人口多，但可被定位、登记、承保、救援或限制通行。 |
| 暗区、深层边缘和遗产隔离区 | 数量不稳定，远多于显示节点 | 可作为后续剧情、探查任务和地图扩展的主要来源。 |

这个尺度下，星海足够大，边境和暗区仍然遥远；但核心文明之间仍然能发生贸易、战争、移民、外交和连续事件，不会变成彼此完全隔绝的孤岛。

## 三、四文明疆域与显示节点

主星图建议初始显示约 `440-460` 个节点。其中四文明各显示约百个核心节点，中立、争议、暗区和遗产相关节点显示 `40-60` 个。这个数量适合公共星海图和静态地图表达，也足以表现文明格局。

| 文明或区域 | 主星图显示节点 | 实际成熟定居恒星系 | 其他登记或影响范围 | 疆域形态 |
| --- | ---: | ---: | ---: | --- |
| 星海联邦 | 100-120 个，建议初始 110 个 | 约 3500-5000 个 | 边境殖民地、军管站、科研站和资源站约 8000-12000 个；航图登记恒星系约 50000-70000 个 | 最大连续片区，核心密集，边境向外伸出多条开拓臂。 |
| 永恒环带 | 80-100 个，建议初始 90 个 | 约 1200-2200 个 | 高度维护的生态核心约 150-250 个；封存、观察和限制开发恒星系约 5000-9000 个 | 沿稳定星群和生态带展开的弧带或环带，不是均匀圆形领土。 |
| 深渊裂谷 | 80-100 个，建议初始 90 个 | 约 700-1300 个 | 实验站、改造殖民地和深潜前哨约 2500-5000 个；高风险登记点约 20000-40000 个 | 沿深层以太异常带、暗区边缘和危险航道分布，稀疏但纵深大。 |
| 商盟 | 100-130 个，建议初始 110 个 | 约 2500-4000 个 | 核心合约恒星系约 300-500 个；服务覆盖恒星系约 10000-20000 个；商业影响可达登记市场 50000 个以上 | 不表现为连续国土，而是贸易港、结算枢纽、护航节点和契约自治港网络。 |
| 中立、争议、暗区和遗产节点 | 40-60 个 | 不按文明统计 | 作为事件、禁航、遗产隔离、虫群或深层异常入口 | 分布在文明接壤处、悬臂间隙、暗区边界和旧航道断裂区。 |

显示节点应优先选择具有政治、航行、经济、军事、生态或剧情意义的恒星系。普通农业星系、资源星系、居住星系和小型研究站可以存在于设定背景中，不必全部出现在主星图上。

## 四、空间分布

当前主星图应表现为“悬臂段文明区”，而不是平面棋盘。

| 区域 | 星海位置感 | 主要文明或节点 |
| --- | --- | --- |
| 悬臂内侧密集带 | 恒星密度高、星门修复较早、以太洋流和共鸣中继成熟。 | 星海联邦核心区、部分商盟枢纽和早期星门城市。 |
| 稳定弧带 | 以太场平缓、生态维护价值高、灾害阈值清晰。 | 永恒环带的生态核心、封存门户和长期维护节点。 |
| 支臂交汇区 | 航道交叉、商贸频繁、政治边界复杂。 | 商盟港口网络、联邦边境枢纽、中立仲裁港和跨文明市场。 |
| 悬臂断裂与暗区边缘 | 以太扰动强、深层信号活跃、旧航道断裂。 | 深渊裂谷实验城邦、深潜前哨、遗产隔离区和虫群风险点。 |
| 悬臂外缘稀疏区 | 恒星密度下降、救援链变薄、开拓成本增加。 | 联邦远征线、商盟特许开拓地、失联殖民地和新剧情节点。 |

四文明不应按简单四象限切开。联邦可以拥有最大连续核心区；环带应呈弧形或带状；裂谷应像沿异常带生长的长裂隙；商盟应穿插在交通交汇处和跨文明边界上。

## 五、星图节点分层

| 层级 | 显示规则 | 典型对象 |
| --- | --- | --- |
| `display_tier: 1` | 主星图常驻显示。 | 文明首府、星区首府、主星门枢纽、一级空间锚、舰队总港、核心生态星系、商盟结算总港。 |
| `display_tier: 2` | 缩放或筛选时显示。 | 区域港、二级空间锚、舰队基地、封存门户、深潜站、重要贸易港、跨文明仲裁港。 |
| `display_tier: 3` | 剧情、事件或专题需要时显示。 | 新开拓星系、事故地点、失联殖民地、遗产样本点、临时锚标、局部战场。 |
| `display_tier: 4` | 背景存在但默认不显示。 | 普通居住星系、资源点、小型研究站、农业和工业附属星系。 |

新增剧情星系时，通常从 `display_tier: 3` 开始。若后续成为长期政治、航行或经济中心，再提升为 `display_tier: 2`；只有文明核心、主枢纽或反复影响全局的地点才应提升为 `display_tier: 1`。

## 六、航行时间基准

单次自由跃迁或星门通行的物理过程可能只有数十分钟到数小时，但星际旅行的实际时间还包括离港排期、锚定解计算、能源额度申请、航权许可、检疫安检、跃迁窗口、抵达复检、保险确认和目的地接纳。

| 航线类型 | 典型距离 | 客运时间 | 货运时间 | 说明 |
| --- | ---: | ---: | ---: | --- |
| 同一核心星群内 | 10-80 光年 | 12 小时-3 天 | 2-6 天 | 航权成熟、救援覆盖强，普通人可视为远程公共交通。 |
| 稳定锚链航道 | 80-300 光年 | 2-7 天 | 5-14 天 | 需要空间锚、星港排程和常规复检。 |
| 跨星区航线 | 300-1000 光年 | 1-3 周 | 3-6 周 | 常见于工作迁移、军事调动、区域贸易和跨星区医疗转运。 |
| 核心到边境 | 1000-2500 光年 | 3-8 周 | 2-4 个月 | 需要多段航行、补给、锚链复核和边境检疫。 |
| 跨文明远程航线 | 1500-5000 光年 | 1-3 个月 | 3-8 个月 | 时间受跨文明许可、仲裁、保险和检疫影响很大。 |
| 暗区或深层边缘探索 | 不稳定 | 3 个月-1 年以上 | 不适合常规货运 | 以可撤离、可隔离和可复测为目标，不以速度为目标。 |

普通人跨邻近成熟恒星系通常需要 `1-3` 天；从联邦核心去边境通常需要 `1-2` 个月；商队跨文明贸易通常按数月一轮计算；军用快速调动可以明显更快，但连续行动会受维护、补给、政治许可和污染复检限制。

## 七、写作和地图使用边界

- 不要把主星图当成文明全部恒星系清单；它只是具名核心节点、航行节点和事件节点的可视化层。
- 不要把文明疆域画成均匀实心边界；应优先表现星门枢纽、空间锚、航道、核心星区、边境斑块和影响范围。
- 不要让银心黑洞占据当前主星图中心；当前文明星海只是一段悬臂文明区。
- 不要把距离直接等同于旅行时间；航权、窗口、锚链、保险、复检和深层扰动都会改变耗时。
- 不要一次性命名所有恒星系；后续剧情和事件需要的新地点，应按显示层级逐步加入。
- 不要让地图节点只有政治属性；每个核心节点最好同时拥有航行、产业、生态、军事、风险或剧情功能中的至少一种。
