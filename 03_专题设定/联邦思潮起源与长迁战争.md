# 联邦思潮起源与长迁战争

状态：采用  
基准：`../02_总纲/以太纪元世界观总纲.md`、`文明圈对照表.md`、`星际航行和开拓秩序/航行自由跃迁与开拓秩序.md`

本专题用于解释星海联邦的思潮为什么发展成现在的形态，以及联邦系六族为什么能够融为一个文明。它不是把六族写成天然互补的六个不完整文明，也不是把融合写成没有资源竞争的理想主义结果。

联邦的起源来自一段长期战争、迁徙和制度重组史：星海摇篮区曾存在一个被后世称为“星冠体系”的霸权秩序。它控制主干星门、空间锚、航权和关键古文明遗产入口，把通行许可、殖民地承认、样本分配和人口迁徙变成统治工具。后来成为联邦前身的群体，包括被压迫的独立族群、流亡舰队、被俘虏者、被征发殖民地居民，以及星冠体系内部的起义者。

星海联邦的公共叙事中常说“开拓不是占有群星，而是让群星成为可共同生活的空间”。这句话背后的历史经验是：当星门、航图和遗产解释权被单一体系垄断时，整个摇篮区都会变成可以被切断、征发和遗忘的战场。

## 一、星冠体系

星冠体系不是现代联邦登记中的第七大族，也不是单纯的单一种族国家。它更像一个围绕星门主干线形成的军事航权体系，由核心统治阶层、附庸政体、征发舰队、军事殖民地、样本管理机构和星门维护军团共同构成。

星冠体系的优势不在于它拥有某种其他文明无法补齐的科技，而在于它较早控制了摇篮区的关键通道。

- 它掌握若干主干星门和星门间的空间锚链，可以决定哪些殖民地获得稳定通行。
- 它把航图、锚链校验数据和星门维护周期列为军政机密，使对手即使拥有舰队，也很难组织稳定远征和撤离。
- 它控制部分古文明遗产入口，以遗产解释权换取附庸忠诚和军工优势。
- 它把迁徙名额、宜居世界承认、殖民地补给和人口登记绑定到服役、纳贡和样本交付。
- 它用被俘人员、附庸工程师和边境殖民地居民维护高风险锚站，使战争责任和事故代价被层层转移。

因此，后来反抗星冠体系的族群并不是因为自身文明存在明显短板才联合。它们各自都能发展完整科技和社会制度，但在星门网络被封锁、工业腹地被切断、人口迁徙受控制的环境下，任何单一文明都很难长期保存全部力量。

## 二、长迁战争

后世把星冠体系扩张、反抗、流亡迁徙和最终崩解的漫长阶段统称为“长迁战争”。它不是一场持续高烈度正面会战，而是一组跨越数代人的封锁战、迁徙战、救援战、航图战和殖民地承认战争。

长迁战争早期，反抗者无法稳定保有所有母星和工业中心。许多族群采用“迁徙保存”策略：在作战同时测绘可居住星球、拆迁工业核心、转移学校与医院、复制人口底册、建立临时星港，并让舰队携带种子工厂、低功率医疗模块、职业教材和家族档案流动。

这种长期流动塑造了联邦最早的共同制度。

- 移动学校需要承认不同族群儿童、俘虏儿童和混血儿童的学习记录，不能每到一个星港就重新定义身份。
- 流亡医院必须记录灵核损伤、污染暴露、跨种族生育和战伤赔付，不能只按母星籍贯救治。
- 临时法院要处理舰队财产、样本归属、征发赔偿和失踪人口登记，不能等待战后再说。
- 航图队必须给后续舰队留下可复核路径、危险标注、补给状态和救援信标。
- 舰队指挥链要区分命令责任、维护责任、样本责任和个人违规，避免把事故全部推给前线个体。

在这个阶段，被俘虏者和星冠体系内部起义者成为关键变量。星冠体系内部并非铁板一块。长期征发、事故隐瞒、殖民地牺牲和遗产污染使部分附庸舰队、锚站工程军团、边境行政官和普通居民转向反抗。他们带来了星冠内部航权数据、维护口令、俘虏营位置、样本转运记录和军事清洗命令。

这些人不是战后被简单“宽恕”的敌人。他们在战争中已经进入流亡共同体：有人带路，有人拆除功率锁，有人救出被俘舰队，有人公开星冠的事故账本，也有人因为曾经参与压迫而接受战时审查和长期赔偿。

## 三、摇篮中枢遗迹

长迁战争的反转点是“摇篮中枢遗迹”的发现与转译。这个遗迹位于星冠体系主干星门控制区之外的异常暗带边缘，早期被误判为不可通行的古文明残区。

摇篮中枢遗迹没有直接赠予现代文明无成本武器。它提供的是更危险也更基础的信息：

- 一组失落星门与空间锚之间的旧校验关系，证明星冠公开航图故意隐藏了部分绕行解。
- 主干星门维护日志的古文明层级标注，帮助反抗者判断哪些节点可以脱离星冠调度独立复检。
- 深层以太回涌和锚链失稳的风险记录，解释了星冠体系长期隐瞒的若干殖民地事故。
- 一套遗产样本隔离原则，提示某些高收益样本会触发肃正协议或人格回响污染。
- 若干被封存的旧航标，使流亡舰队第一次能在星冠封锁线之外建立稳定救援路线。

遗迹转译由流亡者、被解放俘虏和星冠内部起义者共同完成。没有任何一方能单独证明全部数据可信：流亡舰队有现场测绘经验，被俘锚站人员知道星冠维护口令，内部起义者能解释军用调度格式，遗产学者能区分古文明警告和后世伪造。

这段经历直接塑造了联邦的遗产共享原则。遗产不是某个族群的战利品，而是会改变航权、救援、污染、样本归属和战争责任的公共风险源。谁接触遗产，谁就必须留下样本链、转译记录、事故预案和复核权限。

## 四、星冠体系崩解

摇篮中枢遗迹打破了星冠体系最关键的优势：信息垄断和通道垄断。反抗者获得绕行航图后，流亡舰队不再只能被动逃离；被俘殖民地开始接收稳定补给；内部起义者能把星冠的封锁令、清洗令和事故账本送出主干线。

星冠体系的崩解不是一次首都陷落就结束。它经历了三个过程。

第一，主干星门的政治垄断被瓦解。部分锚站宣布中立，部分维护军团拒绝执行封锁，部分附庸星区要求公开航图和迁徙名额。

第二，星冠内部责任链被拆开。战后清算区分政权命令、军用制度、样本管理、舰队行动、地方胁迫和普通居民生存选择。联邦后来反复强调“罪责按命令、授权、维护和执行链追溯，不按血统追溯”，正来自这一阶段。

第三，战后人口被重新登记。流亡者、被俘虏者、被解放殖民地居民、内部起义者、旧附庸族群和混血后裔不能再按旧星冠籍贯处理。新的登记制度把公民身份、居住地、责任记录、赔偿义务和自治权拆开，避免把某个来源群体永久固定为敌人、附庸或战利品。

星冠体系作为政治身份消亡。它的核心政权、秘密样本库、军用封锁制度和血统等级法被废除。部分战犯被审判，部分军政机构被拆解，部分殖民地获得自治，普通居民与起义者后裔则进入后来的联邦族群与公民体系。现代联邦不承认“星冠族”作为官方第七大族，但旧星冠体系中的生物族群、混血后裔和政治继承者并未被整体灭绝；他们被纳入新的登记、自治和公民制度中。

## 五、摇篮公约

长迁战争结束后，各方签署的一系列战后安排被合称为“摇篮公约”。它不是单一宪法文本，而是星门托管、战后赔偿、身份登记、救援优先、遗产审计和成员自治协议的集合。

摇篮公约确立了后来的联邦底层原则。

- 主干星门和关键空间锚不能被单一政体私有化，必须接受公共托管、定期复检和跨星区审计。
- 航图、危险标注、救援信标和污染通报必须进入公共记录；军事机密可以延迟公开，但不能让民用撤离和救援失去路径。
- 古文明遗产不直接按发现者所有处理；样本链、隔离等级、转译记录、事故预案和跨机构复核优先于战利品分配。
- 战后赔偿不只面向胜利政体，也面向被征发殖民地、被俘虏者、失踪人口家庭、污染事故幸存者和被迫迁徙群体。
- 公民身份以登记主体、居住关系、自治承认和责任记录为基础，不以旧敌我血统决定。
- 联合舰队保留跨种族编制和统一维护标准，防止任何成员单独垄断航权军力。

联邦成立不是六族忽然放弃竞争，而是把已经运行数代的战时生存程序转成正式制度。资源争抢仍然存在：航权税、宜居星球改造权、锚站维护费、遗产样本优先研究权、迁徙配额和军工采购份额都会引发争议。区别在于，争议不再通过封锁星门、抹除人口登记或私藏遗产来解决，而是进入议会、法院、舰队审计、赔偿基金、成员自治和跨星区仲裁。

## 六、六族如何融为一个文明

联邦系六族包括人类、狐人、龙裔、星灵、晶族、潮汐族及其混血后裔。这里的“六族”不是从星门复兴开始就始终平等并列的六个完整民族国家，也不是战后按血统硬凑出的宣传分类。它们是母星传统、流亡舰队、被解放殖民地、旧附庸共同体、内部起义者后裔和混合星港社会长期重组后的稳定文明身份。

六族融合有四个关键机制。

第一是流亡舰队和临时殖民地。长迁战争把原本分散的族群、家庭、职业团体和舰队编制放进同一套生存系统。孩子在移动学校读书，医生共享灵核病历，工程师共用锚站维护规程，法务人员共同处理失踪和赔偿。

第二是跨种族公共服务。基础教育、基础医疗、人口登记、职业执照、救援信标和迁徙资格必须跨族群有效，否则流亡社会无法运转。现代联邦的公共学校、舰队学院和职业认证体系都继承了这套口径。

第三是战后身份重组。被俘虏者、被征发者和内部起义者无法简单回到旧身份。摇篮公约承认他们可以通过居住、服役、赔偿、自治和公共登记进入新共同体。反血统定罪和责任链审计因此成为联邦公民观的一部分。

第四是跨种族家庭和混血后裔。人造子宫、跨种族生育和长期混居使六族边界在家庭层面持续重叠。联邦承认混血后裔与纯血后裔无本质差异，出生记录用于医疗和救济，不应用于职业标签或政治排除。

因此，六族融为一个文明并不意味着所有母星传统消失。龙庭帝国、晶环学邦、边境自治星区、旧流亡舰队社区和混合星港仍保留自己的政治传统和职业文化。联邦文明身份的核心在于：它们共享一套星海生活的公共操作系统，包括航图、救援、审计、公民登记、职业认证、遗产隔离和跨族群公共服务。

## 七、联邦思潮为什么发展成现在这样

联邦现在的思潮可以概括为“程序化开拓主义”：它赞赏开拓、远征和承担风险，但要求风险能够被记录、救援、复盘和追责。

这一思潮来自长迁战争留下的六条历史教训。

| 现在的联邦共识 | 历史来源 |
| --- | --- |
| 开拓必须保留救援链 | 长迁战争中，没有航图、信标和人口底册的迁徙队往往会被彻底遗忘。 |
| 航权和星门不能被单方垄断 | 星冠体系曾用主干星门和空间锚控制人口、补给和殖民地承认。 |
| 遗产必须共享审计 | 摇篮中枢遗迹能反转战局，也暴露了私藏遗产带来的污染和肃正风险。 |
| 责任按链条追溯，不按血统追溯 | 战后必须区分政权命令、军用制度、普通居民、被征发者和内部起义者。 |
| 标准化是救援条件，不只是行政偏好 | 流亡医院、移动学校、联合舰队和锚站维护都依赖跨族群可读记录。 |
| 公民身份高于旧敌我身份 | 被俘虏者、被解放殖民地居民、内部起义者和混血后裔都需要进入可救济体系。 |

这也解释了联邦的矛盾。它厌恶星冠式垄断，却仍需要舰队和中央标准；它赞赏前线冒险，却害怕前线把临时授权常态化；它重视公共救济，却会被救援、复检、赔偿和审计拖慢窗口；它强调反血统定罪，却必须处理战犯后裔、旧附庸机构和历史赔偿的长期争议。

联邦不是没有资源争抢的理想国。它只是把资源争抢从封锁、清洗和抹除，转化为航权仲裁、财政分配、成员自治、公共审计和事故赔偿。联邦六族之所以能成为一个文明，不是因为差异消失，而是因为它们在长迁战争之后接受了同一条底线：任何开拓、研究和竞争，都不能再让某一群人从航图、登记、救援和责任链中消失。
