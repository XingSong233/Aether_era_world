# 星际航行和开拓秩序

状态：采用

本目录承载“以太纪元”星际航行、自由跃迁、空间锚、航图、星际定位、星海地理、航权治理、边境救援和开拓秩序相关正式专题。

当前文件：

- `航行自由跃迁与开拓秩序.md`：星际交通结构、星门、以太洋流、空间锚、自由跃迁、航权、星港治理、开拓权链和四文明航行差异的总入口。
- `星际定位技术原理.md`：说明星舰在共鸣中继不稳定、以太航图失效或暗区行动中，如何依靠物质层天文观测确认自身坐标并绘制物理星图。
- `星海地理与星图节点.md`：当前主星图、悬臂段文明星海、四文明疆域尺度、核心节点显示规则和跨星系航行时间基准的正式专题入口。
- 前身文明扩张、边境战争和联邦制度根系推导已迁入 `90_协作工作台/10_设定推导与蓝图/前身文明扩张策略推导.md`，不作为本目录正式专题或正式建国史。

维护原则：

- 物理星图只说明天体坐标、结构和观测置信度，不自动等于可通行航路。
- 主星图只显示核心具名节点，不代表文明实际恒星系总量；新增剧情星系应按显示层级逐步加入。
- 以太航图必须继续记录以太潮汐、深层扰动、空间锚状态、临时锚标、污染区和航迹残留。
- 通行航图必须叠加航权、保险、检疫、救援承诺、目的地接纳和事故责任。
