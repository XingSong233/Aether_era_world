# 以太能与能源体系

状态：采用
基准：`10_世界观正文/00_总纲/以太纪元世界观总纲.md`、`经济体系.md`、`生产关系与职业结构.md`、`深层以太设定.md`、`星际航行和开拓秩序/航行自由跃迁与开拓秩序.md`、`90_协作工作台/10_设定推导与蓝图/关键科技与生产关系状态表.md`

本文件是“以太纪元”能源体系的正式专题入口，用于说明以太能的来源、获取、生产、储存、传输、调度和社会矛盾。它补足以太炉、以太输能网络、能源额度、星舰跃迁、星门枢纽、护盾、巨构和边境工业之间的底层关系。

以太能不是单一万能能源，也不是廉价无限背景。现代文明能稳定使用以太能维持城市、舰船、工业和民用灵械，但真正稀缺的是高相干、低污染、可调度、可储存、可承保、可跨设施接入的能源能力。能源问题因此不是“有没有能量”，而是“能否在正确地点、正确时段、正确品级和合法责任链下稳定使用”。

## 一、基本口径

| 口径 | 设定说明 | 社会后果 |
| --- | --- | --- |
| 以太能分层 | 以太能按相干度、污染风险、设施依赖和调度权限分为常规整流能、高相干以太能、深层/遗产级能源。 | 基础生活可稳定保障，但星舰跃迁、星门、护盾、巨构救灾和高端工业仍会争夺高品级能源。 |
| 以太炉不创造能量 | 以太炉从以太场势差、潮汐剪切、星门残网、深层回流或遗产炉芯中采集并整流能流。 | 炉址、场域、空间锚、清洁屏蔽和维护链决定产能，不是买到炉体就能无限供能。 |
| 相干度决定用途 | 同样总量的以太能，因相位稳定、信息噪声、污染残留和可同步程度不同，不能直接等价替换。 | 低品级能源可照明、供热和运行低功率灵械，却不能直接支撑自由跃迁、护盾过载或星门复检。 |
| 储存和远传受限 | 常规整流能可较稳定储存；高相干以太能会相位漂移，需要昂贵容器、调频维护和许可复检。 | 边境世界常有基础能源，却缺少能长期保存高相干能源的维护能力。 |
| 跨星际不直接远传能量 | 跨星域流动的是能源额度、炉芯产能预约、相位燃料、维护件、补给窗口和本地炉网接入权。 | 星门、航权、星港补给、物流和保险仍有意义，能源不能通过无限远程输送抹平地理差异。 |
| 灵核不作为燃料 | 灵核、人工灵核、AI、数字人格和运维员只参与校准、识别、调频、熔断、污染预警和责任判断。 | 能源产业需要专业主体，但不能把智慧个体写成人肉电池或廉价监控线程。 |

## 二、能源层级

| 层级 | 主要用途 | 来源与生产 | 稀缺点 |
| --- | --- | --- | --- |
| 常规整流能 | 城市民生、普通工业、低功率民用灵械、基础共鸣网络、生活重力场和日常交通。 | 由城市、舰船、工厂或殖民地以太炉从稳定区场差、恒星以太潮汐、低扰动以太流中采集并整流。 | 本地炉网、维护、调度时段、基础设施安全许可。 |
| 高相干以太能 | 星舰跃迁、舰队护盾、星港、轨道工业环、高端灵械、深层屏蔽、军事和灾害救援。 | 在常规整流基础上经过净化、相干化、相位锁定和复检，也可由高规格炉阵、空间锚群或星门枢纽能源网产生。 | 相位质量、储存漂移、输能同步、专业维护、事故承保和调度优先级。 |
| 深层/遗产级能源 | 星门修复、战略武器、古代枢纽接入、巨构应急、深层以太探查和高危封存工程。 | 来自深层以太井、古文明炉芯、遗产相位材料、失准星门能源接口或古代锚链节点。 | 污染、人格回响、肃正协议警戒、封存许可、文明审查和遗产依赖。 |

这三层不是简单的价格等级。常规整流能可以满足稳定社会的日常运转；高相干以太能决定一座城市、舰队、星港或工业环能否进入高功率状态；深层/遗产级能源则常常是战略资产或事故源，不进入普通民用能源市场。

## 三、来源与获取

### 稳定区场差

稳定区并非没有以太扰动，而是扰动可预测、可维护、可由公共基础设施长期调制。恒星以太晕、行星磁层、空间锚群、星港炉阵和城市输能网络之间会形成可采集的场强差和相位压力。城市级以太炉通常依靠这种来源维持基础供能。

稳定区场差的优点是污染低、维护成熟、适合民用和工业；缺点是高峰产能有限。核心星区的优势不是拥有无限能源，而是拥有更密集的炉网、更多调度冗余、更好的相位维护和更低的事故承保成本。

### 恒星以太潮汐

恒星活动会影响本地以太场。现代文明可以在恒星生态安全阈值内建设潮汐炉阵，把周期性场差转化为常规整流能或部分高相干能源。恒星以太潮汐不是任意改造恒星，也不能支持戴森式无限能源；主动大规模改造恒星仍属于禁限或文明级审查对象。

永恒环带在恒星生态维护和长期潮汐调度上最成熟，但也最强调限额、封存和长期风险。其他文明常指责环带以安全名义控制高质量能源窗口；环带则认为短期抽取会破坏恒星生态和巨构维护周期。

### 以太洋流剪切与空间锚群

以太洋流、空间锚、锚链中继和星门枢纽附近存在较强的流向差和相位剪切。它们既能支撑航行，也能成为高相干能源的来源。星门枢纽因此常同时是交通节点、能源节点、保险节点和军事节点。

这类能源获取必须避免干扰航道。过度抽取可能改变跃迁窗口、增加出口误差、影响航迹残留读取，甚至让空间锚进入失准状态。因此能源调度和航权调度经常被同一套星港或枢纽机构联合管理。

### 深层以太井

深层以太井是在受控窗口中接入深层回流或高压以太环境的高危设施。它能短时提供高品级能源和特殊相位材料，但能流中可能夹带结构化回响、污染残留、古代权限标记、人格回响或肃正协议相关信号。

深层以太井必须配备隔离井、单向整流舱、污染熔断阵列、样本链记录、任务后复检和封存预案。井口不是普通矿井或大发电站，而是高危任务设施。一次深层以太井事故可能同时表现为人员灵核过载、共鸣网络噪声、航道扰动、样本归属争议和跨文明审查。

### 古文明炉芯与遗产相位材料

现代文明可以独立制造城市、舰船和工厂级以太炉，但星门枢纽级、战略高相干级和部分古代网络接入级炉芯仍高度依赖古文明炉芯、遗产相位材料或可复用关键组件。古文明炉芯不是无成本外挂，通常带有权限层、污染记录、维护盲区或肃正协议风险。

谁控制炉芯，不只是拥有发电资产，也可能控制星门窗口、航权、殖民地补给、舰队补给和事故解释权。围绕炉芯的争夺常常在遗产隔离、军用接管、地方自治、环带封存和商盟承包之间展开。

## 四、以太炉与生产链

以太炉是能源基础设施，不是简单发电机。成熟以太炉通常执行以下链条：

| 环节 | 作用 | 主要风险 |
| --- | --- | --- |
| 采能 | 从场差、潮汐、剪切、深层井或炉芯接口引入原始以太能流。 | 入口失稳、航道扰动、深层回响夹带。 |
| 整流 | 把不稳定能流转为可控方向、频率和输出曲线。 | 输出波动、设备过热、局部炉段烧蚀。 |
| 净化 | 去除噪声、污染残留、异常权限标记和不可读回响。 | 污染漏检、净化损耗过高、样本链争议。 |
| 相干化 | 提高相位一致性，使能流可用于跃迁、护盾、星门、灵械阵列和高端工业。 | 相位漂移、同步失败、接口倒灌。 |
| 储存/降级 | 将能流存入容器或降级为更稳定的常规整流能。 | 高相干能源长期储存成本高，降级会损失战略用途。 |
| 分配 | 通过输能网络、舰船管线、星港补给或能源额度进入具体使用场景。 | 调度冲突、熔断延迟、授权滥用。 |

现代城市、舰船和工厂级以太炉可规模制造并进入公共标准。它们能保障大多数稳定文明的基础生活和普通工业。真正受限的是高相干相位锁定、深层污染处理、战略炉芯维护、远程补给和跨设施调度。

以太炉运行需要多种专业主体参与：以太炉运维员负责日常监控和维护，能源额度调度员负责时段和品级分配，场域清洁工程师负责降低背景噪声和污染，熔断复核员负责判断是否停炉，灵脉工程师和 AI 场域调度主体负责复杂相位模型。AI 或数字人格可作为能源调度主体，但其人格稳定、载体维护、人工灵核接口和劳动权必须进入责任链。

## 五、储存、传输与能源额度

### 储存

常规整流能可通过标准储能结构较稳定保存，适合城市应急、民用灵械、普通工业和舰船日常系统。高相干以太能则会随时间出现相位漂移。储存容器必须定期调频、降噪、复检和更换相位材料；长期储存若维护不足，会降级为常规整流能，或变成带污染风险的不可用能流。

因此，战略能源储备不是简单堆满电池。核心区的优势在于有维护人员、复检设备、相位材料、保险承保和调度冗余；边境世界常常有足够常规能源维生，却缺少可支撑跃迁、护盾和高端工业的稳定高相干储备。

### 传输

以太输能网络用于在城市、轨道环、星港、船坞、殖民地和舰船内部传输可用能流。短距输能相对成熟；星系级输能需要空间锚、中继站、场域清洁、相位同步和熔断节点；跨星际不直接远程输电。

跨星际能源调度通常表现为：预约目标星港或炉网的产能，运输相位燃料和维护件，购买补给窗口，获得本地炉网接入许可，或通过能源额度结算一段时间内的可用高相干输出。商船、舰队或殖民地即使在账面上拥有能源额度，也必须抵达可接入设施，并通过安全许可、检疫和复检，才能真正使用对应能源。

### 能源额度五元绑定

能源额度不是全星海通用货币，而是对某类可用能源的配额和结算权。每份正式能源额度至少绑定五项条件：

| 绑定项 | 说明 |
| --- | --- |
| 设施 | 对应哪座以太炉、输能网络、星港、舰船、工厂、殖民地或枢纽能源网。 |
| 时段 | 在哪个运行窗口、生产排程、星门窗口、救援窗口或灾害调度周期内有效。 |
| 品级 | 是常规整流能、高相干以太能，还是受限的深层/遗产级能源接触权。 |
| 许可 | 使用者、用途、功率、接口、维护资格和文明监管是否匹配。 |
| 安全状态 | 当前场域、污染、航道、熔断、保险和复检状态是否允许兑现。 |

同样一份额度在不同星区、炉型、场域和制度下并不等价。能源额度可以参与合约、赔付、航运和工业清算，但不能替代本域信用、星海清算单位、航权配额、授权密钥、维护记录或样本处置权。

## 六、产业与职业影响

能源体系让生产资料从单纯设备所有权转向“设施、时段、品级、维护和责任链”的组合权力。一个小企业可以购买旧分子装配机，却因缺少高相干能源时段和维护资格而无法稳定交付；一个边境殖民地可以维持基础生活，却因缺少星港补给窗口和相位维护件而长期无法建立高端工业。

常见职业与机构包括：

| 职业或机构 | 核心职责 | 社会位置 |
| --- | --- | --- |
| 以太炉运维员 | 监控炉体、能流、冷却、相位漂移、污染读数和熔断状态。 | 城市、舰船、工厂和殖民地能源生命线的日常维护者。 |
| 能源额度调度员 | 分配能源时段、品级、优先级和补给窗口。 | 连接工业、星港、舰队、公共服务和保险清算的关键岗位。 |
| 场域清洁工程师 | 处理以太场噪声、深层残留、屏蔽副作用和远程作业环境。 | 决定高危工区、医疗、遗产隔离和边境殖民能否长期运行。 |
| 熔断复核员 | 判断炉网、护盾、跃迁、星门或深层井是否进入停机、降级或隔离。 | 在事故中拥有强中止权，常与商业订单和军方调拨冲突。 |
| 相位材料维护师 | 维护储能容器、炉芯接口、相位晶格和高相干输能节点。 | 高端工业和星港补给链中的稀缺技术阶层。 |
| AI 场域调度主体 | 参与复杂相位模型、异常预警、调度模拟和责任记录。 | 可以提高调度效率，但必须保留人格权、维护权和任务审计。 |

能源职业的价值不在于体力劳动，而在于读懂复杂反馈、判断安全边界、执行熔断、承担签名责任和维护可承保记录。一次能源事故常常同时牵动运维、保险、司法、军方、居民自治组织和跨文明仲裁。

## 七、四文明差异

| 文明圈 | 能源路线 | 管控方式 | 主要矛盾 |
| --- | --- | --- | --- |
| 星海联邦 | 以公共炉网、舰队后勤、边境补给和标准化能源许可为主。 | 公共审计、军用调拨、救援优先、AI 公民劳动权和事故复检并存。 | 前线需要快速兑现高相干能源，后方担心污染、责任链和军用滥权。 |
| 永恒环带 | 擅长恒星潮汐调度、巨构维护、长期储备和低污染场域。 | 生态阈值、封存令、长期配额、居住资格和能源安全审查强。 | 稳定优先容易形成能源窗口垄断，被其他文明视为以安全名义限制扩散。 |
| 深渊裂谷 | 倾向探索深层以太井、高风险相位材料和改造者承受极限。 | 实验许可、派系担保、身体承受评估、样本接触权和失败者安置协议。 | 高产高危，容易在进化权、实验劳动、污染扩散和人格损伤之间失衡。 |
| 商盟 | 将能源额度、相位燃料、补给窗口、炉芯维护和保险高度商品化。 | 合同、授权密钥、平台风控、仲裁庭、私人安保和维护债权成熟。 | 市场提高接入效率，也制造债务依附、维护断供、责任拆分和高危能源包装销售。 |

## 八、主要冲突

| 冲突 | 触发条件 | 表现 |
| --- | --- | --- |
| 核心区能源时段争夺 | 星港、轨道工业、医疗转运、舰队补给和居民服务同时进入高负载。 | 基础生活不断供，但高相干能源窗口被优先级、保险和公共审查反复重排。 |
| 边境能源贫困 | 殖民地有常规能源和旧设备，却缺少高相干储备、相位维护链和低费率保险。 | 工厂能开机但不能稳定交付，舰船能补给但不能连续跃迁，护盾能展开但不能久撑。 |
| 自由跃迁补给争议 | 商船或救援舰账面有能源额度，却缺少目的地接纳、补给窗口或安全状态。 | 船主认为已付款，星港认为额度未满足设施、时段、品级、许可和安全绑定。 |
| 深层以太井事故 | 井口产能异常升高或净化系统漏检结构化回响。 | 人员灵核过载、共鸣网络噪声、航道漂移、样本归属和文明审查同时爆发。 |
| 古文明炉芯争夺 | 遗产炉芯可支撑星门、舰队或殖民地高相干能源。 | 联邦要求公共审计，环带要求封存，裂谷要求继续接触，商盟要求确认承包和保险收益。 |
| AI 运维争议 | AI 或数字人格承担长期能源调度、异常预警和多线程复核。 | 争议焦点落在其劳动权、人格稳定、载体维护债务和责任签名边界。 |

## 九、写作边界

- 不要把以太能写成廉价无限能源；基础供给稳定不等于高相干能源充足。
- 不要把能源额度写成全星海通用货币；它必须绑定设施、时段、品级、许可和安全状态。
- 不要把深层以太井写成大发电厂；它是高危战略设施，必须伴随污染、熔断、封存、审查和事故责任。
- 不要把古文明炉芯写成无成本外挂；它应伴随遗产隔离、权限盲区、维护难题或肃正协议风险。
- 不要把灵核、AI 或数字人格写成燃料；它们只能作为校准者、操作者、责任主体或受监管调度主体。
- 不要用跨星际远程输能抹平航权和物流；跨星域调度应表现为额度、材料、补给窗口、维护件和本地炉网接入权的组合。
