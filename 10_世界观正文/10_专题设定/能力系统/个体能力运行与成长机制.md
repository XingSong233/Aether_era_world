# 个体能力运行与成长机制

状态：采用  
基准：`10_世界观正文/00_总纲/以太纪元世界观总纲.md`、`能力系统索引.md`

本文件是“以太纪元”个体能力运行、获取、建立、训练、修复、抹除、变更、失效和成长机制的正式专题入口。它说明灵核、灵脉、接口灵脉和灵械如何在个体身上工作；灵脉内部结构的具体改进案例见 `灵脉结构原理与工程演变案例.md`，体内核心、接口层、外置灵械和团队系统的架构分工见 `灵脉架构演变.md`，标准化、许可、认证、商品化和军用规格见 `灵脉与灵械标准化体系.md`，结构继承、分支演化和断代接枝见 `灵脉谱系设定.md`，职业应用见 `职业与灵脉模板库.md`。

## 一、能力体系总览

个体能力建立在“以太 -> 灵核 -> 灵脉 -> 灵械”的工程链条上。

- 以太：能量、信息和意识共振的基础介质，是所有灵脉运作和共鸣网络的底层环境。
- 灵核：智慧生命与以太交互的核心器官，负责承载、调度、运算和稳定灵脉结构。
- 灵脉：封装具体能力的以太信息回路，可自然生成、诱导训练或工程铭刻。
- 接口灵脉：连接灵核与外置灵脉、灵械或大型系统的通用灵脉，负责识别、同步、转译、保护和断连。
- 灵械：外置的人工灵脉装置，用于承载高功率、高风险、大体积或需要频繁维护升级的能力模块。

能力不是单纯天赋，而是灵核参数、灵脉架构、灵械配置、训练路径和心理适配共同形成的技术系统。强者的核心优势不在于堆叠更多灵脉，而在于能在有限资源内搭建稳定、可维护、可升级的能力架构。

## 二、灵核六维模型

灵核位于脑干和脊髓之间，是灵核、神经与以太三者耦合的中心。所有智慧生命生而具有灵核，但参数倾向不同。

| 维度 | 含义 | 主要影响 |
| --- | --- | --- |
| 容量 | 可储存和承载多少以太与灵脉结构。 | 续航、装载上限、中大型灵脉承载能力。 |
| 算力 | 单位时间内处理信息和运行灵脉的能力。 | 并发、复杂操作、推演与精密控制。 |
| 吞吐量 | 每秒输出以太的总量。 | 爆发力、高功率输出、强化与防护强度。 |
| 稳定性 | 长时间运行灵脉而不紊乱的能力。 | 安全阈值、持续作战、医疗与工程精度。 |
| 亲和度 | 吸收、感知和适应以太环境的效率。 | 环境适应、感知灵敏度、训练效率。 |
| 可塑性 | 灵核结构可改造程度。 | 成长空间、后续铭刻、跨体系兼容。 |

常见取舍：

- 低容量高稳定：适合医生、工程师、维护员等高精度低风险职业。
- 高算力低容量：适合研究、分析和指挥，但不适合长期高功率输出。
- 高吞吐低稳定：爆发力强，容易过热、裂纹或失控。
- 高亲和低算力：环境感知强，但需要外置灵械或团队系统补足处理能力。
- 高可塑低稳定：成长空间大，但铭刻和改造必须分阶段进行。

灵核可以通过训练、药剂、神经反馈、灵械辅助和受控铭刻成长，但成长不是无代价扩容。每次扩展都会占用恢复周期，增加维护需求，并可能改变使用者的感知习惯和心理状态。

## 三、灵脉工程结构

灵脉是能力的工程模块，不是单一管路，而是一组可设计、可组合、可维护的微结构。

| 层级 | 说明 |
| --- | --- |
| 灵纹 | 最小功能单位，负责吸收、偏转、稳定、放大等局部功能。 |
| 灵环 | 多个灵纹组成的小型回路，如感应环、滤波环、稳定环。 |
| 灵节 | 多个灵环形成的功能节点，如感知节点、运算节点、转换节点。 |
| 灵段 | 多个灵节串联成的局部功能链，如捕捉链、转译链、反馈链。 |
| 灵脉 | 完整能力模块，由多个灵段组成并接入灵核、神经、器官或灵械接口。 |

灵脉基础结构包括：

- 感应结构：捕捉光、声、热、以太波动、生命电信号等输入。
- 筛选结构：过滤噪声，突出有效信号。
- 放大结构：增强微弱信号或输出，但会提高负载与失真风险。
- 转换结构：把输入转译成可使用的信息或物理效果。
- 接口结构：接入视网膜、视神经、肌肉神经、意识层或灵械控制层。
- 稳定结构：限制过载、反噬、污染和神经损伤。
- 调控结构：支持开启、关闭、参数调整和模式切换。

### 红外视觉示例

红外视觉灵脉可以由多个功能段组成：

- 赤谱感应段：在眼部或外置镜组周围捕捉红外波段。
- 滤波与降噪段：区分背景温差、运动热源和杂散热流。
- 放大段：增强热源轮廓，但必须控制噪声和过曝。
- 转译段：把红外信息映射为颜色、明暗、轮廓或意识层图像。
- 神经接口段：接入视网膜、视神经、视觉皮层或灵械显示接口。
- 调控段：切换夜视、残热追踪、低功耗扫描等模式。

同一能力可以有多种结构实现。体内版本反应快但维护风险高；外置灵械版本更易升级，但依赖接口带宽、供能和抗干扰能力。

### 灵脉功能分类参考

功能分类只说明灵脉主要做什么，不说明它从哪条原型发展而来，也不决定它是否标准化、定制化或属于某条谱系。功能分类和发展谱系必须分开：同一功能可以由不同谱系实现，同一谱系也可能分化出多个功能方向。

常见功能分类包括：

| 功能分类 | 主要目标 | 常见资源压力 | 常见风险 |
| --- | --- | --- | --- |
| 感知类 | 捕捉光、声、热、生命信号、以太波动或异常回响。 | 亲和度、注意力窗口、筛选结构。 | 感知残留、误读环境、反馈过量。 |
| 认知与推演类 | 整理态势、预测路径、压缩复杂信息并辅助决策。 | 算力线程、注意力窗口、稳定冗余。 | 认知过载、路径残留、决策依赖。 |
| 防护类 | 建立护盾、污染隔离、权限保护或神经稳定结构。 | 吞吐通道、稳定冗余、接口带宽。 | 护层破裂、隔离误判、接口倒灌。 |
| 生理与神经类 | 强化反射、运动、疼痛控制、生命维持或器官适配。 | 神经接口、稳定性、结构空间。 | 神经反噬、反射失控、伤情误判。 |
| 通讯与同步类 | 传递身份、坐标、指令、感知摘要或团队状态。 | 接口带宽、稳定冗余、权限结构。 | 延迟、误传、劫持、通讯污染。 |
| 操控与牵引类 | 对物体、载具、外场或自身姿态施加牵引、偏转或稳定。 | 吞吐通道、调控结构、接口带宽。 | 反向拖拽、输出烧蚀、结构撕裂。 |
| 诊断与维护类 | 检查灵脉、灵械、接口、污染残留和失效点。 | 稳定性、算力、精密控制。 | 误诊、校准漂移、维护者感知疲劳。 |

例如“战术推演”是认知与推演类功能，“生命扫描”是感知类和诊断类功能，“护盾发生”是防护类功能，“神经加速”是生理与神经类功能，“污染隔离”是防护类功能，“共鸣通讯”是通讯与同步类功能，“以太牵引”是操控与牵引类功能。这些名称不应直接写成谱系名称，除非已经补足原型来源、主干演化、分支继承和断代关系。

## 四、铭刻、组合与资源模型

灵脉可铭刻在不同位置：

- 灵核内铭：反应快，风险高，适合核心能力。
- 灵核外壳铭刻：较安全，适合接口、稳定和调频类能力。
- 神经并联铭刻：适合感知、反射和运动控制，但对神经适配要求高。
- 器官附着铭刻：适合眼、耳、心脏、肌肉等局部强化。
- 体表或外场铭刻：适合护盾、隐匿和外放能力。
- 外置灵脉：以灵械形式存在，由接口灵脉连接。

灵脉兼容性主要包括：

- 同源兼容：同一设计体系的灵脉更易同步。
- 功能冲突：高灵敏感知和情绪放大并用可能导致过载。
- 频率冲突：运行频率不一致会造成延迟、震荡或失控。
- 认知冲突：反馈信息超出注意力窗口会造成意识混乱。

灵脉内部结构也会随工程经验改进。直连回路可以加入分层滤波、平衡放大环、限流段、断连段、稳相缓冲段和多相导流环等结构，让能力从粗糙可用变成可维护、可复检的方案；具体案例见 `灵脉结构原理与工程演变案例.md`。

灵脉之间还存在谱系关系。谱系用于记录灵脉从原型结构发展为主干、分支、旁支、断代残支或遗产接枝的过程，帮助判断同源兼容、定制来源和维护风险。谱系不等于新的固定能力分类，也不直接决定灵脉是否标准化或定制化；详细规则见 `灵脉谱系设定.md`。

能力架构还会随文明技术和职业需求改变。早期个体更倾向于把完整功能灵脉直接铭刻在体内；现代架构通常让体内保留低延迟核心、接口、限流和稳定结构，把高功率、高风险、复杂处理和维护密集段交给外置灵械或团队系统。架构演变细则见 `灵脉架构演变.md`。

常见组合模式：

- 串联组合：一个灵脉的输出成为另一个灵脉的输入，如红外捕捉 -> 目标识别 -> 弹道预测。
- 并联组合：多个灵脉同时运行，如夜视、红外和声波定位并行。
- 反馈组合：输出反过来调节自身或其他灵脉，提高精度和反应速度。
- 冗余组合：关键能力设置备用路径，避免单点失效。

能力构建受以下资源限制：

| 资源 | 说明 |
| --- | --- |
| 结构空间 | 可铭刻多少灵脉结构，决定能力种类与数量。 |
| 算力线程 | 可同时运行多少灵脉，决定并发与复杂度。 |
| 吞吐通道 | 可同时提供多少以太输出，决定持续功率。 |
| 注意力窗口 | 使用者能处理多少反馈信息，决定认知负荷上限。 |
| 稳定冗余 | 出错时的缓冲余地，决定安全性。 |
| 接口带宽 | 与灵械交换数据和能量的能力，决定外置模块表现。 |

设计能力时必须说明资源占用和替代方案。侦察者可以把多种感知能力外置为灵械，节省体内结构空间；传统近战职业可能把吞吐通道和结构空间投入体内强化，牺牲接口扩展性。

### 资源占用等级

能力设计可以使用五档资源占用口径，避免每次都写成精确数值。等级只描述运行负担，不直接等于能力强弱；同一能力在不同使用者、灵械和环境中可能对应不同等级。

| 等级 | 说明 | 示例场景 |
| --- | --- | --- |
| 低占用 | 日常可用，副作用轻，适合民用、训练和长时间低功率运行。 | 基础感知、身份识别、低功率接口、辅助提示。 |
| 中占用 | 职业使用，需要周期维护和稳定性训练。 | 医疗扫描、工程诊断、职业级夜视、普通灵械同步。 |
| 高占用 | 战斗或高危任务使用，长期运行会损伤灵核、灵脉或神经。 | 战术推演、机甲同步、高强度护盾、污染区作业。 |
| 峰值占用 | 短时间爆发，需要冷却、断连和战后复检。 | 紧急加速、护盾过载、舰队级同步、强制救援输出。 |
| 禁限占用 | 涉及遗产、深层以太、非法超频或未认证改造，必须伴随严重代价。 | 遗产接枝、深层以太直连、黑市功率锁解除、人格回响风险。 |

占用等级必须标明主要压在哪类资源上。例如同为高占用，战术推演主要压算力线程和注意力窗口，护盾发生主要压吞吐通道和稳定冗余，遗产接口主要压接口带宽和污染隔离。

一个完整能力至少要回答三件事：它为什么能运行，为什么不能无限运行，坏了会怎样。

## 五、灵械与接口时代

灵械是离体化的能力模块，本质是外置灵脉、物质载体、供能结构和控制接口的组合。它让能力系统从传统个体修炼转向接口标准、模块化装备、团队协同和工业维护。

灵械与外置灵脉进入工业化阶段后，必须通过标准化和定制化机制管理兼容性、认证、维护、权限和事故责任。基础能力原理仍以本文件为准；体内核心、接口调度、外置模块和协同网络的分工演变见 `灵脉架构演变.md`；接口标准、军用/民用/商用规格、职业套件、定制逻辑和黑市改装风险详见 `灵脉与灵械标准化体系.md`。

常见灵械类型：

- 感知型：红外目镜、声波定位器、生命扫描器。
- 运算型：灵算核心、战术推演器、舰队级辅助计算阵列。
- 强化型：外骨骼、肌肉牵引系统、抗重力适配装置。
- 输出型：以太炮、屏障发生器、工程切割器。
- 防御型：护盾发生器、污染过滤器、反劫持隔离模块。
- 医疗型：生命维持、灵核修补、污染清除、神经稳定装置。
- 工程型：结构分析、材料雕刻、灵脉刻录和调频工具。

灵械优势：

- 节省体内灵核空间。
- 降低高风险铭刻对身体的伤害。
- 便于升级、维护、替换和团队标准化。
- 可承载体内难以容纳的高功率或大型结构。

灵械限制：

- 依赖外部供能和物理载体。
- 存在接口延迟、同步误差和带宽上限。
- 容易受到干扰、污染、劫持或战损影响。
- 维护成本高，使用者需要接口适配和断连训练。

接口灵脉是接口时代的基础能力。它本身不提供显性攻击或感知效果，但决定使用者能否安全接入灵械、舰载系统、团队网络或古文明遗产模块。高阶接口灵脉必须具备识别、同步、转译、权限控制、污染隔离和紧急断连能力。

## 六、个体建立灵脉的过程

个体建立灵脉不是一次性获得能力，而是把能力目标、灵核承载条件、结构来源、试运行结果和维护责任逐步收束成稳定能力架构的过程。不同文明、学校、工坊和家族会采用不同执行环节，但都必须回答同一个核心问题：这条灵脉能否在这个个体身上长期运行、被关闭、被维护，并在失效时可追责。

### 建立前评估

建立灵脉前必须先评估承载对象，而不是直接选择能力效果。

评估内容包括：

- 灵核六维：确认容量、算力、吞吐量、稳定性、亲和度和可塑性中哪些是优势，哪些会成为限制。
- 神经承受力：判断感知反馈、反射强化、疼痛抑制或意识层接口会不会超过神经系统承载范围。
- 既有灵脉架构：检查是否存在同源兼容、频率冲突、接口污染、算力线程争夺或注意力窗口不足。
- 职业目标：明确该灵脉服务于军校训练、医疗、工程、侦察、舰队协同、机甲同步还是深层以太任务。
- 维护条件：确认使用者是否能获得调频、复检、灵械替换、污染隔离和事故处理支持。
- 监管边界：判断该方案是否触及军用管制、环带限制、商盟契约、裂谷人体实验或遗产隔离审查。

评估结论通常不是“能”或“不能”两种，而是形成可承载版本、需外置版本、延后训练版本、降级版本或禁止建立版本。高可塑低稳定者可以适合长期成长，但第一条灵脉必须留出恢复周期和回滚空间；高吞吐低稳定者可以适合爆发输出，但不应直接铭刻高功率持续型灵脉。

### 能力目标拆解

使用者提出的能力愿望必须被拆成可设计结构。灵脉工程师、军校教官或职业导师会把“想看见敌人”“想更快反应”“想保护队友”拆成输入、处理、输出、反馈和断连需求。

拆解时至少明确：

- 输入来源：环境信号、身体状态、灵械数据、队友状态、共鸣网络或深层以太回响。
- 处理路径：需要感应、筛选、放大、转换、推演、权限确认还是污染隔离。
- 输出形式：是意识层提示、神经反射、外场护层、以太牵引、通讯摘要还是灵械控制指令。
- 反馈强度：反馈进入视觉、听觉、肌肉、情绪、痛觉还是纯意识层。
- 关闭条件：使用者主动关闭、达到冷却阈值、权限撤销、污染警报或神经负载超限时如何停机。
- 最小可用形态：在不能完成完整灵脉时，是否能先建立低功率提示、外置辅助或单一功能段。

能力目标越模糊，越容易被写成无成本天赋。可用的目标应能落到结构、资源和代价上。例如“战场直觉”不能只写成预感，而应拆成危险信号捕捉、威胁权重压缩、注意力提示和误判代价。

### 结构来源选择

确定目标后，需要选择灵脉结构来源。结构来源不决定灵脉强弱，也不直接决定它是标准灵脉、定制灵脉或谱系分支。

常见来源包括：

- 自然生成：灵核在长期环境、种族倾向或危机压力下形成粗糙回路，后续需要训练和调频才能稳定。
- 训练诱导：通过重复感知、调息、动作反馈和低功率场域，让相似灵纹或灵环逐步出现。
- 标准铭刻：采用公开教材、职业套件或军校标准，在安全阈值内建立可检测、可维护的主回路。
- 定制改写：基于参考灵脉裁剪、偏移或增补结构，使其贴合个体灵核参数和职业任务。
- 外置协同：把高功率、高风险或复杂处理段放入灵械，体内只保留接口、转译、限流和断连结构。
- 遗产接枝：把古文明蓝图、断代灵段或深层以太回响接入现代架构，必须经过沙箱、单向转译和人工复核。

原创灵脉只说明设计来源基本不依赖既有结构。它可以在验证和公开后进入标准体系，也可能因为只适配少数个体而长期停留在定制体系。遗产型结构也不等于高阶升级路线，它常常先表现为识别困难、接口不透明和维护责任不清。

### 预演与沙箱测试

正式建立前通常会进行低风险预演。预演不是证明能力一定成功，而是暴露不适配点。

常见测试方式包括：

- 灵核负载模拟：用调频台或训练场域估算容量、算力、吞吐和稳定冗余的压力。
- 外置试运行：先让灵械承担主要能力段，使用者只接收低强度反馈。
- 意识层反馈训练：用简化信号测试注意力窗口、反馈偏好和异常识别能力。
- 功能段沙箱：单独测试感应段、筛选段、放大段、转译段、接口段或断连段。
- 污染隔离测试：面对深层以太样本、遗产残片或陌生灵械时，验证隔离和熔断是否可靠。
- 团队兼容测试：检查该灵脉是否会干扰队友通讯、舰队同步、职业套件或共享权限。

预演失败不等于角色没有潜力。失败可能说明能力目标需要降级、反馈方式需要改写、部分结构应外置，或现阶段不适合固化。联邦军校重视预演记录和事故追责；商盟重视授权条款和质保边界；环带更可能在预演阶段直接封停高风险方案；裂谷则可能把预演失败本身视为改造数据。

### 低功率接入

通过预演后，灵脉不会立刻以完整形态运行，而是先低功率接入。这个阶段常被称为试接、初接或临时回路建立。

低功率接入的目标包括：

- 建立最小灵纹或灵环，确认灵核能识别该能力方向。
- 限制输入强度，避免感知洪流、神经反噬或输出烧蚀。
- 设置手动断连、自动熔断和外部监控，确保异常时能停机。
- 记录使用者的反馈偏好，例如更适合视觉提示、肌肉微震、意识层标记还是声音提示。
- 检查与既有灵脉、外置灵械和团队接口的频率关系。

低功率接入最适合表现能力初成时的不稳定：开启慢、反馈粗糙、误差大、需要教官或工程师在旁监控。它也能解释为什么有些能力在训练场可用，进入战场、污染区或深层以太后却会漂移。

### 共适应

共适应是灵脉结构与个体承载系统之间的双向自适应现象，从灵脉结构在个体身上出现的那一刻就开始，不必等到灵脉完全固化。只要最小灵纹、灵环或临时回路已经能被灵核识别，个体就会无意识地调整运行习惯、反馈路径和局部以太流向，使新灵脉更符合自身身体、灵核参数和既有灵脉架构。

共适应不是单向校准。新灵脉会被个体调整，身体、灵核和既有灵脉也会反过来适应新结构。感知类灵脉可能改变使用者对光、声、热或以太波动的注意习惯；神经并联灵脉可能改变动作起手和肌肉反馈；接口灵脉可能让既有灵械同步方式发生轻微偏移。这个过程解释了为什么同一条标准灵脉在不同个体身上会出现细节差异：有人更容易得到清晰反馈，有人更容易降低功耗，有人则会牺牲反应速度换取稳定。

共适应不一定完全正向。如果没有使用者主动训练、教官监督、灵脉工程师调频或灵械监控，共适应可能把灵脉推向更顺手但不安全的方向。常见结果不是严重事故，而是灵脉效果发生轻微变化，有好有坏；严重时才会出现灵脉削弱、结构失效或失控。

常见共适应结果包括：

| 共适应结果 | 表现 | 设定用途 |
| --- | --- | --- |
| 顺向贴合 | 反馈更清楚，开关更自然，功耗或延迟略微下降。 | 表现角色与能力逐渐熟悉，或说明某个灵脉特别适合该个体。 |
| 功能偏移 | 原本标准效果发生轻微变化，例如感知范围变窄但分辨率提高，护层更稳定但展开变慢。 | 区分同类能力的个人风格，制造优劣并存的成长结果。 |
| 旧脉让路 | 既有灵脉降低部分反馈强度，为新灵脉腾出算力线程、注意力窗口或吞吐通道。 | 表现架构取舍，避免角色能力无限叠加。 |
| 旧脉排斥 | 既有灵脉与新结构争夺频率、接口或注意力，新灵脉运行不顺。 | 制造训练瓶颈、兼容事故或职业路线冲突。 |
| 错向固化 | 使用者无意识地保留了错误反馈习惯，使灵脉越练越偏。 | 解释后期需要重构、回滚或专业校准。 |
| 失控共适应 | 新旧结构互相放大误差，导致漂移、反噬、烧蚀或短时失控。 | 用于高风险训练、遗产接枝、黑市改装事故。 |

外力调控的作用不是替代个体共适应，而是设置边界。教官、医生、灵脉工程师和灵械监控系统会通过限流、调频、反馈记录、阶段复测和强制休息，让共适应朝可维护方向发展。联邦军校通常会把共适应记录纳入训练档案；环带更重视共适应是否破坏长期稳定；商盟会把共适应偏移写入质保和适配条款；裂谷则可能主动诱导极端共适应，以观察灵核改造潜力。

### 固化与铭刻

当临时回路经过多次稳定运行后，才进入固化或铭刻阶段。固化是让灵脉主回路在个体身上形成可复现结构；铭刻则是用工程手段把结构固定到灵核、神经、器官、体表或外置灵械中。

常见固化方式包括：

- 自然固化：使用者长期重复某种感知或动作，灵核自行保留稳定结构，成本低但精度有限。
- 训练固化：通过课程、战术动作、反馈训练和周期调频，让灵脉逐步定型。
- 工程铭刻：由灵脉工程师使用刻录工具和调频台固定关键灵环、灵节或灵段。
- 神经并联固化：将反馈接入神经路径，适合反射、运动、感知类能力，但神经副作用更强。
- 器官附着固化：把局部功能接入眼、耳、心脏、肌肉或皮肤，效果明确但修复复杂。
- 外置转译固化：体内只固定接口和转译段，主要能力段保留在灵械中。

固化太早会锁死可塑性，使后续升级只能绕路改造；固化太晚会导致灵脉长期漂移，使用者难以形成可靠习惯。高风险灵脉通常采用分段固化：先固定识别、限流和断连，再固定核心能力段，最后才允许峰值输出或团队协同。

固化应尽量发生在共适应方向被确认之后。未经监控的错向共适应如果过早固化，可能把偏移、排斥或错误反馈习惯写进主回路；反过来，适度共适应后的固化可以保留个体优势，让标准灵脉形成更适合个人的稳定版本。

### 适配训练

灵脉固化后仍然不能视为完成。使用者必须学会把它纳入自身能力架构。

适配训练包括：

- 开启与关闭：在平静、运动、受伤、干扰和疲劳状态下稳定开关。
- 参数调整：调节感应范围、输出强度、反馈密度、冷却间隔和权限模式。
- 异常识别：区分环境噪声、灵脉漂移、接口延迟、污染信号和心理错觉。
- 断连恢复：在灵械战损、权限撤销、污染警报或强制熔断后恢复方向感和身体控制。
- 多模块协同：与其他灵脉、职业套件、团队网络和外置灵械协同运行。
- 压力复测：在战斗、深层以太、舰队同步、污染区或高重力环境中验证稳定性。

适配训练会改变角色习惯。长期使用危险预警者可能更难放松；长期使用推演灵脉者可能把人际交流也拆成路径和权重；长期依赖外置灵械者可能在断连后产生能力空洞感。

### 验收与维护登记

灵脉建立完成后，需要验收与登记。验收不是形式记录，而是决定这条灵脉能否进入职业训练、军用任务、商盟交易、医疗许可或遗产研究。

常见验收内容包括：

- 结构验收：确认主回路、冗余路径、限流、调控和断连结构可被检测。
- 运行验收：测试低功率、职业功率、峰值功率和冷却后的漂移程度。
- 兼容验收：检查与既有灵脉、接口灵脉、灵械模块和团队系统的同步情况。
- 安全验收：确认污染隔离、权限控制、熔断机制和外部救援链路可用。
- 维护登记：记录维护周期、禁用条件、可替换灵械、责任机构和事故追责方式。
- 心理验收：评估反馈方式是否改变决策习惯、情绪稳定或身份认同。

不同文明的验收重点不同。联邦强调任务可靠性、战损冗余和审计记录；环带强调长期生态影响、安全阈值和可逆性；裂谷强调个体承受力、改造潜力和实验价值；商盟强调契约、质保、授权密钥和跨文明兼容。

### 失败、回滚与改道

灵脉建立失败是能力体系的重要组成部分。失败不应只写成“没有天赋”，而应说明是哪一层结构、资源或制度出现问题。

常见失败类型包括：

| 失败类型 | 表现 | 后续处理 |
| --- | --- | --- |
| 承载不足 | 运行时头痛、眩晕、灵核过热或稳定冗余快速耗尽。 | 降低功率、裁剪结构、延后训练或转为外置灵械。 |
| 反馈排异 | 使用者无法理解反馈，出现幻视、误判、情绪波动或动作迟滞。 | 改写反馈方式，增加筛选结构，缩短运行时间。 |
| 频率漂移 | 能力初期有效，长时间运行后延迟、误锁或效果偏移。 | 重新调频，增加稳定环，限制连续运行。 |
| 结构烧蚀 | 灵纹、灵环或灵节受损，能力降级或疼痛残留。 | 停用修复，替换受损段，重新验收。 |
| 接口倒灌 | 灵械、网络或遗产结构沿接口侵入灵核。 | 紧急断连、污染隔离、记忆与人格回响筛查。 |
| 制度阻断 | 能力可行但被军方、环带、商盟或遗产审查机构禁止。 | 封存方案，转入许可流程，改为低风险版本。 |

回滚不等于完全删除灵脉。安全回滚通常会保留识别、限流和断连经验，删除或休眠高风险能力段。改道则是把原目标换成更适配的形态，例如把体内高灵敏感知改为外置目镜，把高功率护盾改为低功率污染隔离，把遗产接枝改为沙箱转译研究。

角色成长可以通过失败体现：一次失败会留下维护债务、心理阴影、制度记录或新的设计线索。接触高风险遗产蓝图的角色不应直接获得完整遗产灵脉，而应先经历识别异常、沙箱转译、低功率接入、人格回响筛查和对应机构审查压力。

## 七、灵脉获取方式与权限限制

灵脉不是只要存在技术资料就能随意获得。一个个体能否获得某条灵脉，取决于结构来源、风险等级、社会身份、职业资质、维护条件和文明监管。获取方式决定了灵脉最初进入个体能力架构的路径，也决定了后续是否能合法维护、升级、交易或转移。

家庭、族群、行会和机构传承会影响个体获取灵脉的起点。它们真正传递的通常不是单条“必定适配”的能力，而是多代筛选出的训练法、灵脉主干、反馈习惯、维护经验、事故记录和担保关系。相关社会分层在 `../生产关系与职业结构.md` 中归入能力资本的代际积累；本节只说明这些资源如何进入个体能力架构。

常见获取方式包括：

| 获取方式 | 说明 | 常见限制 |
| --- | --- | --- |
| 自然生成 | 灵核在环境、危机、种族倾向或长期行为中自发形成粗糙结构。 | 结构不稳定，需要检测、调频和登记；危险能力可能被限制使用。 |
| 家庭或种族传承 | 家族、族群或地方传统长期保留适配自身灵核倾向的训练方法、灵脉主干、维护记录和事故避坑经验。 | 对外兼容性差，可能涉及家族准入、族群传统、职业担保或地方监管；不能保证后代必定适配。 |
| 学校训练 | 军校、职业学校、工会或基础教育机构提供标准灵脉训练。 | 只开放低风险或对应课程范围内的灵脉，需通过阶段考核。 |
| 职业认证 | 医疗、工程、侦察、舰队、机甲等职业通过资质考试开放职业灵脉。 | 必须有执照、维护记录和定期复检，离职或违规时可能降级或停用。 |
| 军方授权 | 军队、舰队、特战或执法系统提供军用、战术或高杀伤灵脉。 | 受权限、军籍、任务审计和退役处理限制，不允许普通平民合法持有。 |
| 商盟购买 | 通过商品规格购买外置灵械、授权模块或低风险灵脉服务。 | 依赖授权密钥、质保条款和维护合约，高风险版本通常只卖给机构。 |
| 医疗植入 | 因伤病、残障、灵核损伤或救援需求植入辅助、稳定或替代灵脉。 | 以治疗和恢复为目的，通常限制功率和用途，需医疗机构追踪。 |
| 实验项目 | 研究机构、裂谷实验室、军工项目或遗产复原计划提供实验灵脉。 | 需要知情授权、隔离审查和事故责任归属，长期副作用不一定清楚。 |
| 遗产接触 | 通过古文明蓝图、深层以太回响或断代灵段获得异常结构。 | 不能直接视为合法获取，必须经过沙箱、封存、隔离和人工复核。 |
| 黑市渠道 | 绕过认证购买非法铭刻、解除功率锁或接入未登记模块。 | 违法且高风险，常伴随后门、污染、维护缺失和追责问题。 |

灵脉获取还受到风险等级限制。民用低风险灵脉可公开训练或购买；职业灵脉需要资质、课程记录或执业机构担保；军用、执法和强杀伤灵脉需要军方、舰队、特战单位或授权执法机构批准；高危任务灵脉需要任务许可、隔离支援和战后复检；遗产、深层以太和人格回响相关灵脉需要遗产隔离审查；黑市灵脉不进入合法维护体系。

一般平民不能合法获得军事化或具有较强杀伤力的灵脉。平民可接触的通常是被限流、外置化、训练锁定或去武装化的民用版本，例如低功率防身护层、工程切割器的安全模式、民用夜视目镜或基础接口灵脉。若平民角色持有军用或高杀伤灵脉，必须说明来源：退役残留、执法授权、任务临时许可、家族封存、走私、黑市改装或遗产事故。

非法获取不只带来法律风险，也会切断维护链路。未登记灵脉无法正常获得调频、修复、替换件和事故救援；商盟模块可能被远程锁定；联邦军用灵脉可能触发权限审计；环带限制模块可能被要求封存；遗产结构可能引发肃正协议警戒。

## 八、灵脉管控与外部限制

灵脉一旦合法构建在个体身上，其本体运行权应归属于铭刻个体。体内灵脉不是可被外部账号接管的设备，也不应被写成任何机构能随时远程操纵的模块。即使是军用、职业或商盟授权灵脉，合法标准也只能规定使用条件、维护资格、外置模块权限、任务审计和违规后果，不能在灵脉本体中留下外人可直接控制的隐藏通道。

因此，所谓灵脉管控通常不是“控制这条灵脉”，而是通过额外手段限制个体使用环境、输出条件、外部接口或维护资格。

常见管控手段包括：

| 管控手段 | 作用方式 | 边界与风险 |
| --- | --- | --- |
| 权限审计 | 记录灵脉使用场景、功率、任务编号或接口对象。 | 只能追踪和追责，不能直接操纵体内灵脉。 |
| 外置灵械锁 | 对武器、机甲、舰载系统、商盟模块或高功率供能装置加锁。 | 限制的是外置能力段，不是个体体内灵脉本身。 |
| 环境限制 | 通过训练场、拘束舱、隔离室、医疗舱或屏蔽场压低灵脉输出环境。 | 适合临时处置，长期使用可能伤害灵核和心理状态。 |
| 抑制灵械 | 以外部装置干扰、限流、降温或隔断接口反馈。 | 需要专业人员监管，过度抑制可能造成反噬、漂移或神经副作用。 |
| 附加监管灵脉 | 额外构建不受个体主动控制的监测、限流、报警或熔断结构。 | 属于高度敏感医疗/监管措施，不能伪装成普通灵脉。 |
| 医疗休眠 | 在污染、过载、神经反噬或人格回响风险下让相关灵脉进入休眠。 | 以救治为目的，需医生和工程师复核，不能作为常规惩罚工具。 |
| 维护资格限制 | 暂停调频、升级、替换件、授权密钥或职业复检。 | 通过制度限制使用上限，可能影响职业和社会身份。 |
| 强制封存 | 在严重违法、遗产污染、军用退役或公共安全事件中封存高危灵段。 | 应伴随法律程序、医疗评估和后续救济渠道。 |

通过完全不受个体控制的附加灵脉或灵械进行限制，在技术上可以存在，但伦理和制度风险极高。它不应被写成普通安全功能，而应被视为医疗急救、监管拘束、战俘处理、污染隔离、实验控制或极端刑事处置的一部分。它控制的是附加限制结构或外部环境，不是把原本属于个体的灵脉本体变成他人的工具。

政府对强制管控手段通常极为慎重。若一个灵脉需要长期依赖外部手段才能防止个体滥用、失控或伤害他人，那么更合理的制度结论应是：这条灵脉不应该构建在该个体身上，或者只能以外置、限流、降级、封存、任务临时授权等方式存在。强制管控可以用于短期危机处置，但不能替代建立前评估、许可审查和职业伦理。

不同文明和组织对此态度差异很大。星海联邦倾向用军籍、任务权限、外置系统锁和战后审计限制高危能力，不愿在士兵体内保留可被上级直接操纵的隐性控制结构；永恒环带更可能在风险评估阶段拒绝构建危险灵脉，必要时采取封存和生态安全隔离；深渊裂谷的激进派可能接受不受个体控制的实验监管灵脉，把它视为进化试验的一部分；商盟通常偏好合同、授权密钥、售后锁和赔付责任；黑市、邪教式组织或非法实验室则可能滥用附加管控，把个体变成可停机、可追踪或可胁迫的资产。

灵脉管控必须区分三件事：第一，灵脉本体的运行权属于铭刻个体；第二，外部系统可以限制供能、接口、环境、维护和法律后果；第三，任何不受个体控制的附加管控结构都应带来伦理争议、医疗风险和社会冲突。

## 九、灵脉抹除、修复与变更

灵脉医疗干预属于灵核医学、神经医学和灵脉工程的交叉领域。医生负责生命体征、灵核损伤、神经修复和心理影响；灵脉工程师负责结构诊断、调频、模块替换和接口校准；授权机构负责许可、封存、事故记录和责任认定。高风险操作不能只由使用者凭意志完成。

### 抹除

灵脉抹除不等于无痕删除。已经在个体身上运行过的灵脉会留下共适应痕迹、神经反馈习惯、维护记录和可能的心理依赖。多数情况下，所谓抹除实际是停用、休眠、降级、剥离、覆盖或封存。

以下实现原理说明的是结构层面的基本机制，不是所有文明、医院或军方机构必须采用的唯一执行流程。

| 抹除方式 | 实现原理 | 说明 | 主要风险 |
| --- | --- | --- | --- |
| 停用 | 切断启动权限、供能路径或接口触发条件，让主回路无法进入有效运行态。 | 关闭灵脉运行权限，保留结构和维护记录。 | 长期不用可能漂移，紧急时恢复慢。 |
| 休眠 | 用稳定环、限流段和最低识别回路压低活动阈值，只允许安全监测信号通过。 | 用稳定结构压低灵脉活动，只保留最低识别和安全回路。 | 受到刺激、污染或错误调频时可能重新激活。 |
| 降级 | 移除、锁定或熔断高功率、高权限或高风险灵段，让低风险主干继续可维护运行。 | 删除或锁定高功率、高权限或高风险灵段，保留低风险功能。 | 功能缩水，共适应习惯需要重新训练。 |
| 剥离 | 分层拆除局部灵纹、灵环或灵段，并同步填补神经反馈空位与稳定残余接口。 | 通过医疗和工程手段移除部分灵纹、灵环或灵段。 | 可能造成灵核裂纹、神经空洞感、旧脉串扰。 |
| 覆盖 | 在旧接口位置建立新结构，让新主回路接管输入输出，并逐步压低旧结构响应。 | 用新结构取代旧结构的位置或接口。 | 新旧残留可能冲突，容易产生错向共适应。 |
| 封存 | 保留结构的可检测状态，加入禁用阈值、监管记录、复审条件和必要的外置锁定。 | 在监管要求下保存结构但禁止运行，常见于军用、遗产或事故灵脉。 | 使用者可能失去职业能力，后续解封需要复审。 |

强制抹除通常只适用于严重违法、污染失控、军用退役、医疗救命或遗产隔离。抹除不是轻松重置；即使抹除成功，个体也可能保留反射习惯、幻感、身份损失、职业断层或维护债务。

### 修复

灵脉修复的目标不是让能力无代价恢复，而是确定损伤层级、阻断恶化、恢复可维护运行，并决定是否允许重新上限使用。

不同修复手段会随损伤层级、医疗条件、授权等级和文明制度变化，但都必须同时处理结构稳定、反馈安全和后续验收。

常见修复方式包括：

| 修复方式 | 实现原理 | 适用情况 | 说明 |
| --- | --- | --- | --- |
| 调频校准 | 重测运行频率、反馈强度、开关阈值和延迟补偿，让回路重新落入可维护区间。 | 轻度漂移、延迟、误读。 | 重新校准运行频率、反馈强度和开关阈值。 |
| 灵纹修补 | 对受损微结构进行局部补线、稳相和重新导通，恢复吸收、放大或稳定功能。 | 局部吸收、放大、稳定结构受损。 | 适合小范围损伤，通常需要短期休养和复测。 |
| 灵环替换 | 拆换失效感应环、滤波环或稳定环，并重新接入主回路和冗余路径。 | 感应环、滤波环、稳定环失效。 | 可替换标准结构，定制或家族灵环修复难度更高。 |
| 灵节重建 | 重建功能节点的输入、处理、输出和安全断连结构，再逐步恢复节点负载。 | 功能节点烧蚀、错位或污染。 | 需要医生和工程师协作，恢复后必须重新适配训练。 |
| 污染清除 | 先隔离异常信号，再分离、稀释、封存或转译残留结构，避免污染沿接口扩散。 | 深层以太残留、接口倒灌或遗产回响侵入。 | 先隔离再清除，必要时封存相关灵段。 |
| 神经修复 | 处理神经并联路径、感觉残留、反射失控和心理后效，降低反馈对身体与认知的误触发。 | 幻痛、反射失控、感觉残留或认知过载。 | 处理神经接口和心理后效，不一定直接修复能力强度。 |
| 共适应再训练 | 让灵核、身体和既有灵脉重新学习修复后的反馈模式，重建开关、调参和异常识别习惯。 | 修复后新旧结构习惯不一致。 | 重新训练身体、灵核和既有灵脉对修复结构的适应。 |

修复结果可分为完全恢复、带限制恢复、降级恢复、外置替代、长期休眠和永久禁用。军用或高危灵脉即使修复成功，也必须通过复检才能回到任务状态。

### 变更

灵脉变更是对既有灵脉进行有目的的重构，不等同于自然成长。它常见于职业转向、伤后复健、装备替换、文明审查、任务适配或角色能力升级。

变更不是把旧能力简单替换成新能力，而是在既有共适应、维护记录和职业许可基础上重新分配结构空间、算力线程、吞吐通道、接口带宽和注意力窗口。

常见变更方式包括：

| 变更方式 | 实现原理 | 说明 | 主要代价 |
| --- | --- | --- | --- |
| 参数调整 | 不改变主结构，只调整范围、分辨率、输出曲线、反馈密度、冷却阈值和安全边界。 | 调整范围、分辨率、输出曲线、反馈密度或冷却阈值。 | 风险较低，但可能改变手感和判断习惯。 |
| 结构裁剪 | 删去无法承载或任务不再需要的灵段，释放容量、算力和稳定性冗余。 | 删除不需要或无法承载的灵段。 | 降低负担，也可能削弱未来升级空间。 |
| 接口重写 | 重映射体内灵脉、灵械、团队网络和神经反馈之间的接口协议、同步频率和断连规则。 | 改写与灵械、团队网络或神经反馈的连接方式。 | 需要重新同步，存在延迟和倒灌风险。 |
| 外置转译 | 把体内高风险能力段迁移到灵械，体内保留识别、限流、控制和紧急断连接口。 | 把体内高风险结构迁移到灵械，体内保留接口和限流。 | 安全性提高，但依赖装备、供能和维护。 |
| 谱系迁移 | 保留可兼容主干，替换分支逻辑、反馈模式和训练路径，让能力转向另一谱系分支。 | 从一个谱系分支转向另一个分支或旁支。 | 兼容风险高，可能遗留旧谱系习惯。 |
| 职业改装 | 按新职业任务重排感知、推演、输出、接口、冗余和维护结构。 | 按新职业要求重排能力架构。 | 需要资质、课程和长期适配，旧职业能力可能退化。 |
| 高危重构 | 通过沙箱、隔离、低功率接入和分段固化接入遗产、深层以太、军用或实验结构。 | 接入遗产、深层以太、军用或实验结构。 | 可能触发污染、人格回响、监管审查或肃正协议警戒。 |

变更必须说明目标、旧结构如何处理、新结构如何接入、共适应如何重新发生、谁负责医疗与工程风险。对角色而言，变更既是能力升级，也是身份、职业和心理状态的再调整。

## 十、成长路径与职业适配

个体能力形成通常经历以下阶段：

- 灵核检测与基础训练：评估六维参数，训练以太感知、基础调息、注意力分配和稳定性。
- 第一轮铭刻：选择安全基础灵脉，如通用接口、基础感知、防护或稳定类灵脉。
- 适配训练：学习开启、关闭、调参、处理反馈和识别异常。
- 架构优化：根据实际使用情况增加调频、滤波、冗余或断连结构。
- 灵械接入：训练同步率、延迟补偿、污染隔离和战损断连。
- 职业定型与进阶：围绕职业需求升级灵脉、替换灵械、调整资源占用。

个体训练和升级常分为六个阶段：

| 阶段 | 说明 | 常见风险 |
| --- | --- | --- |
| 感知唤醒 | 使用者能感到以太、灵械反馈或自身灵核波动，但信息混乱。 | 误判环境、感知残留、注意力分散。 |
| 基础调频 | 学会稳定开启、关闭和低功率运行，建立基础安全阈值。 | 频率漂移、开关迟滞、低强度头痛或眩晕。 |
| 结构固化 | 灵脉形成可靠主回路，能被训练、检测和维护。 | 固化过早会限制后续可塑性，固化不足会导致运行不稳。 |
| 多模块协同 | 开始组合感知、推演、输出、接口或外置灵械。 | 算力线程争夺、反馈过量、接口延迟。 |
| 压力适配 | 在战斗、污染区、深层以太、舰队同步等高压场景保持稳定。 | 神经反噬、污染倒灌、战后断连困难。 |
| 架构重构 | 重写灵脉结构或接入高风险分支，获得新能力形态。 | 人格回响、谱系污染、肃正协议警戒、长期维护依赖。 |

职业不是由单条灵脉决定，而是由能力架构决定：

- 舰队指挥官重视算力、注意力窗口、空间定位、通讯同步和舰队级接口。
- 机甲机师重视吞吐、反射、设备同步、运动预测和疼痛抑制。
- 侦察者重视亲和度、感知灵脉、隐匿、外置感知灵械和接口带宽。
- 灵脉工程师重视稳定性、算力、精密控制、模拟预演和调频维护。
- 医生重视稳定性、亲和度、生命扫描、污染清除和神经修复。
- 研究员重视算力、记忆索引、逻辑强化和灵算核心接口。

## 十一、损伤、代价与伦理

能力系统必须保留限制和代价。每次升级都应带来新的资源占用、维护成本、心理影响或伦理冲突。

常见风险：

- 灵核损伤：裂纹、污染、过热、容量下降、吞吐衰减。
- 灵脉损伤：错位、堵塞、烧蚀、串扰、调频失效。
- 神经副作用：幻视、幻听、反射失控、认知过载、感知残留。
- 灵械副作用：接口磨损、共鸣依赖、断连眩晕、能力空洞感。
- 心理影响：危险感知可能导致焦虑，思维加速可能造成交流脱节，情绪抑制可能削弱共情。

能力事故可以按表现和来源进一步区分：

| 事故类型 | 表现 | 常见来源 | 制度后果 |
| --- | --- | --- | --- |
| 轻度漂移 | 能力精度下降，反馈延迟，参数需要重新校准。 | 长时间运行、维护不足、环境噪声。 | 需要维护复检，可能影响考试、任务许可和保险记录。 |
| 频率串扰 | 两个或多个灵脉互相干扰，出现误判、误锁或反向反馈。 | 组合设计不良、同源分支偏移、接口标准不一致。 | 可能引发团队事故、装备兼容纠纷和战术责任争议。 |
| 过载烧蚀 | 灵纹、灵环或灵节受损，能力短期降级或停用。 | 峰值占用过久、冷却不足、非法超频。 | 需要医疗修复、能力降级和战后事故审计。 |
| 神经反噬 | 幻视、幻听、疼痛残留、反射失控或认知过载。 | 神经并联铭刻过深、反馈信息超出注意力窗口。 | 可能导致心理复健、训练暂停和职业适配重审。 |
| 接口倒灌 | 灵械、共鸣网络或遗产模块中的异常结构沿接口侵入灵核。 | 隔离不足、权限被劫持、污染熔断失败。 | 可能触发遗产隔离、网络追责和肃正协议预警。 |
| 谱系污染 | 某个改型反向影响同源灵脉、职业套件或团队系统。 | 未登记旁支、遗产接枝、实验分支扩散。 | 可能导致事故调查、封存争议和文明监管冲突。 |
| 人格回响 | 使用者出现不属于自身训练经验的认知习惯、记忆残影或决策冲动。 | 深层以太接触、先驱者蓝图、人格化辅助结构。 | 可能触发伦理审查、主体连续性评估和能力封存。 |

文明伦理差异：

- 星海联邦：强调标准化、军用审核、任务适配和可维护性。
- 永恒环带：强调安全阈值、生态影响和危险技术限制。
- 深渊裂谷：接受高风险灵核改造，追求进化和深层以太接触。
- 商盟：推动灵械商品化、接口标准化和能力模块市场。

古文明遗产不能作为无成本外挂。遗产级灵脉蓝图可以带来性能跃迁，但必须伴随识别困难、接口不兼容、污染风险、肃正协议警戒或人格回响等代价。
