# 天灾与灾害体系

状态：采用
基准：`10_世界观正文/00_总纲/以太纪元世界观总纲.md`、`深层以太设定.md`、`以太能与能源体系.md`、`星际航行和开拓秩序/航行自由跃迁与开拓秩序.md`、`信息治理与社会制度.md`、`文明圈对照表.md`

第二星海时代的天灾与星海灾害，通常被视为一组长期存在的复合灾害类型。所谓天灾不只是物质层自然灾害，而是物质层、能量层、信息层、以太层和意识共振场共同作用后的复合灾害。现代文明可以监测、分级、隔离、救援和追责这些灾害，但不能完全消除它们。

当代灾害分类以深层以太、能源、航行、共鸣网络、遗产隔离、肃正协议和共鸣虫群等既有系统为基础；其中的分级、类型和处置差异用于统一描述星海社会对复合灾害的识别、处置和责任分配，不改变已经确立的以太、航行、能源和文明制度基础规则。

## 一、基本认定

| 认定 | 说明 |
| --- | --- |
| 天灾是复合系统失稳 | 许多灾害不表现为单一爆炸或风暴，而是航道、空间锚、能源相位、共鸣网络、灵核状态和治理责任同时失稳。 |
| 可预警不等于可避免 | 核心区可以建立预警、保险、撤离和救援流程，过渡区与暗区则常因维护链不足、航图滞后或能源不足而放大损失。 |
| 灾害影响制度秩序 | 一场灾害通常同时改变航权、能源额度、保险费率、殖民地承认、样本封存、医疗复检和跨文明仲裁。 |
| 灾害受工程边界约束 | 灾害通常由既有工程边界自然累积而成；自由跃迁事故仍受锚定解、跃迁窗口、航图和接纳条件限制，以太能中断也受能源品级、相位质量、许可和安全状态约束。 |
| 文明态度不同 | 联邦倾向救援与审计，环带倾向阈值与封存，裂谷倾向记录化适应和实验，商盟倾向风险定价、合约与保险。 |

## 二、灾害分类总表

| 灾害类型 | 主要层级 | 典型触发 | 主要后果 |
| --- | --- | --- | --- |
| 深潮反涌 | 深层以太、航行、灵核 | 深层回流擦过稳定区或过渡航道 | 航道漂移、锚定误差、共鸣噪声、灵核过载。 |
| 锚链崩塌 | 航行、空间锚、边境补给 | 空间锚长期失准、维护不足、回流叠加 | 殖民地孤岛化、补给中断、撤离序列争议。 |
| 错窗星门 | 星门、遗产、通行网络 | 星门接入错误旧网络或失准中继 | 船队误入高危航段、污染回流、枢纽封锁。 |
| 相干风暴 | 高相干能源、星港、护盾 | 恒星潮汐、星门枢纽或炉网相位暴涨 | 跃迁停摆、护盾误展开、星港调度降级。 |
| 炉网熔断季 | 能源、城市、殖民地 | 周期性以太潮汐与维护压力叠加 | 高功率用途停摆，生命维持优先，产业违约。 |
| 共鸣雪崩 | 信息层、共鸣网络、人格稳定 | 污染过滤失效，感官碎片互相放大 | 错误警报、人格回响扩散、网络断连。 |
| 灵核热潮 | 灵核医学、民生、教育 | 以太背景噪声周期性升高 | 接口过敏、旧灵脉刺痛、儿童灵核监测异常。 |
| 暗区边界漂移 | 星海地理、保险、殖民秩序 | 深层以太区边界长期推进或退缩 | 迁徙资格、保险费率、开拓许可和自治承认变化。 |
| 遗产炉芯醒潮 | 遗产能源、样本封存、肃正预警 | 古文明炉芯在潮汐中自发高输出 | 能源诱惑、权限标记、污染残留和封存冲突。 |
| 肃正余震 | 古文明监管、禁航、自动防御 | 肃正协议残留误判危险路径 | 星域封锁、星门拒握、自动防御和记录封存。 |
| 虫群迁徙潮 | 共鸣虫群、灵核生态、航道 | 深层潮汐、遗产信号、分巢压力、成熟灵核生态或文明工程扰动改变虫群稳定阈值 | 航道关闭、生态巢化、灵核失稳、共鸣干扰和军事救援。 |

## 三、主要灾害类型

### 深潮反涌

深潮反涌是深层以太回流在短期内擦过稳定区、过渡区航道或暗区边缘的现象。它不是日常天气，而是高压以太环境在局部边界上形成的回流脉冲，持续时间可从数小时到数周不等。

典型表现包括以太底噪升高、空间锚小幅漂移、共鸣中继延迟、灵械接口反馈变粗糙，以及高亲和度个体出现轻度灵核过载。若未及时降级航路，补给船队可能在自由跃迁出口出现偏移，或把污染残留误判为普通潮汐纹。

治理方式通常是提前关闭高风险跃迁窗口，降低星港高相干能源输出，启用低带宽共鸣警报，并要求边境锚站进入复检。联邦会优先组织测绘和救援，环带常要求扩大封锁观察，裂谷可能申请受控接触样本，商盟则会重新计算航线保险和延误赔付。

常见治理争议：航图是否及时公开、锚站维护者是否隐瞒失准、灾害窗口中谁有权优先通行。

### 锚链崩塌

锚链崩塌指一串空间锚、临时锚标或古代锚链残段发生连锁脱锁。单个锚站的误差本来可以维护，但当多个节点同时进入微小失准，整条航路会从可重复通行变成不可承保航段。

物质层的恒星和殖民地仍在原处，但航行意义上会变成孤岛。补给船无法稳定抵达，撤离船没有可靠返航窗口，医疗转运和高相干能源补给被迫中止。边境世界在这种灾害中最脆弱，因为它们可能有基础生活能源，却缺少支撑连续跃迁和护盾的高相干储备。

治理方式包括临时投放应急锚标、调派空间锚工程船、组织分阶段撤离、冻结非必要商业窗口，以及封存旧航图。事故后通常要追查维护记录、锚站主权、航权合约和救援拒付责任。

常见治理争议：边境救援承诺是否真实有效，企业或地方政权是否有权关闭公共返航功能。

### 错窗星门

错窗星门是星门在结构上可运行，却在网络握手、窗口同步或风险复核中接入错误通道的灾害。它常见于古代私有星门网络残片、低等级星门越级中继、旧权限层未识别或双端复检不足的场景。

错窗不会让星门凭空通向任意地点。常见失败形态包括：船队进入了同一网络中的失准中继、封存航段、深层扰动区或未登记旧枢纽。若通道带回污染回流，星门枢纽可能必须封锁泊位、隔离通行船体和暂停整片星门排程。

治理方式通常包括双端复检、网络身份隔离、星门窗口降级、遗产权限审查、事故黑匣封存和跨文明仲裁。星门枢纽的政治压力很大，因为停门意味着贸易和撤离受阻，开门则可能把污染带入核心区。

常见治理争议：星门修复派、封存派、军管机构和边境居民之间围绕“何时算安全开放”的争执。

### 相干风暴

相干风暴是高相干以太能在恒星潮汐、星门枢纽、空间锚群或大型炉网附近出现相位异常暴涨的灾害。它不是总能量不足，而是能流相位质量和同步条件失控。

灾区的基础生活照明、低功率灵械和常规工业可能仍能维持，但跃迁、护盾、星港调度、轨道船坞、高端医疗和军用系统会被迫降级。护盾可能出现误展开、局部过热、反向阻断或无法及时收束；星港会强制停止自由跃迁出口和高优先级货运。

治理方式是将高相干输出降级为常规整流能，关闭部分星港补给窗口，启用冷却复检和相位材料更换。核心区的优势在于有冗余维护链；边境港则可能在账面拥有能源额度却无法兑现安全状态。

常见治理争议：医疗转运、军用调拨、商业急件和居民基础服务之间的能源优先级争夺。

### 炉网熔断季

炉网熔断季是某些殖民地、轨道城市或边境工业区在周期性以太潮汐中进入能源脆弱期的现象。它常由当地以太炉维护能力不足、高相干储备漂移、相位材料老化和潮汐高峰叠加造成。

它不是彻底停电，而是公共炉网主动熔断高功率用途，保留医疗、生命维持、基础重力、基础共鸣警报和低功率民生服务。工业订单、星舰补给、护盾演练和大型灵械作业被推迟，能源额度中的安全状态失效。

治理方式包括提前公布熔断日历，建立生命维持优先序列，限制商业高功率合约，外调相位材料维护师，并在灾后进行能源额度赔付和责任复核。

常见治理争议：基础服务权与工业生产、合约履行、军用征调之间的冲突。

### 共鸣雪崩

共鸣雪崩是共鸣网络中的污染过滤、感官记录分级或人格回响隔离失效后，大量低强度异常片段互相放大的信息灾害。它不会把所有人变成同一个意志，但会让大量主体的判断系统受到错误数据冲击。

受影响者可能出现错误警报、幻听、任务记忆混杂、人格回响复述、AI 载体稳定周期缩短和自动执法阈值误升。最危险的情况是灾害广播本身被污染，使避难指令和异常信号难以区分。

治理方式是断开高带宽中继，切换到低带宽冷通道，封存高风险感官记录，启用人工复核和心理复检。司法系统需要判断哪些记录可作为证据，哪些记录会污染观察者。

常见治理争议：公共知情权、司法证据、信息封存、平台责任和危机后扩权残留。

### 灵核热潮

灵核热潮是某片星区以太背景噪声周期性升高后，智慧生命灵核出现群体性压力反应的民生灾害。它常见于深层边缘殖民地、旧锚链附近、恒星以太潮汐强烈的栖居群，或共鸣网络清洁不足的轨道城市。

常见症状包括接口过敏、旧灵脉刺痛、低功率民用灵械误同步、儿童灵核发育监测异常、睡眠紊乱和情绪疲劳。高亲和度个体更早感知异常，高稳定性个体恢复较快，但不代表完全免疫。

治理方式包括降低民用灵械运行功率，暂停高占用训练，给儿童和高风险职业者增加灵核医学筛查，限制沉浸式共鸣娱乐，调整学校、工厂和星港轮班。

常见治理争议：灵核医疗资源分配、职业停工赔付、儿童迁居资格和地方政府是否承认灾害等级。

### 暗区边界漂移

暗区边界漂移是深层以太区边界在数十年、数百年尺度上缓慢推进、退缩或改变内部强度的长期灾害。它更像星海尺度的气候变化，而不是一次性事故。

某个殖民地可能在建立时位于过渡区边缘，数十年后逐渐变成暗区前沿。航图复测周期缩短，保险费率上升，高相干补给窗口减少，生态改造项目暂停，居民迁徙资格、自治承认和开拓权链都要重新评估。

治理方式包括长期观测站、局部稳定工程、空间锚阵列、隔离穹顶、迁徙补贴和殖民地承认复核。它容易制造慢性制度压力，因为没有单一灾害日，也很难认定谁应承担全部责任。

常见治理争议：居民是否有权迁出、企业是否继续拥有开拓权、环带是否有权提前封存、联邦是否应补贴长期救援。

### 遗产炉芯醒潮

遗产炉芯醒潮是古文明炉芯、遗产级能源接口或古代锚链节点在特定潮汐中自发进入高输出状态。它在短期内可能提供罕见的深层/遗产级能源，因此极具诱惑。

危险在于能流可能夹带权限标记、污染残留、人格回响或肃正协议相关信号。地方政府和企业可能急于使用它救灾、修复星门或支撑护盾，封存机构则会要求先完成隔离和样本链复检。

治理方式包括单向整流、污染熔断、权限层审查、AI 与数字人格载体隔离、封存许可和跨文明仲裁。若炉芯输出与公共救灾绑定，继续使用还是立即封存会成为高度政治化的问题。

常见治理争议：救灾必要性与遗产风险、能源短缺与封存伦理、样本所有权与公共安全。

### 肃正余震

肃正余震是肃正协议残留系统在现代星海中被危险技术、深层污染、遗产能源或错误信号触发后的局部反应。它不一定代表完整肃正协议苏醒，更常表现为自动监管残段的误判、迟滞响应或局部封锁。

常见表现包括星门拒绝握手、禁航区扩张、自动防御平台启动、共鸣信标被压制、样本库被强制隔离，以及部分古代设施拒绝现代权限。它像灾害，也像旧时代的执法残影。

治理方式是立刻停止高危接入，建立肃正预警等级，隔离相关样本、蓝图、能源接口和通行记录，并由多文明专家判断是否存在扩大响应风险。环带通常要求长期封存，裂谷可能试图理解规则边界，联邦会优先保障撤离通道，商盟则关注航运损失和合约中止。

常见治理争议：现代文明能否利用旧监管规则、谁有权解释肃正协议、封锁是否侵犯居民和航运权利。

### 虫群迁徙潮

虫群迁徙潮是共鸣虫群受深层潮汐、恒星活动、遗产信号、生态压力、分巢压力或文明工程扰动牵引，跨星区改变活动范围的灾害。它不是普通兽潮，而是生物生命、群体意识、以太网络和局部生态系统共同迁移。

共鸣虫群并非单纯怪物族群或军事敌人。它更接近由母巢、多层虫体、孢体和共生微生态构成的分布式生命系统。单个虫体多数没有完整个体人格，母巢也不是文明级 AI；真正维持虫群连续性的，是母巢相位、群体记忆、化学生态、以太共鸣和迁徙路径共同组成的“巢群”。不同巢群可以有不同迁徙习惯、攻击阈值和对文明信号的敏感度，因此虫群既不是完全无脑，也不能轻易外交化。

共鸣虫群与一般智慧生命文明的根本矛盾，来自两种互斥的以太生命路线。一般智慧文明依赖个体灵核维持人格连续、能力训练、灵脉运行、责任主体和社会分工；虫群则把类似灵核的稳定功能外化为巢群相位场，依赖低差异同步、群体记忆和巢化环境维持整体连续。智慧文明会把以太环境个体化、接口化、制度化；虫群会把以太环境巢群化、同步化、繁殖化。两者不是单纯争夺土地，而是在争夺同一片以太生态能否继续支持“独立个体”。

虫群迁徙的直接原因通常不是“想进攻文明”，而是维持巢群稳定。常见动因包括：原筑巢区的深层底噪升高，幼巢孵化需要更低干扰窗口，局部生物质和微量矿物被耗尽，母巢分裂产生新巢群，恒星活动改变导航基准，遗产设施发出类似繁殖地或危险源的信号，或文明工程改变了空间锚、共鸣中继、生态穹顶和高相干能源场的相位环境。对虫群来说，迁徙是一种生存、繁殖、避灾和网络重校准行为；对文明来说，它表现为一整片航道和生态带被移动中的生命系统重写。

成熟智慧文明反而最容易吸引虫群。人口密集区、灵核医疗系统、灵核教育与训练机构、共鸣中继、灵械工厂、星港、护盾阵列、高相干能源设施和大型生态穹顶，会长期调制局部以太环境，使其形成稳定、复杂、富含灵核活动痕迹的“成熟灵核生态”。这种生态对智慧文明意味着繁荣和基础设施，对幼巢和分巢期虫群却像高价值筑巢基质、导航标记或孵化温床。因此虫群不是随机撞上文明，而是会被文明最发达、最有秩序、灵核活动最密集的区域吸引。

虫群占据或长期经过的区域会出现巢化以太。巢化以太不是简单污染，而是虫群为了维持巢场同步、幼巢孵化和生态循环，对局部以太底噪、微生物群、矿物壳层、共鸣反馈和生物质流向进行长期重写。被巢化的星球或栖居地未必立即毁灭，但会逐步变得不适合一般智慧生命：儿童灵核发育异常率上升，成年人灵脉错频和接口过敏增加，共鸣网络底噪持续升高，灵械维护成本暴涨，AI 人工灵核和数字人格载体稳定周期缩短。换言之，虫群扩张后的区域很难再作为普通文明社会运行。

虫群攻击逻辑应按“影响阈值”理解，而不是按敌我阵营理解。只要某个对象阻断迁徙路径、污染巢群共鸣、抢占筑巢基质、捕获幼巢或持续发出高强度干扰，它就可能被虫群标记为障碍、猎物、噪声源、寄生源或竞争节点。被攻击对象可以是战舰、殖民地、星门锚站、生态穹顶、矿业设施、无人探针、共鸣中继、遗产样本库，也可以是智慧生命群体；智慧生命不是因为“有智慧”而被优先仇恨，而是因为灵核、灵械、护盾、共鸣网络和高相干能源设施会在虫群感知中形成强烈、复杂且持续的以太扰动。

虫群行动通常具有分层反应。低强度接触时，它们可能绕行、贴附、采样或释放孢体；中强度干扰时，会切断中继、啃蚀设施、压制护盾边界或驱赶舰队；高强度威胁时，才会出现战虫集群、母巢护卫、对星港和殖民穹顶的持续围攻。文明误把侦察、采样或驱赶行为当作全面进攻，或误把暂时绕行当作安全，都会扩大灾害。

灾害影响包括航道关闭、共鸣中继被干扰、殖民生态被巢化、舰船感知系统出现群体噪声，以及部分灵核高亲和者被迫承受虫群共鸣压迫。迁徙路径上的行星和栖居地可能出现外来孢体、共生菌膜、矿物壳层、生物质快速抽取、生态链替换、灵核医学异常和长期检疫问题。虫群不一定直接攻击所有目标，但它经过的区域会进入高等级检疫和军事观察；一旦出现稳定筑巢，文明通常必须按长期封锁、撤离或军事清除处理，而不是期待自然共存。

治理方式包括舰队护航、生态隔离、共鸣屏蔽、航道改线、虫群行为记录、样本封存、巢化以太监测和居民灵核复检。联邦常以救援和封锁并重，环带倾向建立长期隔离带，裂谷关注适应与研究，商盟则会发展高价护航和风险保险。但四文明的差异不改变底层结论：一般智慧文明与活跃巢群很难在同一稳定生态位中长期共存。

更稳妥的处置不是单纯歼灭，而是先判断迁徙动因、巢群状态和影响阈值：避灾迁徙优先改线和撤离，分巢迁徙优先建立隔离走廊，筑巢状态优先封存和长期监测，战斗护巢状态才进入军事压制。若文明试图以共鸣诱饵、幼巢样本或遗产信号操控迁徙，可能短期改变路径，却会提高后续攻击阈值、诱发巢群记忆标记，甚至把某个星港或文明设施变成长期风险源。

虫群生命层级、星系内移动方式、作战目标优先级和反虫群处置边界，详见 `10_世界观正文/10_专题设定/共鸣虫群与作战生态.md`。

常见治理争议：虫群是否可被视为生态主体、军事清除边界、样本研究权、诱导迁徙是否合法、成熟殖民地为何必须撤离、巢化区域是否还有恢复价值，以及智慧生命在虫群路径中应被优先撤离还是允许承担观察和研究风险。

## 四、灾害分级

| 等级 | 说明 | 常见处置 |
| --- | --- | --- |
| 观察级 | 读数异常但未影响基础服务或公共航行。 | 加密监测、更新航图注记、限制高风险作业。 |
| 限行级 | 航道、炉网、共鸣网络或灵核医疗出现可见压力。 | 降级能源输出、暂停高功率任务、要求人员复检。 |
| 封锁级 | 事故可能外溢至相邻星港、殖民地或共鸣网络。 | 禁航、封存、撤离序列启动、跨机构仲裁接入。 |
| 撤离级 | 基础服务、返航窗口或主体稳定已经无法保证。 | 平民撤离、医疗优先、样本隔离、军用救援接管。 |
| 文明级审查 | 涉及肃正协议、遗产级能源、大规模虫群迁徙或深层污染扩散。 | 多文明审查、长期封存、航图重写和公开责任复盘。 |

## 五、四文明处置差异

| 文明圈 | 灾害处置倾向 | 常见争议 |
| --- | --- | --- |
| 星海联邦 | 强调公共航图、救援优先、事故审计、居民撤离序列和遗产共享后的隔离。 | 前线认为程序慢，后方担心军用接管和污染外溢。 |
| 永恒环带 | 强调长期阈值、生态安全、封存令、迁徙资格和人格连续性观察。 | 其他文明质疑其以安全名义扩大封锁，居民可能质疑知情权不足。 |
| 深渊裂谷 | 强调记录失败、适应性试验、改造个体承受力和深层样本价值。 | 真实同意、实验劳动、污染扩散和失败者安置长期冲突。 |
| 商盟 | 强调风险定价、能源额度、保险合约、承包救援和航权赔付。 | 合同效率可能转化为弃救、维护断供、责任拆分和高危灾害包装销售。 |

## 六、灾害认定边界

- 天灾不属于纯神秘惩罚；其成因应能落回以太场、能源、航行、共鸣网络、灵核或遗产系统的可研究失稳。
- 现代文明并非完全无能；成熟社会具备预警、降级、复检、撤离和事故审计能力，只是在暗区、边境或遗产灾害中仍会失败。
- 灾害的首要影响不是个体能力收益，而是公共基础设施、居民生活、航权秩序、保险和文明关系的变化。
- 灾害解释不宜耗尽未知边界。大寂灭、深层以太、肃正协议和虫群活动仍保留未完全解释的边界感。
- 单次灾害通常不会造成全星海同步灭绝；更常见的尺度是星港、锚链、殖民地、航道、星区或文明边境带。
