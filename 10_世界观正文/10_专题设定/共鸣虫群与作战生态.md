本专题用于说明共鸣虫群为什么迁徙、为什么与一般智慧文明极难共存，以及虫群在星系内如何移动、感知、攻击和改变生态。`天灾与灾害体系.md` 保留虫群迁徙潮的灾害分类、分级和公共治理口径；涉及虫群层级、指挥层级、物理平台层级、作战目标和反虫群行动时，以本专题为准。

## 一、基本定位

共鸣虫群不是单纯怪物族群、普通军事敌人、统一可外交种族或文明级 AI。它更接近由母巢、迁徙核心、战虫、工虫、侦察虫、共鸣孢群和共生微生态构成的分布式生命系统。单个虫体多数没有完整个体人格，母巢也不是单一指挥官；真正维持虫群连续性的，是母巢相位、群体记忆、化学生态、以太共鸣和迁徙路径共同组成的“巢群”。

不同巢群可以有不同迁徙习惯、攻击阈值和对文明信号的敏感度，因此虫群既不是完全无脑，也不能轻易外交化。文明可以记录、诱导、隔离或压制部分巢群行为，但很难把活跃巢群纳入一般外交、贸易或领土制度。

## 二、根本矛盾与迁徙动因

共鸣虫群与一般智慧生命文明的根本矛盾，来自两种互斥的以太生命路线。一般智慧文明依赖个体灵核维持人格连续、能力训练、灵脉运行、责任主体和社会分工；虫群则把类似灵核的稳定功能外化为巢群相位场，依赖低差异同步、群体记忆和巢化环境维持整体连续。智慧文明会把以太环境个体化、接口化、制度化；虫群会把以太环境巢群化、同步化、繁殖化。两者不是单纯争夺土地，而是在争夺同一片以太生态能否继续支持独立个体。

虫群迁徙的直接原因通常不是“想进攻文明”，而是维持巢群稳定。常见动因包括：原筑巢区的深层底噪升高，幼巢孵化需要更低干扰窗口，局部生物质和微量矿物被耗尽，母巢分裂产生新巢群，恒星活动改变导航基准，遗产设施发出类似繁殖地或危险源的信号，或文明工程改变了空间锚、共鸣中继、生态穹顶和高相干能源场的相位环境。

成熟智慧文明反而最容易吸引虫群。人口密集区、灵核医疗系统、灵核教育与训练机构、共鸣中继、灵械工厂、星港、护盾阵列、高相干能源设施和大型生态穹顶，会长期调制局部以太环境，使其形成稳定、复杂、富含灵核活动痕迹的成熟灵核生态。这种生态对智慧文明意味着繁荣和基础设施，对幼巢和分巢期虫群却像高价值筑巢基质、导航标记或孵化温床。

虫群占据或长期经过的区域会出现巢化以太。巢化以太不是简单污染，而是虫群为了维持巢场同步、幼巢孵化和生态循环，对局部以太底噪、微生物群、矿物壳层、共鸣反馈和生物质流向进行长期重写。被巢化的星球或栖居地未必立即毁灭，但会逐步变得不适合一般智慧生命：儿童灵核发育异常率上升，成年人灵脉错频和接口过敏增加，共鸣网络底噪持续升高，灵械维护成本暴涨，AI 人工灵核和数字人格载体稳定周期缩短。

## 三、虫群层级

虫群作战生态不是单一兵种堆叠，而是围绕巢群相位、迁徙路径、筑巢基质和巢化以太展开的多层系统。本节先说明生命与职能层级；指挥层级和物理平台层级见后文，三者不能简单一一对应。

| 层级 | 定位 | 主要作用 |
| --- | --- | --- |
| 母巢 | 巢群连续性的根节点，可以是深层巢穴、行星级巢区、巨型生物结构或附着在古代设施残骸上的巢核。 | 维持母巢相位、保存群体记忆、调节繁殖节律和长期迁徙方向。母巢不是单一指挥官，也不是文明级 AI；摧毁母巢会造成巢群断裂、狂乱迁徙或多处分巢，而不一定立刻让所有虫体失效。 |
| 迁徙核心 | 离开母巢后负责维持星系内迁徙队列的移动相位节点，通常由幼巢、导航器官、储能生物质、工虫群和战虫护卫共同组成。 | 在迁徙中保持局部巢场同步，选择落点，携带幼巢和共鸣孢群，是反虫群作战中最关键也最危险的目标。迁徙核心受损会使虫群分散成多个难预测的小巢群。 |
| 战虫 | 负责护巢、破障、近距突入和压制火力的战斗个体或战斗群。 | 攻击舰体、护盾边界、空间锚外壳、星港结构和生态穹顶；部分战虫能以群体方式形成生物装甲、撞击阵列或短时共鸣压制场。 |
| 工虫 | 负责拆解、搬运、筑巢、修复、采矿和生态改写的工程个体。 | 啃蚀残骸和岩体，搬运矿物与生物质，铺设矿物壳层、共生菌膜和孢体床，是巢化以太从临时污染变成长期生态改写的主要执行层。 |
| 侦察虫 | 小型、低信号、可附着残骸或漂浮尘埃带的前出感知个体。 | 测量以太底噪、寻找成熟灵核生态、标记高相干能源源、试探共鸣中继和空间锚反应。侦察虫接触不等于全面进攻，但往往意味着迁徙核心已经在附近获得路径信息。 |
| 共鸣孢群 | 由微小孢体、休眠孢囊、共生微生态和低功率以太薄膜构成的扩散层。 | 负责早期标记、环境采样、感知扩散、共鸣干扰和巢化预铺。它们能污染传感器和共鸣过滤器，也能让一片区域在肉眼尚未被虫群占据时，先进入灵核和共鸣网络不稳定状态。 |

## 四、指挥层级与平台层级的分离

虫群作战体系必须同时看两条正交轴线：一条是指挥层级，回答“哪个相位结构正在改变群体反应阈值、迁徙方向和作战节律”；另一条是物理平台层级，回答“这些指挥与作战功能由什么实体承载，是固定巢区、迁徙核心体、群簇阵列、孢云，还是被附着改写的残骸”。同一个指挥层级可以寄宿在多个平台上，同一个平台内部也可以同时包含多个指挥层级。

因此，虫群不能按单一的“个体大小”或“兵种名称”判断指挥价值。迁徙核心既是高价值指挥节点，也是移动巢港和幼巢载体；大型破障战体可能只是前线武器平台，并不承担长期决策；一片看似松散的孢云可能没有指挥权，却能决定整个星港的共鸣过滤器和灵核复检系统是否失效。反虫群作战若只按体型、火力或数量排序目标，容易把高威胁平台误判为指挥核心，或把低可见度的节律节点和分巢核漏掉。

虫群的“指挥”也不是语言命令、军衔命令或远程遥控。高层相位结构通常通过底噪、化学梯度、以太共鸣、群体记忆和阈值调节改变下级行为；低层虫体接收到的不是清晰命令，而是“靠近、绕行、护巢、拆解、孢化、驱散、静默”等反应模式的概率权重。这使虫群在强干扰环境中不容易被彻底斩首，但也使它缺少智慧军队那种临场协商、伦理判断和复杂目标转换能力。

## 五、指挥层级

| 指挥层级 | 常见承载 | 主要作用 | 失效后果 |
| --- | --- | --- | --- |
| 巢群基准层 | 母巢相位、长期巢化区、群体记忆床。 | 保存巢群连续性、迁徙传统、繁殖节律和长期危险标记，决定一个巢群“是什么”。 | 巢群历史连续性断裂，可能出现狂乱迁徙、多处分巢、休眠化或对旧敌意标记的异常放大。 |
| 迁徙统合层 | 迁徙核心、幼巢群、导航器官和储能生物质。 | 维持迁徙队列、选择落点、调节护巢优先级、协调幼巢和孢群扩散，是星系尺度接触中最关键的活动指挥层。 | 迁徙队列分裂，战虫和工虫转入局部自保；碎裂小巢群更难预测，也更容易沿残骸带和旧锚链噪声槽扩散。 |
| 战区节律层 | 护巢节律节点、工巢节律节点、侦采节律节点，可能由特化虫体、菌膜网络或短寿命共鸣器官构成。 | 在局部战场分配战虫、工虫、孢群和残骸附着体的节奏，决定何时破障、何时撤入巢壳、何时转入生态改写。 | 局部攻势失去节奏，群簇仍会按本能和前序标记作战，但难以持续执行复杂破障、围困和筑巢动作。 |
| 群簇反射层 | 战虫群、工虫群、侦采群、护巢群和孢云带。 | 承担近实时反应：扑向高扰动源、保护幼巢、啃蚀结构、修补壳层、释放孢体或绕开强火力。 | 群簇分散为小规模危险源；单个群簇被击溃不会让整个战区停止，但会降低局部压力和扩散速度。 |
| 个体与微生态自律层 | 单个战虫、工虫、侦察虫、休眠孢囊、共生菌膜和低功率以太薄膜。 | 执行局部触发动作，例如附着、采样、咬合、喷孢、封缝、休眠和污染过滤器。 | 指挥层被压制后仍可能长期残留，成为污染、误报警、灵核错频和设备维护事故的来源。 |

指挥层级越高，越倾向改变长期方向和反应阈值；指挥层级越低，越倾向执行快速、重复、局部的动作。反虫群行动中的“斩首”通常只能切断某个尺度的统合能力，不能自动清除全部虫体和巢化以太。相反，过早摧毁迁徙统合层可能让原本可隔离的迁徙队列碎裂成多个低可预测小巢群，因此必须与航道封锁、孢群清理和巢化边界监测同步。

## 六、物理平台层级

| 平台层级 | 形态 | 作战含义 |
| --- | --- | --- |
| 母巢区 / 行星巢区 | 深层巢穴、行星级巢化生态、巨型生物结构、遗产设施残骸上的巢核。 | 固定或半固定的长期平台，负责生产、记忆、繁殖和生态改写。攻击母巢区不是普通据点攻坚，而是对一整套巢化环境、微生态和以太底噪进行处置。 |
| 迁徙核心体 | 携带幼巢、导航器官、储能囊、工虫巢室、战虫护卫和共鸣孢群的移动巢港。 | 星系内最接近“主平台”的单位。它既承载迁徙统合层，又携带分巢资源；护巢战斗通常围绕它展开。 |
| 漂巢壳 / 残骸附着体 | 附着在小行星、战场残骸、废弃船坞、破损舰体或古代结构上的临时壳层。 | 提供遮蔽、质量锚、孢体床和伪装。文明传感器可能把它们误判为普通残骸，直到共鸣底噪、壳层修补和工虫活动暴露。 |
| 破障战体 | 大型战虫、合生战体、短时聚合的撞击阵列或生物装甲团。 | 主要承担突防、撞击、护盾压制和结构撕裂。它们火力或肉体威胁很高，但通常不是长期指挥核心。 |
| 群簇阵列 | 多个战虫、工虫、侦察虫或搬运体按共鸣节律组成的移动阵列。 | 是最常见的战术平台，可在残骸带、星港外壁、生态穹顶表面和行星地表之间重组。其作战单位更接近“群体集合”，不是单个载具。 |
| 孢云 / 菌膜带 | 共鸣孢群、菌膜、休眠孢囊、低功率以太薄膜和微型共生体。 | 承担侦察、污染、环境标记和巢化预铺。它不是火力平台，却能让战场传感、共鸣通讯、医疗复检和灵械接口同时变差。 |
| 分巢核 / 幼巢舱 | 可迁移的幼巢、分巢种子、孵化囊和护巢壳层。 | 战斗能力未必强，但战略价值极高。文明若捕获或误搬运幼巢舱，可能把自身设施变成长期巢群记忆标记。 |

平台层级决定文明能看见什么、打击什么和封锁什么；指挥层级决定虫群为什么这样行动。一个星港外侧的虫群接触，可能同时包含孢云作为环境平台、群簇阵列作为战术平台、破障战体作为突防平台，以及躲在残骸内部的战区节律节点作为局部指挥层。只清除其中一种平台，通常不能结束整场接触。

## 七、平台物理特征

虫群平台通常是生命组织、矿物壳层、共生菌膜、以太薄膜、残骸结构和储能器官的复合体。它们可以利用被啃蚀的舰体、空间锚外壳、岩体和古代设施碎片，但不会自然长成文明舰船那种清晰的龙骨、甲板、舰桥、舱室和推进器布局。文明记录中的“壳”“舱”“战列”“推进带”多是观测标签，不代表虫群内部存在同等工程分类。

| 平台层级 | 典型物理特征 | 观测与打击含义 |
| --- | --- | --- |
| 母巢区 / 行星巢区 | 不是单一巨兽，更像被巢化以太贯穿的地下巢穴、行星生态斑块、矿物壳层森林、菌膜海和巨型器官群。表面可出现脉动裂隙、矿物肋骨、孢体塔、低光辉纹和被改写的遗产结构。 | 远看像异常地貌或遗产污染区，近看才呈现生命组织。军事打击不能只摧毁可见巢核，还要处理地下孢床、菌膜水循环、矿物壳层和以太底噪。 |
| 迁徙核心体 | 最接近“移动巢港”。外层通常是多层矿物甲壳、残骸骨架和生物膜，内部有幼巢囊、储能囊、导航器官、工虫巢室、孢体腔和节律节点。形态可能像巨型囊泡、带壳浮岛、附肢缓慢摆动的生物堡垒，或包裹在废弃舰体里的活体器官群。 | 它可被误认为生物战舰或活体母舰，但没有舰桥和船员，也不是单个“指挥官”。击穿外壳可能释放孢体、幼巢碎片和巢场倒灌，因此反虫群部队通常先隔离、降噪和定向剥壳。 |
| 漂巢壳 / 残骸附着体 | 由舰船残骸、小行星、锚站碎片或古代结构外覆虫群壳层形成，外观不规则，常有修补状矿物痂、脉管状菌膜、半透明孢囊和低功率以太薄膜。 | 最容易伪装成普通残骸。识别重点不是形状，而是壳层是否“自愈”、残骸内部是否有热斑移动、共鸣底噪是否呈现周期性呼吸。 |
| 破障战体 | 大型战虫、合生撞击体或短寿命生物装甲团，常见楔形头壳、钻咬口器、钩爪、喷射囊、烧蚀外壳、牵引脊和可脱落甲片。它们可能携带孢囊或贴附小虫，但内部通常没有复杂巢室。 | 视觉威胁最大，最像“怪物战舰”或攻城锤，却多半只是突防工具。摧毁它能缓解压力，但不等于打掉指挥核心。 |
| 群簇阵列 | 由大量虫体按节律拉成链、网、扇面、包络层或旋涡，形态会随目标、护盾窗口和残骸环境重组。个体之间可用牵引丝、菌膜桥、孢尘带或短时共鸣场连接。 | 远距离传感器可能把它看成一片“活动云墙”或“碎片带”。打击重点是打断节律节点和连接带，而不是逐个清空所有虫体。 |
| 孢云 / 菌膜带 | 由微型孢体、休眠孢囊、共生菌、尘埃附着体和低功率以太薄膜构成，肉眼可呈雾、尘、薄膜、黏附霜、光点雨或过滤器内的絮状沉积。 | 几乎不像军事平台，却会让传感、共鸣通讯、灵核复检、密封系统和灵械接口同时变差。清理需要过滤、烧蚀、以太场清洁和长期复检。 |
| 分巢核 / 幼巢舱 | 常呈多层囊、种子壳、矿物卵、带脉动纹路的残骸球或被工虫拖曳的孵化舱。外层比战虫更趋向隔离和缓冲，内部承载幼巢相位与分巢资源。 | 不一定有强火力，但战略风险极高。误搬运、研究泄漏或封存失败，可能让实验室、星港或救援舰进入长期巢群标记。 |

## 八、尺度参照

虫群尺度应优先使用世界内参照物，而不是固定数值。尺度只说明文明如何观测、拦截、隔离和清理目标，不自动说明指挥层级、智慧程度或战略价值。一个机甲尺度的战虫可能只是低层反射单位，一个肉眼不可见的孢云却可能决定整座星港的共鸣过滤系统是否失效；一个类地行星级巢区也不是“行星大小的单个怪物”，而是一片被巢化以太重写的生态与地理系统。

| 尺度参照 | 世界内参照物 | 常见虫群对象 | 说明 |
| --- | --- | --- | --- |
| 微生态 / 传感器尺度 | 共鸣过滤器、舰体密封缝、机甲接口端口、医疗复检采样盒、空气循环和水循环滤芯。 | 微型孢体、休眠孢囊、菌膜、低功率以太薄膜、尘埃附着体。 | 这个尺度的虫群对象通常不被肉眼当作敌人，而表现为过滤器絮化、传感器漂移、灵械接口刺痛、灵核复检异常和密封系统污染。 |
| 个人装备 / 自动器械尺度 | 工程外骨骼、自动执法机器、维修灵械、无人侦察单元和机甲外部工具臂。 | 小型侦察虫、微型附着体、轻型工虫、贴附战虫。 | 可被设施安保、无人单元或机甲外部工具处理，但若数量上升，会迅速从“异物清理”变成接口污染和孢云预铺。 |
| 载人机甲尺度 | 工程/救援机甲、军用高速机甲、突入队近域作战平台。 | 成熟战虫、重型工虫、小型破障战体、护巢贴附虫。 | 这是近域战斗最常见的直观尺度。单只战虫可以和机甲形成一对一或小队级接触，但战虫通常不是拟人机体，而是多足、钩附、喷射、咬合或撞击结构。 |
| 机甲小队 / 突入舱尺度 | 机甲突入队、登舰队、突入舱、星舰外壳局部作业区。 | 破障突入群、小型群簇阵列、工虫搬运链、孢云局部包络。 | 这类目标需要机甲、无人单元、舰载点防御和低功率护盾协同处置。它们不是单个巨大怪物，而是多个虫体和连接结构组成的局部战术构型。 |
| 星舰外壳 / 护盾节点尺度 | 战舰局部外壳、护盾发生器阵列、星港闸门、船坞泊位、空间锚外层维护结构。 | 大型破障战体、群簇阵列、漂巢壳、残骸附着体。 | 这个尺度的虫群对象足以压制舰体局部结构、护盾短窗或星港外壁。文明火控会把它们当作舰载武器目标，但仍需防止击碎后释放孢云和幼巢碎片。 |
| 舰队主目标尺度 | 主战舰、护航舰群、航母型母舰、维修船坞舰、舰队级护盾面。 | 迁徙核心体、护巢战列、较大的漂巢壳群、战斗护巢状态下的核心护卫阵列。 | 当前战舰具体尺寸尚未专题化，因此这里用“舰队主目标”而非硬数值。护巢战列在这个尺度上像活体主力舰，但本质是复合护巢阵列，不是单艘生物舰。 |
| 星港 / 空间锚设施尺度 | 星港泊区、船坞环、空间锚节点、生态穹顶外壁、轨道工业环局部。 | 工巢推进带、筑巢封锁面、大范围孢云、多个残骸潜伏巢连成的巢化带。 | 这已经不是单纯战斗单位，而是设施围困和生态改写前线。处置目标是切断推进带、清理孢床、保护维护链和撤离通道。 |
| 卫星 / 类地行星生态尺度 | 小型卫星、类地行星局部大陆、海盆、环形山群、行星生态圈和长期殖民区。 | 母巢区、行星巢区、成熟巢化以太区、多处分巢网络。 | 这个尺度的虫群是灾害地理和生态系统，不是可由一场舰队炮击解决的“超巨型敌人”。反虫群行动会转向长期封锁、撤离、巢化边界划定、灵核医学复检和恢复价值评估。 |

尺度参照还会随巢群状态变化。避灾迁徙中的迁徙核心体可以被舰队记录为主目标，但未必主动展开护巢战列；战斗护巢状态下，同一核心体会被战虫、孢云和破障战体包成舰队主目标尺度的复合阵列；筑巢状态下，原本只相当于星舰外壳尺度的残骸附着体可能沿船坞和星港外壁扩展为设施尺度的工巢推进带。

## 九、作战单位组合

虫群的具体作战单位通常由“指挥层级 + 物理平台 + 巢群状态”组合而成。文明记录中常见的单位名称只是观测标签，不代表虫群内部存在同名军衔或固定编制。

| 组合单位 | 指挥核心 | 主要平台 | 战场物理特征 | 常见任务 |
| --- | --- | --- | --- | --- |
| 护巢战列 | 迁徙统合层与护巢节律节点。 | 迁徙核心体、破障战体、战虫群簇、孢云屏障。 | 可以在轮廓上类似“生物体战舰”或活体母舰：迁徙核心体像主舰体，残骸壳和矿物甲层像装甲，破障战体像前出炮塔或撞角，孢云像护盾外雾。但它不是一艘孤立舰船，而是由核心体、护卫虫群、孢云和节律节点组成的动态护巢阵列。 | 保护幼巢和迁徙核心，压制舰队火控、护盾阵列和空间锚，争取迁徙或落巢窗口。 |
| 工巢推进带 | 工巢节律节点与群簇反射层。 | 工虫群、菌膜带、残骸附着体、低速搬运阵列。 | 像缓慢推进的生物工程前线：矿物壳层、菌膜、搬运虫链和孢体床沿残骸或地表铺开，边缘不断吞入材料、排出壳层和孢尘。 | 拆解残骸、搬运矿物、铺设孢体床和矿物壳层，把临时污染推进为稳定巢化边界。 |
| 破障突入群 | 战区节律层。 | 破障战体、贴附战虫、短时生物装甲阵列。 | 呈尖锥、楔形、钻头、撞锤或多体包络形态，常把小型贴附虫和生物甲片叠在前缘，以短时烧蚀和牺牲外层换取护盾短窗。 | 撕开护盾短窗、压制星港外壁、扰乱舰船姿态控制或拖住机甲突入队。 |
| 侦采预铺群 | 侦采节律节点或低层群簇反射。 | 侦察虫、孢云、休眠孢囊、微型附着体。 | 肉眼常看不见完整单位，只表现为尘埃异常、过滤器絮化、微弱辉点、传感器误差和小型附着虫活动。 | 测量成熟灵核生态、标记高相干能源源、污染共鸣过滤器，为迁徙核心提供路径信息。 |
| 残骸潜伏巢 | 低功率分巢核、休眠孢囊或断续节律节点。 | 残骸附着体、漂巢壳、菌膜层。 | 像一块长期漂浮的废舰、小行星或锚站残壳；区别在于裂缝会被矿物痂和菌膜缓慢封合，内部热斑和共鸣噪声会周期性复苏。 | 在废弃航道、战后残骸带或旧锚链附近休眠，等待合适底噪、能流或文明工程扰动唤醒。 |
| 筑巢封锁面 | 分巢核与工巢节律节点。 | 幼巢舱、工虫群、孢云、菌膜带和护巢战虫。 | 像覆盖在生态穹顶、星港外壁或殖民设施边缘的活体围困层：外侧有护巢战虫巡游，内侧有菌膜、孢床、矿物壳和工虫通道持续增长。 | 围困生态穹顶、星港或殖民设施，切断维护链和共鸣中继，逐步把设施外缘改写为巢化环境。 |

同一物理平台在不同巢群状态下会表现为不同单位。迁徙核心体在避灾迁徙中更像需要开辟通道的移动巢港，在战斗护巢状态中会变成所有战虫和孢云围绕保护的核心，在分巢迁徙中则是携带新巢群种子的高风险扩散源。反虫群作战必须先判断当前组合单位，而不是只统计虫体数量或寻找最大的目标。

护巢战列是否“类似生物体战舰”，取决于观测距离和记录口径。对舰队雷达、光学和火控人员来说，它常常像一艘被虫群组织出来的活体主力舰：有中心质量、有前缘、有护卫层、有可持续压制能力，也会在舰队交战距离上承担主目标角色。但从虫群自身结构看，它更接近“迁徙核心体 + 护巢群簇 + 孢云屏障 + 破障战体”的临时战斗构型。它没有单一舰长、舰桥、舱室和固定武器系统；它的“炮塔”可能是短时靠近的破障战体，“护盾”可能是孢云和共鸣噪声，“装甲”可能是可脱落矿物壳和残骸骨架。因此，写作时可以让人物把它俗称为生物战舰，但正文口径应说明它不是独立舰船，而是复合护巢阵列。

## 十、星系内移动方式

虫群在星系内移动通常采用多种方式组合，而不是像文明舰船一样依赖单一推进系统。它们一般不能像现代战舰那样稳定执行自由跃迁；跨星区迁徙更多依赖长期航迹、深层潮汐、旧锚链噪声、残骸跳板和被遗产信号牵引的路径。

| 方式 | 适用场景 | 说明 |
| --- | --- | --- |
| 生物推进 | 短距离机动、护巢战斗、残骸间转移。 | 战虫和工虫可依靠生物推进器官、喷射代谢气体、矿物壳层反作用和群体牵引完成机动。 |
| 以太漂流 | 中距离迁徙、低能耗通过恒星系边缘和尘埃带。 | 迁徙核心会顺着恒星风、尘埃带、以太潮汐和弱稳定航迹漂流，降低能耗并保持巢场同步。 |
| 附着残骸 | 碎星带、战场残骸、废弃船坞或古代设施附近。 | 虫群会把残骸当作屏蔽壳、质量锚、孢体床和临时跳板，也可能借残骸降低文明侦测效率。 |
| 利用旧锚链噪声 | 断裂空间锚、失准锚标、古代锚链残段和废弃航道附近。 | 旧锚链会在以太层留下低频噪声槽，文明舰船会把它们视为不可靠航迹，虫群却能将其当作迁徙导向和巢场校准线。 |
| 共鸣孢群预铺 | 迁徙核心抵达前的侦察、标记和环境改写。 | 孢群先行扩散后，会让传感器、共鸣过滤器、灵核复检和中继底噪出现异常，为后续虫群迁徙提供路径信息。 |

因此，虫群常沿旧航道断裂带、废弃锚站、深层边界和战后残骸带出现。文明若只按正常航道和舰队动力学预测虫群，会低估它们在破碎航路、失准锚链和灰色残骸带中的移动能力。

## 十一、攻击对象与优先级

虫群攻击对象不按文明军事价值排序，而按巢群相位影响排序。只要某个对象阻断迁徙路径、污染巢群共鸣、抢占筑巢基质、捕获幼巢或持续发出高强度干扰，它就可能被虫群标记为障碍、猎物、噪声源、寄生源或竞争节点。

| 对象 | 触发原因 | 常见虫群行动 |
| --- | --- | --- |
| 高相干能源源、护盾阵列、星港高功率输出 | 既像强烈威胁，也像高价值能流标记。 | 战虫压制、共鸣孢群干扰、工虫啃蚀散热和供能结构。 |
| 共鸣中继、大型通讯节点 | 在巢群感知中是持续噪声源，并能帮助文明组织救援、火力和撤离。 | 迅速干扰、污染过滤器、啃蚀外壳或铺设孢体床。 |
| 空间锚、旧锚链节点、临时锚标 | 固定航道、撤离窗口和迁徙路径，可能抢占虫群可利用的噪声槽。 | 压制、污染、占据或将其转化为迁徙校准点。 |
| 舰船、无人探针、样本回收器 | 主动扫描、开火、展开护盾、拖走幼巢样本或阻挡迁徙队列时进入高优先级。 | 驱散、贴附、破障、近距突入或群体噪声压制。 |
| 生态圈、殖民穹顶、成熟居住区 | 在筑巢或分巢状态下提供成熟灵核生态、生物质、水分、微量矿物和长期孵化环境。 | 共鸣孢群预铺、工虫生态改写、战虫护巢和长期围困。 |

智慧生命不是因为“有智慧”而被优先仇恨，而是因为灵核、灵械、护盾、共鸣网络和高相干能源设施会在虫群感知中形成强烈、复杂且持续的以太扰动。被攻击对象可以是战舰、殖民地、星门锚站、生态穹顶、矿业设施、无人探针、共鸣中继、遗产样本库，也可以是智慧生命群体。

不同巢群状态会改变攻击优先级。

| 巢群状态 | 优先级变化 |
| --- | --- |
| 避灾迁徙 | 倾向驱散舰队、绕开高能区并保持迁徙核心完整；若文明让出通道，虫群可能不扩大接触。 |
| 分巢迁徙 | 优先争夺空间锚噪声槽、成熟灵核生态和可筑巢残骸；殖民地、星港和废弃锚站风险显著上升。 |
| 筑巢状态 | 生态圈、共鸣中继、灵核医疗与训练设施、高相干能源源会同时进入高风险清单。 |
| 战斗护巢状态 | 战虫优先压制护盾阵列、舰队火控、空间锚和任何试图接近迁徙核心或幼巢的单位。 |

## 十二、作战反应层级

虫群行动通常具有分层反应。

- 低强度接触：绕行、贴附、采样、释放孢体、测试共鸣中继和空间锚反应。
- 中强度干扰：切断中继、啃蚀设施、压制护盾边界、驱赶舰队、污染临时锚标。
- 高强度威胁：战虫集群、母巢护卫、迁徙核心转向、对星港和殖民穹顶的持续围攻。

文明误把侦察、采样或驱赶行为当作全面进攻，或误把暂时绕行当作安全，都会扩大灾害。反虫群作战的关键不是第一时间歼灭所有虫体，而是判断巢群状态、迁徙动因、幼巢位置、共鸣孢群扩散范围和旧锚链噪声对虫群的吸引强度。

## 十三、反虫群处置边界

反虫群行动不只是军事清除，也包括航道改线、居民撤离、生态隔离、共鸣屏蔽、虫群行为记录、样本封存、巢化以太监测、灵核复检、长期封锁和恢复价值评估。一般处置边界如下：

| 情况 | 优先处置 |
| --- | --- |
| 避灾迁徙 | 优先改线、撤离、降低高相干能源输出和清理阻断路径的舰队活动。 |
| 分巢迁徙 | 优先建立隔离走廊，封锁成熟灵核生态节点，切断虫群对旧锚链噪声槽和残骸带的利用。 |
| 筑巢状态 | 优先封存、长期监测、居民撤离、灵核医学复检和巢化以太边界划定。 |
| 战斗护巢状态 | 军事压制、护盾阵列协调、迁徙核心隔离和幼巢样本防扩散并行。 |

若文明试图以共鸣诱饵、幼巢样本或遗产信号操控迁徙，可能短期改变路径，却会提高后续攻击阈值、诱发巢群记忆标记，甚至把某个星港或文明设施变成长期风险源。

常见治理争议包括：虫群是否可被视为生态主体、军事清除边界、样本研究权、诱导迁徙是否合法、成熟殖民地为何必须撤离、巢化区域是否还有恢复价值，以及智慧生命在虫群路径中应被优先撤离还是允许承担观察和研究风险。

处置方案应同时标注目标的指挥层级和物理平台层级。摧毁破障战体可能只是解除突防压力；压制战区节律节点可能让局部攻势降级；隔离迁徙核心体可能改变整个迁徙队列；清理孢云和菌膜带则决定战后是否还能恢复共鸣网络、灵核医疗和灵械维护。若只记录“击杀虫体数量”或“摧毁大型生物结构”，无法判断虫群是否真正失去组织能力，也无法评估巢化以太的长期恢复成本。
