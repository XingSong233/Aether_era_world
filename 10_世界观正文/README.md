# 世界观正文

本目录承载“以太纪元”的权威设定正文。这里的“世界观内部文本”指总纲、正式专题、术语、角色与稳定设定口径，不包括写作指南、推导蓝图、发布层导览或纯风格样稿。

## 目录分工

- `00_总纲/`：当前权威总纲。普通扩写先读总纲，再进入对应专题。
- `10_专题设定/`：正式专题正文，承载稳定设定、制度结构、技术边界、文明差异和示例对象。

## 使用原则

- 正式专题不放写作模板、检查清单、风格候选和协作规则。
- 设计蓝图、状态表、理论推导和原型工具统一放入 `90_协作工作台/10_设定推导与蓝图/`。
- 世界内报道、档案、通讯、展示文和发布对象统一放入 `20_展示发布层/` 或样稿库。
- 核心口径变化后，同步检查总纲、专题、协作规则、版本索引和 CHANGELOG。
